# Amazon EC2 High Memory U7i インスタンスが追加のリージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月24日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ec2-high-memory-u7i/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ec2-high-memory-u7i/)

## 概要

<p>Amazon EC2 High Memory U7i-8TB インスタンス（u7i-8tb.112xlarge）は、AWS ヨーロッパ（ストックホルム、チューリッヒ）リージョンで利用可能になりました。U7in-16TB インスタンス（u7in-16tb.224xlarge）は、AWS US East（オハイオ）リージョンで利用可能になりました。U7in-24TB インスタンス（u7in-24tb.224xlarge）は、AWS ヨーロッパ（ストックホルム）リージョンで利用可能になりました。U7i インスタンスは AWS 第7世代の一部で、カスタム第四世代 Intel Xeon Scalable プロセッサ（Sapphire Rapids）を搭載しています。U7i-8TB インスタンスは DDR の 8 TiB を提供します。
