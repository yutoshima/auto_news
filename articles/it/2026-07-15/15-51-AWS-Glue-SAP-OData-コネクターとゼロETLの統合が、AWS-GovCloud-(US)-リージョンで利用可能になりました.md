# AWS Glue SAP OData コネクターとゼロETLの統合が、AWS GovCloud (US) リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月15日 15:51
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/aws-glue-sap-zero-etl-govcloud](https://aws.amazon.com/about-aws/whats-new/2026/07/aws-glue-sap-zero-etl-govcloud)

## 概要

AWS Glue SAP OData コネクタとゼロETL統合が現在、AWS GovCloud（US-West）およびAWS GovCloud（US-East）リージョンで利用可能になり、ソースとしてAmazon DynamoDB、Salesforce、SAP ODataをサポートします。規制環境で運用している顧客は、カスタムデータパイプラインを構築・維持することなく、これらのソースからAmazon Redshift、Amazon S3、または他のサポート対象の宛先へデータをレプリケーションできます。SAP OData コネクタは、SAPシステムからデータを抽出することを可能にします。
