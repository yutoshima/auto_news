# Amazon OpenSearch Serviceがクラスタインサイトを拡張、新たなインサイトを追加します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月05日 10:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-opensearch-cluster-insights/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-opensearch-cluster-insights/)

## 概要

<p>Amazon OpenSearch Service は、クラスタの健康状態とパフォーマンスの可視化を Console を通じて提供する「Cluster Insights」機能を、すべての OpenSearch バージョンおよび Elasticsearch バージョン 6.8 以降に拡張します。さらに、新機能の「Unused Index Insight」では、過去 30 日間の検索およびインデックス作成のアクティビティがゼロだった OpenSearch クラスター内のインデックスを特定し、コスト最適化のための実践的な推奨事項を提供します。<br /> <br /> Cluster Insights は現在、対応バージョンを拡張しています。
