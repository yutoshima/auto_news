# AWSセキュリティエージェントはペンテストの所見に対する検証スクリプトを追加します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月22日 17:21
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-security-agent/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-security-agent/)

## 概要

<AWS Security Agent> は現在、ペンテストの所見の検証スクリプトを生成します。これにより、セキュリティチームは発見した脆弱性を独立して再現・検証できます。  
以前は、チームが所見の詳細から再現手順を手動で追跡していました。現在、AWS Security Agent は確認済みの各所見に対して、すぐに実行できるスクリプトを自動生成します。チームはスクリプトをダウンロードし、環境変数を設定して、再度実行します。
