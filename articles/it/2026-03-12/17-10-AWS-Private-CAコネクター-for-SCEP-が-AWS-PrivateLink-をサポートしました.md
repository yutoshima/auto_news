# AWS Private CAコネクター for SCEP が AWS PrivateLink をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月12日 17:10
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-private-ca-connector-scep-privatelink/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-private-ca-connector-scep-privatelink/)

## 概要

<p>AWS Private CA Connector for SCEP は現在 AWS PrivateLink をサポートしており、クライアントが公衆インターネットを経由せずに Amazon Virtual Private Cloud (VPC) 内で証明書をリクエストできるようにします。今回のリリースにより、SCEP コネクターにプライベートに接続するための VPC エンドポイントを作成でき、すべてのトラフィックを AWS ネットワーク内に留めることができます。<br /><br /> AWS Private CA Connector for SCEP はマネージドコネクターで、Simple Certificate Enrollment Protocol (SCEP) を使用して証明書を発行することを可能にします。</p>
