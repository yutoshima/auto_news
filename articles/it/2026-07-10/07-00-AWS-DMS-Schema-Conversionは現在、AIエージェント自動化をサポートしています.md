## 概要

<AWS Database Migration Service (DMS) Schema Conversion>は現在、AWS MCP Serverを通じたAIエージェント自動化をサポートしています。Kiro、Claude Code、Cursorを含むAIコーディングエージェントをDMS Schema Conversionに接続し、IDEから自然言語で直接、完全な移行ワークフローを実行できます。エージェントはプロジェクトを作成し、ソースのメタデータを閲覧し、スキーマを変換し、評価レポートを作成し、エクスポートします。
