# Amazon EC2 C7a インスタンスが米国西部（ノースカリフォルニア）リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月22日 21:42
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-c7a-instances-us-west-ncalifornia-region/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-c7a-instances-us-west-ncalifornia-region/)

## 概要

本日より、計算最適化された Amazon EC2 C7a インスタンスが AWS US West (N. California) リージョンで利用可能になりました。第4世代 AMD EPYC プロセッサー（コードネーム Genoa）を搭載し、最大周波数は 3.7 GHz、C6a インスタンスと比較して最大 50% の性能向上を実現します。C7a インスタンスは AVX-512、VNNI、bfloat16 などの新しいプロセッサ機能を提供します。メモリへの高速データアクセスを可能にする DDR5 メモリを搭載し、データのアクセスが 2.25 倍向上します。
