# Amazon RDS for SQL Server が X2m インスタンスをサポートするようになりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月16日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/rds-sqlserver-supports-x2m/](https://aws.amazon.com/about-aws/whats-new/2026/06/rds-sqlserver-supports-x2m/)

## 概要

<a href="https://aws.amazon.com/rds/sqlserver/" target="_blank">Amazon Relational Database Service (Amazon RDS) for SQL Server</a> は、メモリ最適化型の X2m データベースインスタンスを提供開始します。Amazon EC2 の X2iedn インスタンスをベースとする X2m データベースインスタンスは、<a href="https://aws.amazon.com/about-aws/whats-new/2025/12/amazon-rds-sql-server-optimized-cpu-lower-prices/" target="_blank">Amazon RDS Optimize CPU</a> 機能を提供し、SQL Server のソフトウェアライセンス費用を最大50％削減できるようにします。
