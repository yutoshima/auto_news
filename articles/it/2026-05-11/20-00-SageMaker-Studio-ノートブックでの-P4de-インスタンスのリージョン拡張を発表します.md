# SageMaker Studio ノートブックでの P4de インスタンスのリージョン拡張を発表します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月11日 20:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/02/p4de-region-expansion-sagemaker-studio-notebooks/](https://aws.amazon.com/about-aws/whats-new/2026/02/p4de-region-expansion-sagemaker-studio-notebooks/)

## 概要

私たちは、SageMaker Studioノートブック上でのアジア太平洋（東京、シンガポール）およびヨーロッパ（フランクフルト）でのAmazon EC2 P4deインスタンスの一般提供を開始したことをお知らせします。  
Amazon EC2 P4deインスタンスは、8つのNVIDIA A100 GPU（80GBの高性能HBM2e GPUメモリ）を搭載しており、現在のP4dインスタンスのGPUより2倍の容量を実現しています。新しいP4deインスタンスは合計640GBのGPUメモリを提供し、最大で60%の性能向上を実現します。
