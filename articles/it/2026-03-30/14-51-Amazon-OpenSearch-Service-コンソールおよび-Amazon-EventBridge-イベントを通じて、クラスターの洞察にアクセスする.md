# Amazon OpenSearch Service コンソールおよび Amazon EventBridge イベントを通じて、クラスターの洞察にアクセスする

- **カテゴリ**: it
- **公開日時**: 2026年03月30日 14:51

## 概要

<p>Amazon OpenSearch Service は、既存の OpenSearch UI Dashboards に加え、AWS Management Console からも Cluster Insights へのアクセスを拡張します。このリリースにより、同じコンソール内でパフォーマンスとレジリエンスの推奨事項を確認し、必要な設定変更を行うことが容易になります。さらに、Cluster Insights は現在、イベントとして Amazon EventBridge に発行します。<br /> <br /> Cluster Insights は、クラスターの運用状態に関する厳選された洞察と、それに対する行動を提示します。</p>
