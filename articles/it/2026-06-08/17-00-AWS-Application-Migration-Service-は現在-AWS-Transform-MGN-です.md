# AWS Application Migration Service は現在 AWS Transform MGN です

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月08日 17:00

## 概要

AWS Application Migration Service (MGN) は現在、 <b>AWS Transform MGN</b> として利用可能です。この名称変更は、MGN が AWS Transform を支える実証済みの複製エンジンとしての役割を反映しています。Transform はエージェント型の移行サービスです。

2つのリホスト体験のいずれかを選択できます。複製とカットオーバーを直接制御できる AWS Transform MGN コンソールを使用する方法。あるいは、探索、ウェーブ計画、ランディングゾーンの設定、ネットワーク作成、リホストをエージェントが処理する AWS Transform のエージェント型ワークフローを使用する方法。
