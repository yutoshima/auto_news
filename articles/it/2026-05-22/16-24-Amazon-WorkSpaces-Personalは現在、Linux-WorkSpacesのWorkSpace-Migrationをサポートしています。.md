# Amazon WorkSpaces Personalは現在、Linux WorkSpacesのWorkSpace Migrationをサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月22日 16:24
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/workspaces-linux-migration](https://aws.amazon.com/about-aws/whats-new/2026/05/workspaces-linux-migration)

## 概要

Amazon WorkSpaces は、Amazon WorkSpaces が提供するすべての Linux OS に対して、WorkSpace Migration 機能をサポートするようになりました。これにより、顧客は WorkSpaces を一つの Linux OS から別の Linux OS へシームレスに移行でき、より新しい OS バージョンへの移行や、異なる Linux OS への移行を自動化できます。  
顧客が WorkSpaces をある OS から別の OS に移行する際、Linux WorkSpace のホームディレクトリ上のユーザデータは、移行元と移行先の OS によって取り扱いが異なる場合があります。
