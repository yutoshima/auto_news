# Amazon Bedrock AgentCore Browser は OS レベルの対話機能を追加します

- **カテゴリ**: it
- **公開日時**: 2026年04月08日 17:03

## 概要

<p>Amazon Bedrock AgentCore Browser は、OSレベルの操作機能をサポートするようになり、Chrome DevTools Protocol（CDP）機能だけでは対応できない直接的なOS制御を必要とするブラウザワークフローの自動化を可能にします。この強化により、マウス操作、印刷ダイアログ、ネイティブなシステムアラート、キーボードショートカットなど、CDPだけでは不十分な自動化シナリオに対応します。本機能は、AIエージェント開発者、テスト自動化エンジニア、およびLLMを活用したウェブアプリを構築する組織を支援します。
