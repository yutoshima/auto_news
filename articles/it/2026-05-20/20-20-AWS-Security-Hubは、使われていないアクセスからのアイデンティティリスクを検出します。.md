# AWS Security Hubは、使われていないアクセスからのアイデンティティリスクを検出します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月20日 20:20

## 概要

今日は、AWS Security Hub がアイデンティティリスクを、脅威・露出・セキュリティの姿勢所見を一元化したコンソールと同じ場所に統合します。Security Hub は現在、使用されていない IAM の権限・ロール・資格情報を AWS 組織全体で検出し、中央のセキュリティチームがアイデンティティリスクを大規模に特定・低減できるよう支援します。これまで百を超えるアカウントにわたるアイデンティティリスクの管理は、複数のツールを切り替えながら行う必要があり、未使用の権限と他の要素を結びつける統一ビューがありませんでした。
