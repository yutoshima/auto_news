# Amazon SageMakerはSCP準拠のために権限境界を追加します

## 概要

<p>Amazon SageMaker Unified Studio はカスタム IAM 権限境界をサポートするようになり、すべての IAM ロールに権限境界を要求するサービス制御ポリシー（SCP）を適用している組織でも、セキュリティ体制を変更せずに SageMaker Unified Studio を導入できます。<br /> <br /> ユーザーがプロジェクトを作成すると、SageMaker Unified Studio は3つの IAM ロールを用意します：プロジェクトユーザー ロール、Amazon Bedrock サービス ロール、 Bedrock Lambda 実行ロールです。今回のリリースにより、管理者は仕様を指定できます。</p>
