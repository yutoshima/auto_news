# AWS DMS Schema Conversion は、オフラインの SQL Server 変換をサポートしました。

## 概要

AWS Database Migration Service (DMS) Schema Conversion は現在、Microsoft SQL Server のオフラインソース変換をサポートしています。これにより、ソースデータベースに直接接続せずに、SQL Server のスキーマとコードを変換できます。自分の環境で標準的なデータベースコマンドを用いてメタデータを抽出し、それを DMS Schema Conversion にアップロードして処理します。これによりセキュリティ審査やファイアウォールの制約を回避できます。
