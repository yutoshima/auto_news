# AWS Glue は Snowflake への接続に対して OAuth 2.0 をサポートするようになりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月21日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-glue-oauth-snowflake-connectivity/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-glue-oauth-snowflake-connectivity/)

## 概要

本日より、AWS Glue はネイティブ Snowflake 接続に対する OAuth 2.0 認可・認証をサポートします。ユーザーの認証情報を共有することなく、Snowflake への読み取り・書き込みが行えるようになり、企業がデータ統合パイプラインを構築する際のセキュリティ遵守を維持しやすくなります。OAuth サポートにより、Temporary token-based authorization を使用して、AWS Glue 内の Snowflake データに安全にアクセスできます。  
AWS Glue は Snowflake への組み込みコネクタを提供します。
