# Amazon GuardDuty が機微なファイル変更を検知する脅威検出機能を追加しました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月01日 16:58
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-guardduty-sfm/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-guardduty-sfm/)

## 概要

<p>Amazon GuardDuty Runtime Monitoringには、新たに3つの脅威検知が追加されました。これにより、Amazon EC2インスタンス上およびAmazon EKS／Amazon ECS上で実行されているコンテナのワークロードで機密ファイルが変更された場合に、セキュリティ担当者へ警告します。これらの検出結果は、設定ファイル・認証設定・システムログなど、重要なシステムファイルを監視することで、侵害後の攻撃者の活動を特定するのに役立ちます。この機能は、セキュリティチーム、DevSecOpsの専門家、クラウドセキュリティアーキテクト向けに設計されています。
