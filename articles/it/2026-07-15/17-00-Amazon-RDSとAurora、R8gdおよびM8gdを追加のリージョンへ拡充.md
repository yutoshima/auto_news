# Amazon RDSとAurora、R8gdおよびM8gdを追加のリージョンへ拡充

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月15日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-rds-aurora-r8gd-m8gd-regions/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-rds-aurora-r8gd-m8gd-regions/)

## 概要

Amazon Relational Database Service (RDS) は、Amazon Aurora PostgreSQL、RDS for PostgreSQL、RDS for MySQL、RDS for MariaDB の Optimized Reads を利用する R8gd データベースインスタンスと M8gd データベースインスタンスを、追加の 12地域および 6地域でサポートします。  
R8gd および M8gd インスタンスは、Aurora PostgreSQL に対して R6g インスタンスと比較してスループットを最大 165%、価格性能を最大 120% 向上させます。Optimized Reads は、ローカルの NVMe ベースの SSD ブロックストレージを使用して、一時的なデータを保存します。
