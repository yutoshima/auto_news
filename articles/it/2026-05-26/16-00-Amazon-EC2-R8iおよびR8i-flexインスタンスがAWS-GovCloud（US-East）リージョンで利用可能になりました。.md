# Amazon EC2 R8iおよびR8i-flexインスタンスがAWS GovCloud（US-East）リージョンで利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月26日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-r8i-r8i-flex-govcloud-east/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-r8i-r8i-flex-govcloud-east/)

## 概要

本日より、Amazon Elastic Compute Cloud（Amazon EC2）のR8iおよびR8i-flexインスタンスが、AWS GovCloud (US-East)リージョンで利用可能になります。これらのインスタンスは、AWS専用のカスタムIntel Xeon 6プロセッサーを搭載しており、クラウド上の同等のIntelプロセッサの中で最高のパフォーマンスと最速のメモリ帯域を実現します。R8iおよびR8i-flexインスタンスは、従来の世代のIntelベースのインスタンスと比較して、価格性能比を最大15%向上させ、メモリ帯域を2.5倍拡張します。
