# Amazon S3 Tablesが、2つの追加のAWSリージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月29日 04:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-s3-tables-aws-regions/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-s3-tables-aws-regions/)

## 概要

<p>Amazon S3 Tables は、アジア太平洋（台北）およびアジア太平洋（ニュージーランド）リージョンで利用可能になりました。</p>
<p>Amazon S3 Tables は、組み込みの Apache Iceberg サポートを備えた最初のクラウドオブジェクトストアを提供し、スケールでの表形式データの格納を合理化します。S3 Tables は、データレイクの成長や進化に合わせて、クエリの効率を最適化し、ストレージコストを削減するための継続的なテーブルメンテナンスを自動的に実行します。S3 Tables は Apache Iceberg 標準をサポートしているため、データは AWS の他のサービスを含むさまざまなクエリ手段で容易に参照できます。</p>
