# Amazon SageMaker Unified Studio が IAM ベースのドメインのカタログにカスタム資産タイプを追加します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月09日 16:04
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/smus-custom-asset-types-iam/](https://aws.amazon.com/about-aws/whats-new/2026/07/smus-custom-asset-types-iam/)

## 概要

<p>Amazon SageMaker Unified Studio は現在、IAMベースのドメイン用のカスタム資産タイプをサポートしています。カスタム資産タイプを使うと、ドメイン管理者は SageMaker Unified Studio 内で任意の形式の資産をカタログ化でき、たとえば Amazon S3 に保存された医用画像ファイル、PowerBI で作成された収益ダッシュボード、サードパーティプラットフォームが生成したPDFの研究レポートなどが対象になります。カスタム資産タイプを導入することで、基盤となる形式に関係なくすべての資産を SageMaker のカタログに取り込み、チームは検索・発見・購読を行えるようになります。
