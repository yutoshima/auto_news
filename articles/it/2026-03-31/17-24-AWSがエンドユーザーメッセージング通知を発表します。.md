# AWSがエンドユーザーメッセージング通知を発表します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月31日 17:24
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-end-user-messaging-notify/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-end-user-messaging-notify/)

## 概要

企業はワンタイムパスコード（OTP）を送信したいと考えています。OTPは顧客が自分の身元を最も簡単かつ迅速に確認できる方法であることが多いためです。しかし、多くの企業は、電話番号の取得、キャリア登録の完了、送信者IDの設定に数週や数か月かかることに驚くことがよくあります。今日、AWSはこのすべてを変える AWS End User Messaging Notify を発表します。数分のうちに、開発者は AWS が所有する電話番号と送信者IDを利用して OTP のユースケースを動かし、すぐに送信を開始できます。
