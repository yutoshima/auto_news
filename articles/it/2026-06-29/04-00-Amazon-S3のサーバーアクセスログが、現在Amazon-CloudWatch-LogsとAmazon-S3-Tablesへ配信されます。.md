# Amazon S3のサーバーアクセスログが、現在Amazon CloudWatch LogsとAmazon S3 Tablesへ配信されます。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月29日 04:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-s3-cloudwatch-logs-tables/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-s3-cloudwatch-logs-tables/)

## 概要

<p>Amazon S3は現在、サーバーアクセスログをAmazon CloudWatch Logsへ配信する機能をサポートしており、即座のクエリ実行、アラーム、アカウント間・リージョン間の集約、アクセスログデータのAWS Key Management Service（KMS）による暗号化を実現します。さらに、追加のストレージコストなしで、Apache Iceberg形式のAmazon S3 Tablesへログをミラーリングすることもできます。これらの新しい配信パスは、従来のS3汎用バケットへのサーバーアクセスログの無料配信を補完し、ログの取り扱いにおける柔軟性を高めます。</p>
