# Amazon EC2 I8ge インスタンスがヨーロッパ（アイルランド） AWSリージョンで一般提供開始となりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月05日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-I8ge-expansion-ireland/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-I8ge-expansion-ireland/)

## 概要

Amazon Web Services（AWS）は、欧州（アイルランド）リージョンでの Amazon EC2 I8ge インスタンスの提供を発表します。大規模なストレージ I/O 集約ワークロードに適したこの新しいインスタンスは、5世代 Intel Xeon Scalable プロセッサを搭載し、全コア・ターボの周波数は3.2 GHzで、既存の I3en インスタンスと比べて計算性能を最大で40%、価格性能を最大で20%改善します。  
I8ge インスタンスは、最大で 120TB のローカル NVMe ストレージ密度を提供します—これは利用可能な最高水準です。
