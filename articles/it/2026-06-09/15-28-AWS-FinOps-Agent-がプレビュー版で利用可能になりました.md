# AWS FinOps Agent がプレビュー版で利用可能になりました

## 概要

本日、AWSは「AWS FinOps Agent」のプレビューを発表します。FinOps実務者とエンジニアリングチームのための最先端エージェントで、コストに関する質問に答え、最適化の機会を提示し、コストの異常を自動で調査し、あなたが定義したスケジュールで繰り返しのFinOpsワークフローを実行します。

AWS FinOps Agentを使えば、AWSコストについて質問を投げかけ、財務・エンジニアリングチーム向けのクラウドコストレポートを作成できます。エージェントは適正規模化（ rightsizing ）やアイドル状態の検出などを表示します。
