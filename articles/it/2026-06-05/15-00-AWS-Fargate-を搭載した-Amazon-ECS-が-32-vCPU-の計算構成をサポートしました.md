# AWS Fargate を搭載した Amazon ECS が 32 vCPU の計算構成をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月05日 15:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ecs-fargate-32vcpu](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ecs-fargate-32vcpu)

## 概要

Amazon Elastic Container Service (Amazon ECS) with AWS Fargateは、32vCPUの計算構成をサポートするようになり、より柔軟性とパフォーマンスを備えた demanding なアプリケーションの実行を可能にします。AWS Fargateは、Linux上のx86系およびARM系ワークロード向けに、60 GiB、120 GiB、または244 GiBのメモリ構成を持つ32vCPUタスクを提供します。これらの新しいタスクサイズは、Amazon ECSの高性能計算機能、大規模なデータ処理、AI推論などのユースケースをサポートする能力を拡張します。
