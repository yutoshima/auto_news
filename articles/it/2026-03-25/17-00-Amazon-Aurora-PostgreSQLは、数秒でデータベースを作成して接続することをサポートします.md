# Amazon Aurora PostgreSQLは、数秒でデータベースを作成して接続することをサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月25日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-aurora-postgresql-database/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-aurora-postgresql-database/)

## 概要

<p>Amazon Aurora PostgreSQL は、新しい「エクスプレス設定」でクラスターを作成する体験を提供します。これにより、数秒で Aurora Serverless データベースを作成・クエリできます。事前設定済みの設定を用いることで、初期設定を迅速化し、最初のクエリまでの時間を短縮します。作成時に一部の設定を変更する柔軟性があり、ほとんどの他の設定は後から調整できます。</p>
<p>エクスプレス設定を用いて作成された Aurora クラスターは、仮想プライベートクラウド（VPC）ネットワークの外部に配置されます。</p>
