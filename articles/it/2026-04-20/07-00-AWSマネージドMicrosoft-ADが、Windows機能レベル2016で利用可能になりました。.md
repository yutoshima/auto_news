# AWSマネージドMicrosoft ADが、Windows機能レベル2016で利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月20日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-managed-microsoft-ad-2016-functional-level/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-managed-microsoft-ad-2016-functional-level/)

## 概要

本日から、すべての AWS Directory Service for Microsoft AD（AWS Managed Microsoft AD）ディレクトリが Windows ファンクショナルレベル 2016 で動作します。既存の AWS Managed Microsoft AD ディレクトリには Windows ファンクショナルレベル 2016 へのアップグレードが自動的に適用されています。ファンクショナルレベルのアップグレードには、認証機構の強化と特権アクセス管理のセキュリティ向上が含まれており、クラウド上の Active Directory 基盤をより適切に保護するのに役立ちます。
