# AWS Transformは現在、顧客が所有するアーティファクトストアをサポートしています

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月14日 19:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-transform-customer-owned-artifact/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-transform-customer-owned-artifact/)

## 概要

<AWS Transform>は、評価、移行、近代化をAI対応の1つの体験にまとめ、企業の全ての変革プロセスを案内します。本日、AWSは顧客所有のAmazon S3バケットのサポートを発表し、変革アーティファクトの保存場所とセキュリティ管理を顧客が完全にコントロールできるようにします。

このリリースにより、独自のS3バケットを設定し、任意でアーティファクトをご自身のAWS KMSキーで暗号化し、アクセスポリシーを管理できます。
