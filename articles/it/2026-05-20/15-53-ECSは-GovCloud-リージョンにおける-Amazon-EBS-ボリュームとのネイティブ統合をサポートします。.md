# ECSは GovCloud リージョンにおける Amazon EBS ボリュームとのネイティブ統合をサポートします。

## 概要

<p><a href="https://aws.amazon.com/ecs/">Amazon Elastic Container Service</a>（ECS）は、AWS GovCloud Regions のコンテナに Amazon Elastic Block Store（EBS）ボリュームをマウントすることを現在サポートしています。この機能により、ETLジョブ、メディアのトランスコーディング、ML推論ワークロードなど、サーバーレスコンテナを使用したストレージおよびデータ集約型アプリケーションのデプロイが容易になります。<br /> EBSタスクアタッチメントを使用すると、各新しい ECS タスクごとに ECS が EBSボリュームのプロビジョニング、管理、およびデプロビジョニングを行えるようになります。
