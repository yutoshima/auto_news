# Amazon EC2 R8i および R8i-flex インスタンスが AWS GovCloud (US-East) リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月26日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-r8i-r8i-flex-govcloud-east/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-r8i-r8i-flex-govcloud-east/)

## 概要

本日より、Amazon Elastic Compute Cloud（Amazon EC2）のR8iおよびR8i-flexインスタンスが、AWS GovCloud（US-East）リージョンで利用可能になります。これらのインスタンスは、AWSのためだけに提供されるカスタムIntel Xeon 6プロセッサを搭載しており、クラウド環境における同等のIntelプロセッサの中で最高のパフォーマンスと最速のメモリ帯域幅を実現します。R8iおよびR8i-flexインスタンスは、前世代のIntelベースのインスタンスと比較して、価格対性能が最大で15%向上し、メモリ帯域幅は2.5倍高くなっています。
