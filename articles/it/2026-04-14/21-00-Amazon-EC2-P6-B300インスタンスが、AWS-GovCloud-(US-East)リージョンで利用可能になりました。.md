# Amazon EC2 P6-B300インスタンスが、AWS GovCloud (US-East)リージョンで利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月14日 21:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/ec2-p6-b300-govcloud-us-east/](https://aws.amazon.com/about-aws/whats-new/2026/04/ec2-p6-b300-govcloud-us-east/)

## 概要

本日より、Amazon Elastic Cloud Compute（Amazon EC2）P6-B300インスタンスがAWS GovCloud（US-East）リージョンで利用可能になります。P6-B300インスタンスは、8つのNVIDIA Blackwell Ultra GPUを搭載し、2.1 TBの高帯域幅GPUメモリ、6.4 TbpsのEFAネットワーキング、300 Gbpsの専用ENAスループット、そして4 TBのシステムメモリを提供します。P6-B300インスタンスは、P6-B200インスタンスと比較してネットワーク帯域幅2倍、GPUメモリ容量1.5倍、GPU TFLOPS（FP4、非稀少性の場合）1.5倍を実現しており、適しています。
