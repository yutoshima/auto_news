# AWS BackupはAmazon S3のバックアップコピーの性能を向上させます。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月25日 10:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-backup-amazon-s3-copy-enhancement/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-backup-amazon-s3-copy-enhancement/)

## 概要

AWS Backupは、変更追跡を強化したことで、数百万のオブジェクトを持つバケットやバックアップコピー間の変更率が低い場合、S3バックアップコピーを最大8倍の速度で実行します。この改善により、宛先アカウントまたはリージョン内のすべてのオブジェクトをスキャンする必要がなくなるため、S3バックアップのアカウント間・AWSリージョン間のコピーに要する時間を短縮します。

この改善により、AWS Backupはオブジェクトイベントを発生時に記録するため、コピー処理が速くなり、処理時間が短縮されます。
