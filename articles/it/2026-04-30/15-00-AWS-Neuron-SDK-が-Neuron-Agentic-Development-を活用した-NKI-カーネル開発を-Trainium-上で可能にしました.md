# AWS Neuron SDK が Neuron Agentic Development を活用した NKI カーネル開発を Trainium 上で可能にしました

- **公開日時**: 2026年04月30日 15:00

## 概要

AWS Neuronは、AIコーディングアシスタントの開発を加速させるためのエージェントとスキルのオープンソース集であるNeuron Agentic Development機能を発表します。これにより、AWS Trainiumと AWS Inferentia上の開発を促進します。初回リリースは、Neuron Kernel Interface（NKI）カーネル開発のためのエージェント型コーディング機能を提供し、作成からプロセスまでのワークフローをカバーします。
