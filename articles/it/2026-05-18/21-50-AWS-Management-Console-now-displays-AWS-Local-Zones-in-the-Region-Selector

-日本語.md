# AWS Management Console now displays AWS Local Zones in the Region Selector

 日本語訳: AWS マネジメントコンソールは現在、リージョンセレクターに AWS Local Zones を表示します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月18日 21:50

## 概要

本日、AWSは AWS マネジメントコンソールのリージョンセレクターに AWS Local Zones を追加し、AWS グローバルインフラストラクチャ全体で統一されたエクスペリエンスを提供します。AWS Local Zones は現在、コンソールの最上部ナビゲーションにある AWS リージョンと並んで表示され、顧客が AWS Local Zones のリソース管理ページへすばやく移動できるよう、操作性が向上します。

リージョンセレクターの Local Zones タブを選択すると、オプトイン済みのすべての AWS Local Zones が表示されます。
