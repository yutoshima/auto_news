# AWS Security Hubは、統合セキュリティ管理をMicrosoft Azureにも拡張します。

## 概要

本日、AWSは AWS Security Hub が Microsoft Azure リソースを監視するようになったことを発表します。これにより、リスク分析、クラウドセキュリティポスチャ管理、脆弱性管理、セキュリティ対応管理を両方のクラウドにまたがって拡張します。AWSおよびAzureでワークロードを実行している多くの顧客は、それぞれの環境で個別のセキュリティツールを運用せざるを得ず、リスクを全体的に優先順位付けしたり、一貫した対応を行うことが難しくなっていました。Security Hub は now 1つの統合された体験を提供し、検出と対応を行えるようになります。
