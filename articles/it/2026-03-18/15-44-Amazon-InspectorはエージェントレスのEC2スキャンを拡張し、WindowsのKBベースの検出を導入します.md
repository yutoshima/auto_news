# Amazon InspectorはエージェントレスのEC2スキャンを拡張し、WindowsのKBベースの検出を導入します

- **カテゴリ**: it
- **公開日時**: 2026年03月18日 15:44

## 概要

<p>Amazon Inspectorは、エージェント不要のEC2スキャンを拡張し、検出範囲を強化しました。エージェントを必要とせずにWindowsオペレーティングシステムの脆弱性スキャンを新たにサポートすることで、セキュリティチームやIT管理者は、WordPress、Apache HTTP Server、Pythonパッケージ、Ruby gemsなどのソフトウェアやアプリケーション、さらにはWindows OSの脆弱性を、エージェントレススキャンを通じてより広範囲に検出できるようになりました。</p>
