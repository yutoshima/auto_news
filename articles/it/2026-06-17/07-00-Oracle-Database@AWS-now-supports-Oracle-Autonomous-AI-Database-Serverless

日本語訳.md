# Oracle Database@AWS now supports Oracle Autonomous AI Database Serverless

## 概要

<p>Oracle Database@AWSは現在、Exadataインフラストラクチャ上でパッチ適用、チューニング、スケーリングを自動的に処理する完全管理型のOracleデータベースサービス「Oracle Autonomous AI Database Serverless（ADB-S）」をサポートしています。ADB-SはAWS Marketplaceの公開オファーとプライベートオファーの両方で提供され、Bring Your Own License（BYOL）およびLicense Includedオプションに対応しています。<br /> <br /> ADB-Sを使用すると、AWSマネジメントコンソール、AWS CLI、またはAWS APIから直接Oracle Autonomous AI Databaseをプロビジョニングできます。
