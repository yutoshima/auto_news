# Amazon Bedrock AgentCore Identity で自分の秘密情報を AWS Secrets Manager で持ち込むことが可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月01日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/agentcore-identity-secrets-manager/](https://aws.amazon.com/about-aws/whats-new/2026/06/agentcore-identity-secrets-manager/)

## 概要

<p>Amazon Bedrock AgentCore Identity は現在、顧客が既存の AWS Secrets Manager の secret ARN を直接 AgentCore Identity Credential Providers で参照できるようにします。<br /> <br /> 以前は、AgentCore Identity はサービス管理された Secret アプローチを採用しており、秘密はサービス側で顧客の代わりに作成・管理されていました。このアプローチでは、作成時のリソースタグの適用、顧客管理キー（CMK）での秘密の暗号化、その他の組織的設定の適用が顧客に制限されていました。
