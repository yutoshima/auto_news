# AWS Compute Optimizer は、追加のパフォーマンス指標を用いて EBS ボリュームの推奨を強化します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月18日 18:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-compute-optimizer-enhances-ebs-recommendations/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-compute-optimizer-enhances-ebs-recommendations/)

## 概要

Compute Optimizerは、Amazon EBSボリュームのリサイズ提案を提供する際に、IOPSとスループットの急増をより可視化できるようになりました。Compute Optimizerは、2つの追加のAmazon CloudWatch指標であるVolumeIOPSExceededCheckとVolumeThroughputExceededCheckを分析します。これらは、ワークロードが任意の1分間にボリュームの提供性能を超えてIOPSまたはスループットを継続的に引き上げようとしたかどうかを報告します。これらのシグナルを組み込むことで、Compute Optimizerは適切なリソースを見極めるのに役立ちます。
