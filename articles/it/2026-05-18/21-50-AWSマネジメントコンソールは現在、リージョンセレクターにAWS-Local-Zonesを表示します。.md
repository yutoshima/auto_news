# AWSマネジメントコンソールは現在、リージョンセレクターにAWS Local Zonesを表示します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月18日 21:50
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-local-zones-region-selector/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-local-zones-region-selector/)

## 概要

今日、AWSはAWS Management Consoleのリージョン選択にAWS Local Zonesを追加し、AWSのグローバルインフラストラクチャ全体で統一されたエクスペリエンスを提供します。AWS Local Zonesは現在、ConsoleのトップナビゲーションでAWS Regionsと並んで表示され、顧客がAWS Local Zonesのリソースを管理するためのConsoleページへすばやく移動できるようになります。

これで、リージョン選択のLocal Zonesタブを選択すると、オプトイン済みのすべてのAWS Local Zonesが表示されます。
