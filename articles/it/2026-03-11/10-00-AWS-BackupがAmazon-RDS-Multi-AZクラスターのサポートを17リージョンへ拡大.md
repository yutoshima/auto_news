# AWS BackupがAmazon RDS Multi-AZクラスターのサポートを17リージョンへ拡大

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月11日 10:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-backup-expands-amazon-rds-multi-az-clusters-17-regions/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-backup-expands-amazon-rds-multi-az-clusters-17-regions/)

## 概要

RDSのマルチAZクラスタ用 AWS Backup が、追加で17のAWSリージョンで利用可能になりました。対象リージョンは、アジア太平洋（ムンバイ、オオサカ、ソウル、香港、ジャカルタ、ハイデラバード、メルボルン、マレーシア）、ヨーロッパ（ロンドン、パリ、ミラノ、チューリッヒ、スペイン）、南米（サンパウロ）、アフリカ（ケープタウン）、カナダ（中部）、そしてカナダ西部（カルガリー）です。

この拡張により、AWS Backup の包括的なデータ保護機能（自動ライフサイクル管理や AWS Backup Vault Lock による改ざん不可バックアップを含む）が強化されます。
