# Amazon CloudWatch、Query Studio PreviewでPromQLクエリを導入

## 概要

<p>Amazon CloudWatch は公開プレビューで Query Studio を発表します。これは、CloudWatch にネイティブな PromQL クエリを初めて組み込んだ、統一されたクエリと可視化のエクスペリエンスです。Query Studio は PromQL と CloudWatch Metric Insights を1つのインターフェースに統合し、コンソール間を切り替えることなく、好みの言語を使って AWS 提供メトリクスと OpenTelemetry メトリクスをクエリできるようにします。<br /> <br /> Query Studio は autocomplete を備えたビジュアルフォームビルダーと、構文ハイライトを備えたコードエディタを提供します。
