# Amazon Lightsail CDN ディストリビューションは、IPv6 専用インスタンスをオリジンとしてサポートするようになりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月18日 18:49
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-lightsail-cdn-ipv6/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-lightsail-cdn-ipv6/)

## 概要

<p>Amazon Lightsail コンテンツデリバリーネットワーク（CDN）ディストリビューションは、開始済みの IPv6 専用インスタンスをオリジンとしてサポートするようになりました。この機能により、グローバルに低遅延かつ高い転送速度で Lightsail CDN ディストリビューションを通じてコンテンツを配信するために、IPv6 専用インスタンスを顧客が利用できるようになります。今回のリリースにより、顧客は費用対効果の高い IPv6 専用インスタンス上でウェブサイトやアプリケーションを実行しつつ、IPv6 に対応していないネットワークのエンドユーザーを含むすべてのユーザーへシームレスにコンテンツを提供できます。
