# AWS Elastic Beanstalk のコンソールが、Logs タブで CloudWatch Logs を統合しました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月11日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/elastic-beanstalk-cloudwatch-logs/](https://aws.amazon.com/about-aws/whats-new/2026/06/elastic-beanstalk-cloudwatch-logs/)

## 概要

AWS Elastic Beanstalkは現在、Elastic Beanstalkコンソールの環境「Logs」タブにCloudWatch Logsの統合機能を直接提供しています。従来は、環境に関連するロググループとログストリームをCloudWatchコンソールに移動して探す必要がありました。今回のリリースにより、Elastic Beanstalkコンソールを離れることなくCloudWatchのログイベントを閲覧できます。

Logsタブには、環境がログをストリーミングするロググループや、一致するロググループが表示されます。
