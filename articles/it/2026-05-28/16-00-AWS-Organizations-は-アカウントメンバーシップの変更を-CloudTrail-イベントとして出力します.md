# AWS Organizations は アカウントメンバーシップの変更を CloudTrail イベントとして出力します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月28日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-organizations-cloudtrail/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-organizations-cloudtrail/)

## 概要

<p>AWS Organizations は、アカウントが組織に参加・離脱するたびに、管理アカウントへ CloudTrail イベントを自動で出力するようになりました。新しいイベントである AccountJoinedOrganization および AccountDepartedOrganization は、セキュリティチームやクラウド管理者に対して、組織内のメンバーシップ変更をより詳しく可視化する手段を提供します。これにより、これまで見逃されがちだった不正な行為や潜在的なセキュリティインシデントの検知に役立ちます。</p>
