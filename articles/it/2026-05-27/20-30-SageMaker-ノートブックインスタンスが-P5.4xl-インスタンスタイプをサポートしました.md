# SageMaker ノートブックインスタンスが P5.4xl インスタンスタイプをサポートしました

## 概要

Amazon EC2 P5.4xl インスタンスを SageMaker ノートブックインスタンスで一般提供開始することをお知らせします。  
Amazon EC2 P5.4xl インスタンスは NVIDIA H100 Tensor Core GPU 搭載で、ディープラーニング（DL）および高性能コンピューティング（HPC）アプリケーション向けに高いパフォーマンスを提供します。前世代の GPU ベース EC2 インスタンスと比較して、ソリューションの立ち上がり時間を最大で約4倍まで加速させ、コストを削減します。
