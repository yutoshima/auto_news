# AWS HealthOmicsはキャンセルされたワークフロー実行のキャッシュをサポートするようになりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月11日 19:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-healthomics-caching-cancelled-runs/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-healthomics-caching-cancelled-runs/)

## 概要

AWS HealthOmicsは現在、キャンセルされた実行の完了済みタスク出力をキャッシュできるようになり、出力を再利用して以前完了したタスクの再計算を避けることができます。キャッシュを有効にして実行をキャンセルした場合、HealthOmicsは自動的に完了済みタスクの出力を顧客のS3バケットに保存し、キャンセル地点からの再開を可能にします。 <a href="https://aws.amazon.com/healthomics/">AWS HealthOmics</a> は HIPAA適格なサービスで、医療およびライフサイエンス分野を支援します。
