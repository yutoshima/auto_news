# Amazon EC2 High Memory U7i インスタンスが追加のリージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月23日 00:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-high-memory-u7i/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-high-memory-u7i/)

## 概要

Amazon EC2 High Memory U7in-16TB インスタンス（u7in-16tb.224xlarge）が AWS サウパウロ（南米）リージョンで利用可能になり、U7in-24TB インスタンス（u7in-24tb.224xlarge）は AWS ヨーロッパ（アイルランド）リージョンで利用可能になりました。U7i インスタンスは AWS の第7世代に属し、カスタム第四世代 Intel Xeon Scalable プロセッサー（Sapphire Rapids）で動作します。U7in-16TB インスタンスは DDR5 メモリを 16 TiB、U7in-24TB インスタンスは DDR5 メモリを 24 TiB 搭載しており、顧客がスケールアウトを促進できるようになっています。
