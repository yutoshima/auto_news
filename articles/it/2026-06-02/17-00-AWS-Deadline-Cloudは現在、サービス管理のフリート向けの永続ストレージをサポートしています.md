# AWS Deadline Cloudは現在、サービス管理のフリート向けの永続ストレージをサポートしています

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月02日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/deadline-cloud/persistent-storage](https://aws.amazon.com/about-aws/whats-new/2026/06/deadline-cloud/persistent-storage)

## 概要

<p>AWS Deadline Cloud は現在、サービス管理のフリート（SMF）の永続ストレージをサポートしており、ワーカーのライフサイクルイベントをまたいでデータを維持できます。AWS Deadline Cloud は、可視効果、アニメーション、製品設計、シミュレーション、ゲームなどのクラウド上で計算量の多いワークロードをチームで実行するための完全マネージドサービスです。<br /><br /> 以前は、Deadline Cloud の SMF ワーカーは断続的ストレージのみに依存しており、ワーカーごとにソフトウェアやアセットを再インストールする必要がありました。
