# AWS Client VPNが4つの追加AWSリージョンで提供可能に

## 概要

AWS Client VPN が新たに4つの地域で利用可能になりました：カナダ西部（カルガリー）、メキシコ（中央部）、およびアジア太平洋地域のニュージーランドと台北の2拠点です。この完全に管理されたサービスにより、リモート勤務者を AWS 内のリソースやオンプレミスネットワークへ安全につなぐことができます。

AWS Client VPN はハードウェアVPNアプライアンスや複雑な運用管理を不要にし、従量課金モデルで提供されます。組織はVPN接続を一元的なコンソールで容易に管理・監視できます。
