## 概要

Amazon ElastiCache は、以下の AWS リージョンで T4g ノードタイプをサポートするようになりました： Africa (Cape Town)、Asia Pacific (Jakarta)、Asia Pacific (Osaka)、AWS GovCloud (US-East)、および AWS GovCloud (US-West)。T4g ノードは AWS Graviton2 プロセッサによって動作し、基礎的な CPU パフォーマンスを提供するとともに、いつでも CPU 使用率をバーストできる能力を備えているため、使用量の一時的な急増が発生するアプリケーションに最適です。
