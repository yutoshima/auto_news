# AWS Lambda の関数が、S3 ファイルシステムとして Amazon S3 バケットをマウントできるようになりました。S3 Files 付き

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月21日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-lambda-amazon-s3/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-lambda-amazon-s3/)

## 概要

AWS Lambda は現在、Amazon S3 Files をサポートしており、Lambda 関数が Amazon S3 バケットをファイルシステムとしてマウントし、データを処理のためにダウンロードすることなく、標準的なファイル操作を実行できるようになりました。Amazon EFS を使用して構築された S3 Files は、ファイルシステムのパフォーマンスと操作性を、S3 の拡張性・耐久性・コスト効率と結び付けています。複数の Lambda 関数が同じ S3 Files ファイルシステムに接続できます。
