# AWS Glue Data Catalog は現在、ビジネスコンテキストとセマンティック検索をサポートしています（プレビュー）

## 概要

本日、AWSは AWS Glue Data Catalog のビジネス文脈と意味検索のプレビューを発表します。意味的な意味を通じてデータを発見・理解するのに役立ちます。S3 テーブルをバックエンドに持つものを含む Glue Data Catalog のテーブルを、用語集とカスタムメタデータフィールドで強化できるようになります。また、データに関する追加の文脈をエージェントに指示する「スキル」をカタログに追加することもできます。技術的メタデータとともにビジネス文脈がインデックスされることで、新しい Glue Search API を使って検索できます。
