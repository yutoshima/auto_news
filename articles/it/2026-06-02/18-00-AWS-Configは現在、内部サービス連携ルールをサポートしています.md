# AWS Configは現在、内部サービス連携ルールをサポートしています

## 概要

<p>AWS Configは現在、内部のサービス連携ルールをサポートしており、AWSサービスがAWS Configが管理するルールを使用してAWSリソースの構成を評価できるようになりました。内部のサービス連携ルールは、既存のサービス連携レコーダー機能を拡張し、AWS Security Hub CSPMなどのAWSサービスがサービス固有の機能のためのルール評価をデプロイおよび管理できるようにします。<br /> <br /> 内部のサービス連携ルールを使用すると、AWSサービスはAWS Configが管理するルールを活用して、統合されたセキュリティとコンプライアンスを提供できます。
