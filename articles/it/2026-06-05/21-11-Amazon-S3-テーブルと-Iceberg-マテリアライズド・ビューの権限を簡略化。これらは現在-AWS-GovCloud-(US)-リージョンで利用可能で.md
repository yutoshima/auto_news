# Amazon S3 テーブルと Iceberg マテリアライズド・ビューの権限を簡略化。これらは現在 AWS GovCloud (US) リージョンで利用可能です。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月05日 21:11
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/gdc-s3tables-simplified-permissions-in-aws-govcloud/](https://aws.amazon.com/about-aws/whats-new/2026/06/gdc-s3tables-simplified-permissions-in-aws-govcloud/)

## 概要

<p>AWS Glue Data Catalogは現在、Amazon S3テーブルとApache Icebergのマテリアライズドビューに対するAWS IAMベースの認可をサポートします。IAMベースの認可を使用すると、ストレージ、カタログ、クエリエンジン全体の必要な権限を1つのIAMポリシーで定義できます。<br /><br />この機能により、S3テーブルやマテリアライズドビューをAmazon Athena、Amazon EMR、Amazon Redshift、AWS Glueなどの任意のAWS Analyticsサービスと統合する作業が簡素化されます。さらに、AWS Lake Formationのオプションにも参加できます。
