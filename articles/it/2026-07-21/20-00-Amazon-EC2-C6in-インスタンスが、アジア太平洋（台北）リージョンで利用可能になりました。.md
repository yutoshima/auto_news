# Amazon EC2 C6in インスタンスが、アジア太平洋（台北）リージョンで利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月21日 20:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-c6in/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-c6in/)

## 概要

本日より、Amazon Elastic Compute Cloud (Amazon EC2) の C6in インスタンスが AWS アジア太平洋（台北）で提供開始されます。第六世代のネットワーク最適化インスタンスで、3世代目 Intel Xeon Scalable プロセッサを搭載し、<a href="https://aws.amazon.com/ec2/nitro/">AWS Nitro System</a> 上に構築されています。最大で 200Gbps のネットワーク帯域を実現し、同等の第五世代インスタンスと比べてネットワーク帯域が 2 倍になります。<br />  &nbsp;<br /> お客様は C6in インスタンスを利用して、ペーストスケールを拡張できます。
