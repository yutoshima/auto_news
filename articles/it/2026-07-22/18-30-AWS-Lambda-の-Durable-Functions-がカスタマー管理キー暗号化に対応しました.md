# AWS Lambda の Durable Functions がカスタマー管理キー暗号化に対応しました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月22日 18:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/durablefunctions-cmk/](https://aws.amazon.com/about-aws/whats-new/2026/07/durablefunctions-cmk/)

## 概要

AWS Lambdaの耐久性関数が、AWS KMS（AWS Key Management Service）の顧客管理キーを使用した耐久実行データの暗号化をサポートしました。Lambdaの耐久性関数は、状態管理を自動で行い、長時間実行される信頼性の高いワークフローをLambda関数コード内で直接構築できるようにします。実行状態の静止時暗号化はデフォルトでAWSが所有するキーを用いて行われます。今回はAWS KMSのサポートにより、カスタマー管理キーを使用した暗号化が可能になりました。
