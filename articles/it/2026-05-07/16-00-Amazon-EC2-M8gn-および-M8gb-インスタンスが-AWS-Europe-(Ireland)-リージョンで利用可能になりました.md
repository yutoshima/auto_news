# Amazon EC2 M8gn および M8gb インスタンスが AWS Europe (Ireland) リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月07日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-m8gn-m8gb-aws-europe/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-m8gn-m8gb-aws-europe/)

## 概要

本日より、Amazon Elastic Compute Cloud（Amazon EC2）M8gnおよびM8gbインスタンスがAWS Europe（アイルランド）リージョンで利用可能になります。これらのインスタンスはAWS Graviton4プロセッサを搭載し、AWS Graviton3プロセッサより最大30%高い計算性能を提供します。また、最新の第6世代AWS Nitroカードを搭載しています。M8gnインスタンスは最大600 Gbpsのネットワーク帯域幅を提供し、ネットワーク最適化EC2インスタンスの中で最も高い帯域幅を誇ります。M8gbは最大300 GbpsのEBS帯域幅を提供します。
