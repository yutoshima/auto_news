# Amazon Lightsail CDN ディストリビューションは現在、IPv6 のみを origins とするインスタンスをサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月18日 18:49
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-lightsail-cdn-ipv6/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-lightsail-cdn-ipv6/)

## 概要

<p>Amazon Lightsail のコンテンツ デリバリ ネットワーク（CDN）ディストリビューションが、IPv6 のみを起点とするインスタンスをオリジンとしてサポートしました。この機能により、IPv6 のみを搭載したインスタンスを使用して、Lightsail CDN ディストリビューションを通じて世界中に低遅延・高転送速度でコンテンツを配信できます。今回のリリースにより、コスト効果の高い IPv6 のみを搭載したインスタンス上でウェブサイトやアプリケーションを実行しつつ、IPv6 接続をまだサポートしていないネットワークの利用者を含むすべてのエンドユーザーにシームレスにコンテンツを提供できます。</p>
