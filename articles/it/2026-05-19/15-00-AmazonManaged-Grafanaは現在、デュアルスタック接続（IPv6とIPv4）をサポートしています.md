# AmazonManaged Grafanaは現在、デュアルスタック接続（IPv6とIPv4）をサポートしています

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月19日 15:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-managed-grafana-ipv6/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-managed-grafana-ipv6/)

## 概要

Amazon Managed Grafana はデュアルスタック接続に対応し、ワークスペースが IPv4 と IPv6 の両方で通信できるようになりました。デュアルスタックモードは Grafana バージョン 10.4 以降を実行しているワークスペースで利用できます。

デュアルスタック対応により、顧客は VPC 内の競合するアドレス空間を管理する必要を排除することで、ネットワークスタックを簡素化できます。IPv6 へ移行する顧客は、Grafana のワークスペースへ接続できます。
