# AWS Elemental MediaConnectがNDI®入力をサポートします

## 概要

本日から、 <a href="https://aws.amazon.com/mediaconnect/">AWS Elemental MediaConnect</a> は <a href="https://ndi.video/">NDI®（Network Device Interface）</a> をライブビデオソースとしてサポートし、放送局やライブプロダクションチームが NDI ストリームを取り込み、SRT などの下流配信向けのトランスポートストリーム出力へ変換できるようになります。NDI はライブプロダクション環境で広く採用されている IP ビデオ技術で、500 超のハードウェア製品と 400 のソフトウェアアプリケーションに対応しています。
