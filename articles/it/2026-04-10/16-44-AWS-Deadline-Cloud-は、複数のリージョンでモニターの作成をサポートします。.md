# AWS Deadline Cloud は、複数のリージョンでモニターの作成をサポートします。

- **カテゴリ**: it

## 概要

今日は、AWS Deadline Cloud が IAM Identity Center の追加設定を行うことなく、複数の AWS リージョンでモニターを作成できる機能をサポートすることを発表します。AWS Deadline Cloud は、クリエイティブチームがクラウドでレンダリングワークロードを管理・スケールするのを支援する、完全マネージドサービスです。  
これにより、既存の IAM Identity Center の設定を変更することなく、複数リージョンにまたがるモニターを備えたレンダーファームをデプロイできます。適切に配置することで、運用をより効率化できます。
