## 概要

<p>Amazon EC2 は、マネージドインスタンスの提供によってプロビジョニングされたリソースを、EC2 コンソールのビューや API のリスト操作に表示するかどうかを制御できるようになりました。</p>
<p>Amazon EC2 Managed Instances とは、Amazon EKS、Amazon ECS、AWS Lambda、Amazon WorkSpaces などの指定サービスプロバイダーによってプロビジョニング・管理されるインスタンスのことです。マネージド EC2 インスタンスおよび関連するリソース（EBS ボリューム、スナップショット、ネットワークなど）の設定・パッチ適用・ヘルス管理は AWS が担当します。</p>
