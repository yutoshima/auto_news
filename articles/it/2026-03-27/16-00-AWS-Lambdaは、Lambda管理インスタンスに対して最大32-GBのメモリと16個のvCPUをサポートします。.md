# AWS Lambdaは、Lambda管理インスタンスに対して最大32 GBのメモリと16個のvCPUをサポートします。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月27日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/lambda-32-gb-memory-16-vcpus/](https://aws.amazon.com/about-aws/whats-new/2026/03/lambda-32-gb-memory-16-vcpus/)

## 概要

AWS Lambdaは現在、Lambda Managed Instances上で実行される関数のメモリを最大32GB、vCPUを16までサポートしており、大規模なデータ処理、メディアのトランスコード、科学的シミュレーションなどの計算集約型ワークロードを、インフラを管理することなく実行できます。顧客はワークロードのリソースプロファイルに合わせて、メモリ対vCPUの比率を2:1、4:1、または8:1に設定することもできます。Lambda Managed Instancesを利用すると、マネージドされたAmazon EC2インスタンス上でLambda関数を実行できます。
