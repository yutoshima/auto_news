# S3 テーブルと Iceberg マテリアライズド・ビューの権限を簡素化しました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月17日 04:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/gdc-simplified-permissions-s3tables-iceberg-views/](https://aws.amazon.com/about-aws/whats-new/2026/03/gdc-simplified-permissions-s3tables-iceberg-views/)

## 概要

AWS Glue Data Catalogは現在、Amazon S3テーブルとApache Icebergマテリアライズドビューに対するAWS IAMベースの認可をサポートします。IAMベースの認可を使用すれば、ストレージ、カタログ、クエリエンジン全体の必要な権限を1つのIAMポリシーで定義できます。 

この機能により、S3テーブルやマテリアライズドビューをAmazon Athena、Amazon EMR、Amazon Redshift、AWS Glueを含む任意のAWS Analyticsサービスと統合する作業が容易になります。また、オプトインしてAWS Lake Formationを利用することも可能です。
