# AWS、EFAを用いたNIXLのサポートを追加し、LLM推論を大規模に加速

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月19日 18:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-support-nixl-with-efa/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-support-nixl-with-efa/)

## 概要

AWSは、Elastic Fabric Adapter（EFA）とNVIDIA Inference Xfer Library（NIXL）をサポートすることを発表し、Amazon EC2上で分散された大規模言語モデル（LLM）推論を加速します。この統合により、KVキャッシュのスループット増加、トークン間遅延の低減、KVキャッシュメモリ利用の最適化という3つの主要な改善を通じて、分散推論サービングが強化されます。  
NIXL with EFA は、プリフィル（prefill）ノードとデコード（decode）ノード間での KVキャッシュ転送を高スループット・低レイテンシで実現し、それを可能にします。
