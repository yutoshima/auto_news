# AWS Parallel Computing Serviceは現在Slurm 25.11をサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月23日 20:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-pcs-slurm-25-11/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-pcs-slurm-25-11/)

## 概要

AWS Parallel Computing Service（AWS PCS）は、Slurm バージョン 25.11 をサポートし、Prometheus 互換の OpenMetrics エンドポイントに対応しました。また、スケジューラ監査ログなどの新しいログタイプを追加しています。  
この Slurm 25.11 のリリースには、ノードの問題によって影響を受けたジョブを最高優先度で自動的に再スケジュールできる「迅速な再キュー（expedited re-queue）」機能が導入され、ワークロードの回復を迅速化します。ジョブ、ノード、スケジューリングのリアルタイム可視化のために、新しい OpenMetrics エンドポイントを有効化できます。
