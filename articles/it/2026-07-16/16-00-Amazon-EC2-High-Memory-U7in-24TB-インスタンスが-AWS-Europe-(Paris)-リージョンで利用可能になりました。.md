# Amazon EC2 High Memory U7in-24TB インスタンスが AWS Europe (Paris) リージョンで利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月16日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-high-memory-europe/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-high-memory-europe/)

## 概要

Amazon EC2 High Memory U7in-24TB インスタンス（u7in-24tb.224xlarge）は、AWS Europe（パリ）リージョンで利用可能になりました。U7i インスタンスは AWS の第7世代に属し、カスタムの第4世代 Intel Xeon Scalable プロセッサ（Sapphire Rapids）を搭載しています。U7in-24TB インスタンスは 24 TiB の DDR5 メモリを提供し、急成長するデータ環境でトランザクション処理のスループットを迅速にスケールさせることができます。U7i インスタンスは、既存の U-1 インスタンスと比べて最大で約45%のコスト性能向上を実現します。
