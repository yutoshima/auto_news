# AWS Lambda Managed Instances が Rust をサポートするようになりました

- **カテゴリ**: it

## 概要

AWS Lambda Managed Instancesは現在Rustをサポートしており、Rustベースの高性能な関数をLambdaが管理するAmazon EC2インスタンス上で実行しつつ、Lambdaの運用のシンプルさを維持できます。この組み合わせにより、サーバー管理の複雑さを抱えることなく、性能重視のアプリケーションをこれまで以上に容易に実行できるようになります。  
Lambda Managed Instancesは、最新世代のプロセッサや帯域幅の高い特殊用途のコンピュート構成を含む、Lambda開発者向けの高度な計算構成へのアクセスを提供します。
