# AWS Control Towerのアカウントファクトリ for Terraformは、アカウントがOU間を移動するときにカスタマイズを再適用します

## メタデータ

- **カテゴリ**: it
- **公開日時**: 2026年07月16日 04:07

## 概要

AWS Control Tower アカウントファクトリ for Terraform（AFT）は、アカウントが別の組織単位（OU）に移動した際に、そのアカウントのカスタマイズを自動的に再適用できるようになりました。従来は、登録済みアカウントをOU間で移動する際にカスタマイズの再適用を手動でトリガーする必要があり、運用の負荷や設定のずれ（コンフィグレーション・ドリフト）のリスクがありました。この機能により、AFT の展開で自動再適用を選択しておくことができ、アカウントがOU固有の設定と一貫性を保つようになります。
