# Amazon EC2 Capacity Manager は現在、タグベースのディメンションをサポートしています

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月09日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/ec2-capacity-manager-tag-based-dimensions/](https://aws.amazon.com/about-aws/whats-new/2026/04/ec2-capacity-manager-tag-based-dimensions/)

## 概要

本日より、Amazon EC2 Capacity Manager はタグベースのディメンションをサポートします。これにより、EC2リソースのタグを使用して容量指標をグルーピングおよびフィルタリングできます。EC2 Capacity Manager は、On-Demand Instances、Spot Instances、Capacity Reservations の容量使用状況を監視・最適化するのに役立ちます。今回のリリースでは、Account Name を新しい組み込みディメンションとして追加します。

最大5つのカスタムタグキー（例：environment、team、cost-center など）を有効化し、それらをビルトインディメンションと併用して使用できます。
