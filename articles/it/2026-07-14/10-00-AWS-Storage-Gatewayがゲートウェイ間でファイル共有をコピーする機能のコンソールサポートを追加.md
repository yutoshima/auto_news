# AWS Storage Gatewayがゲートウェイ間でファイル共有をコピーする機能のコンソールサポートを追加

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月14日 10:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/aws-storage-gateway-console-copy-file-shares/](https://aws.amazon.com/about-aws/whats-new/2026/07/aws-storage-gateway-console-copy-file-shares/)

## 概要

AWS Storage Gatewayは現在、Storage Gatewayコンソールから直接ゲートウェイ間でファイル共有をコピーできるようになりました。コピーを開始すると、コンソールはソース共有の設定を読み取り、互換性のある設定を保持したまま宛先ゲートウェイに新しいファイル共有を作成します。

以前は、宛先ゲートウェイで各ファイル共有を手動で再作成し、設定を一つずつ手入力する必要がありました。今はファイル共有をコピーするだけで、その設定が自動的に適用されます。
