# Amazon EC2 R6in および R6idn インスタンスが、追加のリージョンで利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月21日 20:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-r6in-r6idn/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-r6in-r6idn/)

## 概要

本日より、Amazon Elastic Compute Cloud（Amazon EC2）のR6inおよびR6idnインスタンスが、AWS Europe（パリ）および Canada（中部）リージョンで利用可能になります。第六世代のネットワーク最適化インスタンスで、3世代目Intel Xeon Scalableプロセッサを搭載し、<a href="https://aws.amazon.com/ec2/nitro/">AWS Nitro System</a>上に構築されています。これにより、従来世代のインスタンス比で最大200Gbpsのネットワーク帯域幅、ネットワーク帯域幅が2倍、パケット処理性能が最大2倍向上します。
