# Amazon ECS Express Modeは現在、カスタムタスク定義をサポートします。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月01日 16:05
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ecs-express-mode-custom-task-def/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ecs-express-mode-custom-task-def/)

## 概要

Amazon Elastic Container Service（Amazon ECS）Express Modeはカスタムタスク定義をサポートするようになりました。これにより、Express Modeのシンプルなデプロイ体験を損なうことなく、既存のECSアプリケーション設定や高度なタスクレベルのカスタマイズを利用できます。また、既存のCI/CDパイプラインやインフラストラクチャ・アズ・コード（IaC）ワークフローからタスク定義を再利用できるようになり、維持を容易にします。
