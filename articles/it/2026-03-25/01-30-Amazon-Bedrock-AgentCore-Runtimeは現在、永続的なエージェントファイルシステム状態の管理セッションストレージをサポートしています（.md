# Amazon Bedrock AgentCore Runtimeは現在、永続的なエージェントファイルシステム状態の管理セッションストレージをサポートしています（プレビュー）。

## 概要

<p>Amazon Bedrock AgentCore Runtime は現在、パブリックプレビューで管理型セッションストレージを提供しており、エージェントが停止と再開のサイクルをまたいでファイルシステムの状態を保持できるようになりました。現代のエージェントはコードを記述し、パッケージをインストールし、成果物を生成し、ファイルシステムを介して状態を管理します。これまでその作業はセッションが停止すると失われていました。管理型セッションストレージを使えば、エージェントが設定済みのマウントパスに書き込んだすべてのデータは、計算環境が終了しても自動的に保持されます。</p>
