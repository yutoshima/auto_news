# AWS Lambdaは、Lambda Managed Instances上の関数に対してスケジュールされたスケーリングをサポートします。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月12日 19:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-lambda-managed-instances/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-lambda-managed-instances/)

## 概要

AWS Lambda は現在、Lambda Managed Instances 上で動作する関数のスケジュールスケーリングを Amazon EventBridge Scheduler を使用してサポートしています。この機能により、ピーク時のパフォーマンス目標を満たすために、予想されるトラフィックに先んじて関数の容量制限を事前に調整する、1回限りまたは定期的なスケジュールを定義できます。アイドル期間中のコストを回避します。  
Lambda Managed Instances は、組み込みルーティングとロードバランシングを備えたマネージドな Amazon EC2 インスタンス上で Lambda 関数を実行できるようにします。
