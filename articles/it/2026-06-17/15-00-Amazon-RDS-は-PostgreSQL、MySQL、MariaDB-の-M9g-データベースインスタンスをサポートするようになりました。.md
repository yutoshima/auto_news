# Amazon RDS は PostgreSQL、MySQL、MariaDB の M9g データベースインスタンスをサポートするようになりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月17日 15:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-rds-postgresql-mysql-mariadb-m9g-instances/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-rds-postgresql-mysql-mariadb-m9g-instances/)

## 概要

AWS Graviton5ベースのM9gデータベース（DB）インスタンスが、Amazon Relational Database Service（RDS） for PostgreSQL、MySQL、MariaDBで一般提供開始となりました。Graviton5ベースのインスタンスは、同等サイズのGraviton4ベースインスタンスに対して、オンデマンド価格で最大30%の性能向上、最大23%の費用対性能改善を提供します（データベースエンジンによって異なります）。
