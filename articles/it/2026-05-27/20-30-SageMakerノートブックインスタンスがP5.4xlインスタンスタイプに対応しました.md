# SageMakerノートブックインスタンスがP5.4xlインスタンスタイプに対応しました

## 概要

SageMaker ノートブックインスタンスで Amazon EC2 P5.4xl インスタンスの一般提供を開始できることをお知らせします。  
Amazon EC2 P5.4xl インスタンスは NVIDIA H100 Tensor Core GPU を搭載しており、ディープラーニング（DL）およびハイパフォーマンスコンピューティング（HPC）アプリケーション向けの高いパフォーマンスを Amazon EC2 で実現します。前世代の GPU ベースの EC2 インスタンスと比較して、解決までの時間を最大約4倍速くし、コストを削減します。
