# AWS Transformは現在、顧客所有のアーティファクトストアをサポートします。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月14日 19:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-transform-customer-owned-artifact/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-transform-customer-owned-artifact/)

## 概要

<p>AWS Transformは、評価、移行、モダナイゼーションを1つのAIパワード体験に統合し、企業の全体的な変革の旅を案内します。今日、AWSは顧客が所有するAmazon S3バケットのサポートを発表し、変革アーティファクトを保存する場所とその保護方法を顧客が完全に管理できるようにします。</p>
<p>このリリースにより、自分のS3バケットを設定し、任意でAWS KMSキーを使用してアーティファクトを暗号化し、アクセスポリシーを管理できます。</p>
