# Amazon SageMaker HyperPodは現在、EFAのみのネットワークインターフェースをサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月01日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-sagemaker-hyperpod-efa-only/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-sagemaker-hyperpod-efa-only/)

## 概要

<p>Amazon SageMaker HyperPod はクラスタインスタンスグループ向けの EFA 専用ネットワークインターフェイスをサポートしました。これにより、IP ネットワーキングのための従来の Elastic Network Adapter (ENA) を使用せず、専用の Elastic Fabric Adapter (EFA) デバイスを構成できます。SageMaker HyperPod は AI/ML モデル開発のための専用インフラストラクチャで、組み込みのフォールトトレランスと自動的なクラスタ回復機能を備えた、堅牢で高性能な環境を提供します。EFA 専用になったことで、AI/ML クラスターをさらに拡張できます。</p>
