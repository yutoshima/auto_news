# Amazon CloudWatch Application Signalsは、インフラストラクチャ、ログ、およびトレースの文脈を提供し、トラブルシューティングを迅速化します。

## 概要

<p>Amazon CloudWatch Application Signals は、アプリケーションマップ上のサービスのヘルスランク付けと、サービス概要ページの新しいインフラストラクチャ、ログ、トレースのタブを導入します。これらの機能を使えば、オペレーターは健全でないサービスのトリアージを行い、基盤の計算環境、ログの断片、トレースの詳細を一つの場所で確認でき、ツールを切り替えることなく根本原因を見つけやすくなります。顧客は分散アプリケーションのヘルスを監視するために Application Signals を使用しますが、なぜを特定することが難しい場合があります。
