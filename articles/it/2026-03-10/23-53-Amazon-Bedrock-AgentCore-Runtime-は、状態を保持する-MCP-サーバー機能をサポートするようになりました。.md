# Amazon Bedrock AgentCore Runtime は、状態を保持する MCP サーバー機能をサポートするようになりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月10日 23:53
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-bedrock-agentcore-runtime-stateful-mcp/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-bedrock-agentcore-runtime-stateful-mcp/)

## 概要

Amazon Bedrock AgentCore Runtime は現在、状態保持型の Model Context Protocol (MCP) サーバ機能をサポートしており、開発者は既存のリソース、プロンプト、ツールのサポートに加えて、誘導、サンプリング、進捗通知を活用する MCP サーバを構築できるようになっています。これらの機能により、AgentCore Runtime にデプロイされた MCP サーバは、ツール実行中にユーザー入力を対話的に収集し、クライアントから LLM によって生成されたコンテンツを要求し、長時間実行される処理のリアルタイム進捗を通知することが可能になります。
