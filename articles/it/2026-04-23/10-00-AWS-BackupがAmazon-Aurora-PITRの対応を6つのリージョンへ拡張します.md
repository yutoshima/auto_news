# AWS BackupがAmazon Aurora PITRの対応を6つのリージョンへ拡張します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月23日 10:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-backup-amazon-aurora-pitr/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-backup-amazon-aurora-pitr/)

## 概要

AWS Backup の Amazon Aurora Point-in-time Recovery (PITR) のサポートが、以下の6つの追加AWSリージョンで利用可能になりました：アジア太平洋（マレーシア、タイ、台北、ニュージーランド）、カナダ西部（カルガリー）およびメキシコ（中部）。  

この拡張により、これらの新しくサポートされたリージョンの Amazon Aurora クラスターに対して、PITR のサポート付きのポリシーベースのデータ保護とリカバリが提供されます。  

AWS Backup を利用して PITR をサポートする Aurora クラスターの保護を開始するには、Aurora クラスターを追加します。
