# AWS Secrets Manager は Paddle と GitLab のマネージド外部シークレット対応を追加しました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月06日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/secrets-manager-managed-external-secrets-paddle-gitlab/](https://aws.amazon.com/about-aws/whats-new/2026/07/secrets-manager-managed-external-secrets-paddle-gitlab/)

## 概要

AWS Secrets Managerは、Paddle APIキーとGitLabアクセスTokenを含む、マネージド外部シークレット機能を拡張しました。マネージド外部シークレットは、サポート対象のサードパーティサービスとの一流の統合を提供することにより、顧客が第三者の認証情報をAWS Secrets Managerから自動的にローテーションできるようにします。  
このローンチにより、PaddleのネイティブローテーションAPIを使用してPaddle APIキーのローテーションを管理できるようになります。ローテーションAPIは、適用可能なアプリケーションに柔軟な猶予期間を提供します。
