# AWS End User Messaging RCSは現在、リッチメディアとインタラクティブなメッセージ機能をサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月30日 18:21
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-end-user-messaging-rcs/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-end-user-messaging-rcs/)

## 概要

AWS End User Messaging は、すべての22カ国でRCS対応のリッチメディアとインタラクティブメッセージングをサポートします。新しい SendRcsMessage API を使用すると、リッチカード、カルーセル、画像、動画、そして受信者がメッセージングアプリ内で直接行動を起こせるインタラクティブな提案ボタンを送信できます。

RCS メッセージの受信者は、予約の確認をタップしたり、商品カタログを閲覧したり、ウェブビューで支払いを完了したり、位置情報を共有したり、AIエージェントとやり取りしたりといったことを、アプリを離れずに行えます。
