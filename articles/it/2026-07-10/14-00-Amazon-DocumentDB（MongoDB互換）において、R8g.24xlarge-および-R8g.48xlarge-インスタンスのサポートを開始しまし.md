# Amazon DocumentDB（MongoDB互換）において、R8g.24xlarge および R8g.48xlarge インスタンスのサポートを開始しました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月10日 14:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-documentdb-r8g-24xl-48xl/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-documentdb-r8g-24xl-48xl/)

## 概要

Amazon DocumentDB（ MongoDB 互換性つき）は、R8g.24xlarge および R8g.48xlarge データベースインスタンスのサポートを開始しました。R8g インスタンスは AWS Graviton4 プロセッサと DDR5 メモリを搭載しており、顧客がより高いスループットを実現し、メモリ内でより大きな作業セットを扱えるようにします。R8g.24xlarge（96 vCPU、768 GiB メモリ）および R8g.48xlarge（192 vCPU、1,536 GiB メモリ）を用いることで、同時実行性の高いトランザクション系アプリケーションや大規模なドキュメント処理など、より要求の厳しいワークロードを実行できるようになります。
