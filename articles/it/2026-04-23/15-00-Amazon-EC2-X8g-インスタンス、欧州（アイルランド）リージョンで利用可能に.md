# Amazon EC2 X8g インスタンス、欧州（アイルランド）リージョンで利用可能に

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月23日 15:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ec2-x8g-instances-europe-ireland/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ec2-x8g-instances-europe-ireland/)

## 概要

本日より、Amazon Elastic Compute Cloud（Amazon EC2）X8g インスタンスがヨーロッパ（アイルランド）リージョンで利用可能になります。これらのインスタンスは AWS Graviton4 プロセッサによって動作し、AWS Graviton2 ベースの Amazon EC2 X2gd インスタンスと比較して最大60％のパフォーマンス向上を実現します。X8g インスタンスは総メモリを最大3 TiB搭載し、他の Graviton4 ベースのインスタンスと比べて vCPU あたりのメモリ容量が増えています。EC2 Xシリーズの中でもコストパフォーマンスが最も優れており、メモリ集約型ワークロードに最適です。
