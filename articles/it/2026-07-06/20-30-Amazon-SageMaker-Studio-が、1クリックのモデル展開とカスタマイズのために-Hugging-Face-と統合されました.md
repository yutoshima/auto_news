# Amazon SageMaker Studio が、1クリックのモデル展開とカスタマイズのために Hugging Face と統合されました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月06日 20:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/sagemaker-studio-hugging-face-integration/](https://aws.amazon.com/about-aws/whats-new/2026/07/sagemaker-studio-hugging-face-integration/)

## 概要

<p>Amazon SageMaker Studioは、Hugging Faceとの直接統合を現在サポートしており、モデルを発見してから、完全に設定されたStudio環境内でワンクリックで作業を開始できるようになりました。Hugging Face上の対応モデルを選択し、「SageMaker AIでカスタマイズ」または「SageMaker AIでデプロイ」を選ぶと、モデルが事前ロード済みでそのワークフローのページに直接遷移します。</p>
<p>従来は、モデルを発見して作業環境を作成するには、ナビゲーションをたどる必要がありました</p>
