# AWS Parallel Computing Service は、インプレースで Slurm の主要バージョンをアップグレードします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月30日 19:12
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-parallel-computing-service-upgrade/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-parallel-computing-service-upgrade/)

## 概要

AWS Parallel Computing Service（PCS）は、既存クラスター向けに管理済みのインプレース Slurm バージョンアップグレードを提供します。実行中のジョブに影響を与えることなく、クラスターを最大3つの Slurm メジャーバージョン先へ移行できます。

アップグレードするには、AWS Management Console、AWS CLI、または UpdateCluster API を使用してターゲット Slurm バージョンでクラスター設定を更新してください。PCS は、コントローラー、会計データベース、REST API を含むすべての管理される Slurm コンポーネントのアップグレードを自動的に処理します。実行中のジョブは継続します。
