# Claude Tagは現在、AWS MarketplaceのClaude Enterpriseを通じてベータ版で利用可能です。

## 概要

AnthropicはClaude Tagをローンチします。チームがすでに使っているチャネルに直接Claudeを組み込む形で、まずはSlackから開始します。Claude Tagは、本日AWS Marketplaceを通じてClaude EnterpriseにアクセスするAWSのお客様向けのベータ版として提供されています。

Claude Tagは、Claudeとチームが連携する新しい方法です。Claudeに選択したチャネルへのアクセスを許可し、任意のツール・データ・さらにはコードベースと接続します。マルチプレイヤー形式なので、チャネル内の誰でも@Claudeを呼び込み、タスクを委任できます。
