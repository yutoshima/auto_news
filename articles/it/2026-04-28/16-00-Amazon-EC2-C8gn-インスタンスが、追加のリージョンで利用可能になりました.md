# Amazon EC2 C8gn インスタンスが、追加のリージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月28日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ec2-c8gn-milan-hong-kong/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ec2-c8gn-milan-hong-kong/)

## 概要

本日より、最新世代の AWS Graviton4 プロセッサを搭載した Amazon Elastic Compute Cloud（Amazon EC2） C8gn インスタンスが、AWS Europe (Milan) および Asia Pacific (Hong Kong) リージョンで利用可能になります。新しいインスタンスは、Graviton3 ベースの Amazon EC2 C7gn インスタンスと比べて、最大で約30%の計算性能向上を提供します。Amazon EC2 C8gn インスタンスは、最新の第6世代 AWS Nitro Cards を搭載し、最大600 Gbps のネットワーク帯域幅を提供します。これはネットワーク帯域幅の中でも最高水準です。
