# Amazon EC2 R8gd インスタンスが、追加の AWS リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月26日 15:39
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-r8gd-aws-regions/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-r8gd-aws-regions/)

## 概要

Amazon Elastic Compute Cloud（Amazon EC2）R8gdインスタンスは、最大11.4 TBのローカルNVMeベースSSDブロックストレージを搭載し、US West（N. California）、Asia Pacific（Seoul、Hong Kong、Jakarta）、Africa（Cape Town）、Canada West（Calgary）AWSリージョンで利用可能になりました。これらのインスタンスはAWS Graviton4プロセッサで動作し、Graviton3ベースのインスタンスと比べて最大30％のパフォーマンス向上を実現します。I/O集約型のデータベースワークロードに対しては最大40％のパフォーマンス向上を、そして待機時間の短縮同等の効果で最大20％の高速化を提供します。
