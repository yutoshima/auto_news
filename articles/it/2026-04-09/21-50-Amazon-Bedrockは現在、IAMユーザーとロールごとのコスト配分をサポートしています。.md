# Amazon Bedrockは現在、IAMユーザーとロールごとのコスト配分をサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月09日 21:50
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/bedrock-iam-cost-allocation/](https://aws.amazon.com/about-aws/whats-new/2026/04/bedrock-iam-cost-allocation/)

## 概要

Amazon Bedrockは現在、AWS Cost and Usage Report 2.0（CUR 2.0）およびCost Explorerにおいて、IAMプリンシパル（IAMユーザーやIAMロールなど）によるコスト配分をサポートしています。これにより、Bedrockモデル推論コストをユーザー、チーム、プロジェクト、アプリケーションごとに把握・配分できます。

このリリースにより、顧客はIAMユーザーとロールに「チーム」「プロジェクト」「費用センター」などの属性をタグ付けし、それをコスト配分タグとして有効化し、Bedrockモデル推論コストを分析できるようになります。
