# Amazon Managed Grafana は現在、デュアルスタック接続（IPv6とIPv4）をサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月19日 15:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-managed-grafana-ipv6/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-managed-grafana-ipv6/)

## 概要

Amazon Managed Grafanaは現在デュアルスタック接続をサポートしており、ワークスペースがIPv4とIPv6の両方で通信できるようになりました。デュアルスタックモードはGrafanaバージョン10.4以降を実行しているワークスペースで利用可能です。

デュアルスタックのサポートにより、顧客はVPC内でのアドレス空間の重複管理を回避することでネットワークスタックを簡素化できます。IPv6へ移行する顧客は、Grafanaワークスペースへ接続できます。
