## 概要

<p>AWSは、Microsoft Windows Server 2025を搭載した新しいバンドルの提供を開始しました。これらはAmazon WorkSpaces PersonalおよびAmazon WorkSpaces Core向けに提供されます。これらのバンドルを使用すると、Windows Server 2025 WorkSpacesを起動し、最新のWindowsサーバーOS機能を活用できます。お客様は、最新のWindowsバージョンを必要とする対象のMicrosoft 365 Apps for enterpriseなどのアプリケーションを実行できます。<br /> <br /> Windows Server 2016、2019、2022を搭載したWorkSpacesのサポートはまだ続いています。</p>
