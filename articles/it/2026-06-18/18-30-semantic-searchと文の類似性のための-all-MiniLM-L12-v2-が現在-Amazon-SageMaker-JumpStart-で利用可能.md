# semantic searchと文の類似性のための all-MiniLM-L12-v2 が現在 Amazon SageMaker JumpStart で利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月18日 18:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/all-minilm-l12-v2-on-sagemaker-jumpstart/](https://aws.amazon.com/about-aws/whats-new/2026/06/all-minilm-l12-v2-on-sagemaker-jumpstart/)

## 概要

本日、AWSはAmazon SageMaker JumpStartで全ての MiniLM-L12-v2 の利用可能性を発表し、AWSのお客様が利用できるモデルのポートフォリオを拡張しました。この Sentence Transformers のモデルは文と段落を384次元の密なベクトル空間にマッピングするため、AWSのインフラ上で高品質なセマンティック検索、テキストクラスタリング、文の類似度アプリケーションを構築することを可能にします。

all-MiniLM-L12-v2 は、文や短い段落を密なベクトル表現へエンコードすることに優れています。
