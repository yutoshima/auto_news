# AWS HealthOmics が Nextflow プロファイルをサポートしました

- **カテゴリ**: it

## 概要

AWS HealthOmicsは現在Nextflowプロファイルをサポートしており、実行時に predefined の実行設定を有効化できます。Nextflowプロファイルを使用すると、再利用可能な設定を定義し、実行時にそれを選択することができ、ワークフローのソースコードを変更することなく実行設定を簡単に切り替えられます。<a href="https://aws.amazon.com/healthomics/">AWS HealthOmics</a>はHIPAA適合性を有するサービスで、医療・ライフサイエンス分野の顧客が科学的ブレークスルーを加速させるのを支援します。
