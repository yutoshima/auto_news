# Amazon CloudWatch、Amazon EKS向けのOTel Container Insights（プレビュー）を開始

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月02日 20:41
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/cloudwatch-otel-container-insights-eks/](https://aws.amazon.com/about-aws/whats-new/2026/04/cloudwatch-otel-container-insights-eks/)

## 概要

<p>Amazon CloudWatchは、Amazon EKS向けの OpenTelemetry 指標を備えた Container Insightsをパブリックプレビューで提供します。既存の Container Insights 体験を基盤とし、この機能は、広く採用されているオープンソースおよび AWS コレクタからより多くの指標を収集し、OpenTelemetry Protocol（OTLP）を使用して CloudWatch に送信することで、EKS クラスターの可視性をより深く向上させます。各指標には、Kubernetes メタデータやカスタムラベルを含む最大150件の説明的ラベルが自動的に付加されます。
