# Amazon RDS for SQL Server は、独自のメディアを使用することをサポートしています。

- **カテゴリ**: it

## 概要

Amazon Relational Database Service (Amazon RDS) for SQL Server が、Microsoft SQL Server の Bring Your Own Media（BYOM）を開始します。BYOM を利用すると、オンプレミス環境から SQL Server アプリケーションを移行したお客様は AWS のマネージドデータベースサービスを利用でき、Microsoft のライセンス移行プログラムを通じて Software Assurance を含む既存の Microsoft SQL Server ライセンスを再利用できます。
