# Amazon S3 Tables が、2つの追加の AWS リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月29日 04:00

## 概要

<p>Amazon S3 Tables は、アジア太平洋（台北）地域およびアジア太平洋（ニュージーランド）地域で利用可能になりました。</p>
<p>Amazon S3 Tables は、組み込みの Apache Iceberg サポートを備えた最初のクラウドオブジェクトストアを提供し、規模の大きい表形式データの格納を簡素化します。S3 Tables は、データレイクの成長と進化に合わせてクエリ効率を最適化し、ストレージコストを削減するために、継続的なテーブル保守を自動的に実行します。S3 Tables は Apache Iceberg 標準をサポートしているため、データは AWS のどちらのツールからも容易にクエリ可能です。</p>
