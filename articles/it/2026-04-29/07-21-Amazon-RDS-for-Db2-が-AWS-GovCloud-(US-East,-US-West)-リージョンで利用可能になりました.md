# Amazon RDS for Db2 が AWS GovCloud (US-East, US-West) リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月29日 07:21
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-rds-db2-available-aws-govcloud-us-east-us-west-regions/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-rds-db2-available-aws-govcloud-us-east-us-west-regions/)

## 概要

Amazon Relational Database Service（Amazon RDS） for Db2 が AWS GovCloud（US-East、US-West）リージョンで利用可能になりました。Db2 用の Amazon RDS は、クラウド上で Db2 データベースを設定・運用・スケールさせるのを容易にします。顧客は、最適なパフォーマンスのために自動設定されたパラメータを備えた Db2 データベースを数分でデプロイできます。Multi-AZ 構成でデータベースを設定する場合、Amazon RDS は同期レプリケーションを実行します。
