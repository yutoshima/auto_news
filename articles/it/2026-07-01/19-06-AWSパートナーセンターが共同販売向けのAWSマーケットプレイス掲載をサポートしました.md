# AWSパートナーセンターが共同販売向けのAWSマーケットプレイス掲載をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月01日 19:06
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/aws-marketplace-co-selling-support/](https://aws.amazon.com/about-aws/whats-new/2026/07/aws-marketplace-co-selling-support/)

## 概要

本日、AWSは、パートナーが自社の AWS Marketplace カタログ内の1つ以上のソリューションおよび製品リスティングを、AWS Partner Central の共同販売機会に直接関連付けられるようになったことを発表します。これまでは、共同販売用に特別に作成されたソリューションを利用する必要があり、パートナーは AWS Marketplace カタログ用のソリューションと共同販売用のソリューションを別々に管理していました。パートナーは現在、既存の AWS Marketplace リスティングを共同販売機会と関連付けることができます。
