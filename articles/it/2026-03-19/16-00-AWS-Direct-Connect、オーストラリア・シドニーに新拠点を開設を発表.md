# AWS Direct Connect、オーストラリア・シドニーに新拠点を開設を発表

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月19日 16:00

## 概要

今日は、AWSがオーストラリア・シドニーのEquinix SY5に新しい AWS Direct Connect ロケーションを開設したことを発表しました。これにより、公開AWSリージョン（中国を除く）、AWS GovCloudリージョン、および AWS Local Zones へのプライベートで直接的なネットワークアクセスをこのロケーションから確立できるようになります。ここはシドニーで4番目の AWS Direct Connect ロケーションで、AWS Direct Connect の第10拠点となります。
