## 概要

本日、AWSは KNFSD File Cache の提供を発表します。KNFSD File Cache は、オープンソースで Apache-2.0 ライセンスのソリューションで、AWS上にスケーラブルで高速なネットワークファイルシステム（NFS）キャッシュをデプロイするためのものです。KNFSD File Cache は、1つまたは複数のソースNFSサーバーのエクスポートをマウントし、オンプレミス、他のAWSアベイラビリティゾーンまたはリージョン、あるいは AWS Interconnect を介した他のクラウド（マルチクラウド）上のいずれかで稼働しているかを問わず、それらを AWS の NFS クライアントへ再エクスポートします。複数のオンプレミスファイラーや、Amazon FSx などのクラウド内ファイルシステムを前面に置くことができます。
