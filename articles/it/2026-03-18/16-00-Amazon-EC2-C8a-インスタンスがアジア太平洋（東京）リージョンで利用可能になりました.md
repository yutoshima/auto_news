# Amazon EC2 C8a インスタンスがアジア太平洋（東京）リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月18日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-r8a-instances-asia-pacific-tokyo-regions/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-r8a-instances-asia-pacific-tokyo-regions/)

## 概要

本日より、計算性能に最適化された Amazon EC2 C8a インスタンスがアジア太平洋（東京）リージョンで利用可能になります。C8a インスタンスは第5世代 AMD EPYC プロセッサ（旧コードネーム Turin）を搭載し、最大周波数は 4.5 GHz、C7a インスタンスと比較して最大で 30% のパフォーマンス向上、価格性能比も最大で 19% 向上します。  
C8a インスタンスは C7a インスタンスと比べてメモリ帯域幅を 33% 拡張しており、レイテンシーに敏感なワークロードに最適です。続く部分は省略されているようです。
