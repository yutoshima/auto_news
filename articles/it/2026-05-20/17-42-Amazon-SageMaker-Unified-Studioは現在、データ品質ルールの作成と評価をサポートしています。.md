# Amazon SageMaker Unified Studioは現在、データ品質ルールの作成と評価をサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月20日 17:42
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/smus-data-quality](https://aws.amazon.com/about-aws/whats-new/2026/05/smus-data-quality)

## 概要

Amazon SageMaker Unified Studio は現在、AWS Glue Data Quality によって強化されたデータ品質ルールの作成と評価をサポートします。データエンジニア、アナリスト、データサイエンティストは、データ品質ルールを定義し、ルールセットの評価を実行し、SageMaker Unified Studio 内で結果を直接確認できます。対象はカタログテーブルに格納されたデータ（静止データ）と Visual ETL ジョブ内のデータ転送中データ（データ在中）です。これにより、データレイクへ不良データが流入したり、下流の分析や機械学習に影響を及ぼす前にデータ品質の問題を検出できます。
