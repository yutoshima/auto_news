# Amazon SageMaker Unified Studio がノートブックのスケジューリングをサポートしました

- **カテゴリ**: it
- **公開日時**: 2026年06月03日 20:13

## 概要

<p>Amazon SageMaker Unified Studioは、ノートブックのインターフェイスから直接、外部のオーケストレーション基盤を管理することなく、ノートブックの実行をスケジュール・パラメータ設定・オーケストレーションできるようになりました。これにより、顧客はノートブックを実験段階から本番運用へ移行し、日次レポート作成・データ品質チェック・モデル再学習などの定期的なワークロードを自動化しやすくなります。</p>
<p>対話型セッションを妨げることなく、専用の計算資源上でオンデマンドのバックグラウンド実行を開始できます。</p>
