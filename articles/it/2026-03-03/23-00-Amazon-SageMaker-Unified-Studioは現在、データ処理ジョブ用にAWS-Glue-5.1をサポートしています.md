# Amazon SageMaker Unified Studioは現在、データ処理ジョブ用にAWS Glue 5.1をサポートしています

- **公開日時**: 2026年03月03日 23:00

## 概要

<Amazon SageMaker Unified Studio>は現在、<Visual ETL、ノートブック、コードベースのデータ処理ジョブ>向けに <AWS Glue 5.1> をサポートしています。SageMaker Unified Studio の AWS Glue 5.1 を使うと、データエンジニアとデータサイエンティストは Apache Spark 3.5.6、Python 3.11、Scala 2.12.18 でジョブを実行し、Apache Iceberg 1.10 を含む更新された Open Table Format ライブラリを利用できます。
