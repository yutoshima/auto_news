# Amazon Aurora PostgreSQLがAWS Free Tierで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月25日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-aurora-postgresql-aws-free-tier/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-aurora-postgresql-aws-free-tier/)

## 概要

Amazon Aurora PostgreSQL は現在 AWS フリーティアで利用可能です。新規のお客様にはサインアップ時に AWS クレジットとして100ドルが提供され、Amazon RDS をはじめとするサービスを利用することで追加の100ドルのクレジットを獲得することができます。フリープランのアカウントを使用すると、<a href="https://console.aws.amazon.com/rds/home" rel="noopener noreferrer" target="_blank">Amazon RDS コンソール</a>、AWS CLI、または AWS SDK を使って、エクスプレス設定で Aurora PostgreSQL サーバーレス・クラスターを作成することができます。
