# AWS Certificate Manager がネイティブな証明書検索を正式にサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月07日 04:01
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-certificate-manager-search/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-certificate-manager-search/)

## 概要

<p>AWS Certificate Manager (ACM)は現在、証明書の検索に使用できる検索バーをコンソールに提供しています。ドメイン名、証明書ARN、および/または証明書の有効期限など、1つ以上の証明書パラメータを使用して証明書を検索できます。たとえば、複数の証明書を管理しているACMユーザーは、間もなく有効期限を迎える特定のドメインを含む証明書を検索できます。<br /> <br /> 使い始めるには、新しい <a href="https://docs.aws.amazon.com/acm/latest/APIReference/API_SearchCertificates.html"><i>SearchCertificates</i></a> を使用します。</p>
