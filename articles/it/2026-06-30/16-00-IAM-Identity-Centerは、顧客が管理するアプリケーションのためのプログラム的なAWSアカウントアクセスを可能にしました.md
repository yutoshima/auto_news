# IAM Identity Centerは、顧客が管理するアプリケーションのためのプログラム的なAWSアカウントアクセスを可能にしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月30日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-iam-identity-center-account-access-customer-managed-apps/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-iam-identity-center-account-access-customer-managed-apps/)

## 概要

IAM Identity Center は、顧客が管理するアプリケーションがユーザーを代行して AWS アカウントへプログラム的にアクセスできるようにします。これには、ユーザーに割り当てられたアカウントとロールを検出し、AWS アカウントアクセスに必要な一時的な認証情報を取得する機能が含まれます。

外部アイデンティティプロバイダー（IdP）を通じてユーザーを認証する顧客管理アプリケーションをお持ちの場合、その IdP を <a href="https://docs.aws.amazon.com/singlesignon/latest/userguide/using-app">アプリケーションとして使用</a>として構成できます。
