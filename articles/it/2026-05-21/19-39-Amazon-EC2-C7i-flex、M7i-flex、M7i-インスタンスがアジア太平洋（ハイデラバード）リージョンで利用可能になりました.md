# Amazon EC2 C7i-flex、M7i-flex、M7i インスタンスがアジア太平洋（ハイデラバード）リージョンで利用可能になりました

## メタデータ

- **カテゴリ**: it
- **公開日時**: 2026年05月21日 19:39
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-c7i-flex-m7i-flex-m7i-instances-HYD-region/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-c7i-flex-m7i-flex-m7i-instances-HYD-region/)

## 概要

本日より、カスタム第4世代 Intel Xeon Scalable プロセッサー（コードネーム Sapphire Rapids）を搭載した Amazon Elastic Compute Cloud（Amazon EC2） C7i-flex、M7i-flex、M7i インスタンスが Asia Pacific (Hyderabad) リージョンで提供開始されます。これらのカスタムプロセッサーは AWS のみで提供され、他のクラウドプロバイダーが使用する同等の x86 ベース Intel プロセッサーと比較して最大で約15%のパフォーマンス向上を実現します。  
C7i-flex および M7i-flex インスタンスは、価格性能を手軽に享受できる最も簡単な方法です。
