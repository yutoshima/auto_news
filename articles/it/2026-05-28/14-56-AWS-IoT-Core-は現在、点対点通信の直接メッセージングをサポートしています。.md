# AWS IoT Core は現在、点対点通信の直接メッセージングをサポートしています。

## 概要

<p><a href="https://aws.amazon.com/iot-core/">AWS IoT Core</a>は、接続された任意のデバイスにポイントツーポイントメッセージを送信できる機能を提供するようになり、メッセージ配信の可視性が向上し、メッセージコストが低減されました。AWS IoT Coreは、IoTデバイスをAWSクラウドに安全に接続する完全に管理されたサービスであり、IoTデバイスとクラウドサービスとの間で双方向のメッセージ通信を可能にします。<br /> <br /> これまでは、単一のIoTデバイスにメッセージを送るには、デバイスが購読しているトピックへ公開する必要がありました。
