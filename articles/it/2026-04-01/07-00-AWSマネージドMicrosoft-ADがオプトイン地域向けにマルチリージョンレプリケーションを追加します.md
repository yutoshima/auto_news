## 概要

AWS Directory Service for Microsoft Active Directory（別名 AWS Managed Microsoft AD）は、AWS Opt-In Regions でのマルチリージョン同期を新たにサポートします。これにより、既存のマルチリージョン同期機能が追加の AWS リージョンへ拡大され、各リージョンで独立したディレクトリを作成して手動で同期させる必要がなくなり、該当リージョンのドメイン参加ワークロードが AWS Managed Microsoft AD に接続できるようになります。
