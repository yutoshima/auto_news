# Amazon Aurora DSQL は現在、変更データキャプチャ（Preview）をサポートしています。

- **情報源**: AWS What's New
- **カテゴリ**: it

## 概要

<p>Amazon Aurora DSQLは、変更データキャプチャ（CDC）のプレビュー機能を導入し、リアルタイムのデータベース変更を Amazon Kinesis Data Streams に直接ストリーミングできるようにします。 この完全にマネージドされた機能により、カスタムのストリーミングパイプラインを構築・保守する必要がなくなり、イベント駆動型アプリケーションの構築、リアルタイム分析パイプラインの実現、システム間のデータ同期が容易になります。<br /> <br /> Aurora DSQLは、挿入・更新・削除操作の結果を自動的にキャプチャします。
