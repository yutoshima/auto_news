# AWS Neuron 2.30.0 が NKI 0.4.0 と 22 個の新しい NKI Library カーネルを搭載して利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月26日 20:17
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-announce-neuron-2-30-0](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-announce-neuron-2-30-0)

## 概要

本日、AWSは AWS Neuron 2.30.0 の一般提供を発表し、新しい AWS Trainium3 専用ハードウェア機能を備えた NKI 0.4.0、22 の新規 NKI ライブラリカーネル、モデルの移植と検証のための拡張された Neuron Agentic Development スキルを提供します。このリリースは、カスタムカーネルを構築したり、トレーニングと推論ワークロードを最適化したり、モデルを AWS Trainium および Inferentia に移植する ML デベロッパー向けです。  
  
NKI 0.4.0 は Trn3 用の activate2 Scalar Engine 命令を導入します（OCP FP について）。
