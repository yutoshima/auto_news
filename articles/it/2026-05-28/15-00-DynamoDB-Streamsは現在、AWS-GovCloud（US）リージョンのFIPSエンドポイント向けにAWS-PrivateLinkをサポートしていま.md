# DynamoDB Streamsは現在、AWS GovCloud（US）リージョンのFIPSエンドポイント向けにAWS PrivateLinkをサポートしています

- **カテゴリ**: it
- **公開日時**: 2026年05月28日 15:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-dynamodb-streams-privatelink-fips-govcloud/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-dynamodb-streams-privatelink-fips-govcloud/)

## 概要

<p>Amazon DynamoDB Streams は現在、AWS GovCloud (US) リージョンの FIPS（Federal Information Processing Standard）エンドポイント向けに AWS PrivateLink をサポートしています。DynamoDB Streams は DynamoDB テーブルの項目レベルの変更を時系列順にキャプチャし、リアルタイムのデータ処理やイベント駆動型アーキテクチャを可能にします。この強化により、政府機関や連邦遵守要件を持つ組織は、VPC と DynamoDB Streams FIPS エンドポイントとの間にプライベート接続を確立できるようになります。
