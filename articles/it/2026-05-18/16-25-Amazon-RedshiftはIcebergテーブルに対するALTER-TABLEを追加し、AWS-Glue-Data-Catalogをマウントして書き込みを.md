# Amazon RedshiftはIcebergテーブルに対するALTER TABLEを追加し、AWS Glue Data Catalogをマウントして書き込みを行います。

## 概要

<p><a href="http://aws.amazon.com/redshift">Amazon Redshift</a> は現在、AWS Glue Data Catalog (awsdatacatalog) マウント経由で Apache Iceberg テーブルへの直接書き込みと、ALTER TABLE DDL 文を使用した Iceberg テーブルのスキーマ・パーティショニング・プロパティの変更をサポートしています。自動マウントされた awsdatacatalog を介して書き込みアクセスが可能なため、外部スキーマを作成することなく、データレイクに Redshift の変換結果を格納して任意のエンジンでクエリできるようになります。特に Iceberg テーブルをフェデレーションする用途で有用です。
