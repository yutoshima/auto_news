# Amazon EC2 X8i インスタンスがヨーロッパ（パリ）で利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月10日 23:54
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ec2-x8i-instances-CDG-region/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ec2-x8i-instances-CDG-region/)

## 概要

Amazon Web Services（AWS）は、AWSのみで利用可能なカスタムIntel Xeon 6プロセッサを搭載した次世代のメモリ最適化インスタンス「Amazon EC2 X8i」の一般提供を発表します。X8iインスタンスはSAP認定を取得しており、クラウド上の同等のIntelプロセッサの中で最高のパフォーマンスと最速のメモリ帯域幅を実現します。前世代のX2iと比較して、最大43%のパフォーマンス向上、メモリ容量が1.5倍（最大6TB）、メモリ帯域幅が3.3倍の向上を実現します。
