# AWS Resource Explorer は、AWS GovCloud (US-East) および (US-West) で利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月07日 12:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-resource-explorer-available-aws-govcloud/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-resource-explorer-available-aws-govcloud/)

## 概要

私たちは、リソースの検索と発見を簡素化するマネージド機能である AWS Resource Explorer が、AWS GovCloud リージョン（US-East）および（US-West）で利用可能になったことをお知らせします。

AWS Resource Explorer コンソール、AWS Command Line Interface（AWS CLI）、AWS SDK、または AWS Management Console のどこからでも統合検索バーを使用して、AWS リソースを検索できます。コンソールに表示される検索結果から、該当リソースの詳細ページへ移動することができます。
