# Amazon ElastiCache Serverless が IPv6 とデュアルスタック接続に対応しました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月02日 21:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-elasticache-serverless-ipv6-dual-stack/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-elasticache-serverless-ipv6-dual-stack/)

## 概要

<p>Amazon ElastiCache Serverless はIPv6とデュアルスタック接続に対応しました。これにより、従来のIPv4接続だけに限定されていた範囲を超え、サーバーレスキャッシュへの接続方法により柔軟性が広がります。</p>
<p>ElastiCache Serverless キャッシュを作成する際、IPv4・IPv6・デュアルスタックの3つのネットワークタイプのオプションから選択できるようになりました。デュアルスタック接続を利用すると、キャッシュはIPv4とIPv6の両方の接続を受け付けます。</p>
