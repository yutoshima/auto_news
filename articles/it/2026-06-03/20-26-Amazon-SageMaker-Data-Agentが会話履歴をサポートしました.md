# Amazon SageMaker Data Agentが会話履歴をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月03日 20:26
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-sagemaker-data-agent/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-sagemaker-data-agent/)

## 概要

Amazon SageMaker Data Agent は現在 SageMaker Unified Studio で利用可能で、会話履歴をサポートします。これによりデータ実務者は分析セッション間の継続性を維持でき、データ分析者やデータサイエンティストは以前にエージェントが生成したコードを参照したり、複数ステップの分析を再開したり、ノートブックや Query Editor のワークフロー内で過去のトラブルシューティングのやり取りを確認したりできます。

会話履歴を使えば、スクロールして前回の続きから正確に再開することが可能です。
