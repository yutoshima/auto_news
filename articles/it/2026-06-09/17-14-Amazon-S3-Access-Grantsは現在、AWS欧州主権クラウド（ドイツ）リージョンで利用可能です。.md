# Amazon S3 Access Grantsは現在、AWS欧州主権クラウド（ドイツ）リージョンで利用可能です。

## 概要

Amazon S3 Access GrantsをAWSヨーロッパ規 sovereigCloud（Germany）リージョンで作成できるようになりました。

Amazon S3 Access Grantsは、Microsoft Entra IDなどのディレクトリのIDや、AWS Identity and Access Management（IAM）プリンシパルを、S3のデータセットへと対応付けます。これにより、企業のIDに基づいてエンドユーザーに自動的にS3アクセスを付与することで、データ権限を大規模に管理するのに役立ちます。
