# AWS Private CA が現在、利用状況メトリクスを Amazon CloudWatch に公開します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月31日 13:00

## 概要

<p>AWS Private Certificate Authority（AWS Private CA）は現在、CAの利用状況を可視化するために証明書機関（CA）利用状況メトリクスをAmazon CloudWatchに公開します。AWS Private CAは、CAが発行できる証明書の数と、リージョンごとに作成できるCAの数に対するサービスクォータ制限を適用します。新しいメトリクスは、各CAが発行した証明書の数と、リージョンごとのCAの総数を追跡し、これらのクォータに対する使用状況を監視し、積極的に管理できるようにします。
