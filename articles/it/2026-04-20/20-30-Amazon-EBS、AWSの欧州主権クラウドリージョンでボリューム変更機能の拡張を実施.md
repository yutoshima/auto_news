# Amazon EBS、AWSの欧州主権クラウドリージョンでボリューム変更機能の拡張を実施

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月20日 20:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ebs-four-volume-modifications-european-sovereign-region/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-ebs-four-volume-modifications-european-sovereign-region/)

## 概要

<p>Amazon Elastic Block Store（Amazon EBS）は、現在、AWS欧州主権クラウド（ドイツ）リージョン内のボリュームごとに、ローリング24時間ウィンドウ内で最大4つのElastic Volumes変更をサポートしています。Elastic Volumesの変更を使用すると、EBSボリュームのサイズを拡張したり、タイプを変更したり、性能を調整したりできます。今回の更新により、過去24時間内に4件未満の変更を開始していれば、前の変更が完了した直後に新しい変更を開始できます。</p>
