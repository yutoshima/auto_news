## 概要

AWS Service Catalog が、新たに2つのAWSリージョン（アジア太平洋(ニュージーランド) と Canada West(カルガリー)）で提供開始されました。<br /><br /> AWS Service Catalog は、承認済みのInfrastructure as Code（IaC）製品のカタログを作成・管理・配布し、AWS上へ展開することを可能にします。管理者は AWS CloudFormation や Terraform などの他の IaC ツールを用いて製品を定義します。製品とは、単一の計算インスタンスから完全に構成されたリソースのセットまで、さまざまなAWSリソースの集合です。
