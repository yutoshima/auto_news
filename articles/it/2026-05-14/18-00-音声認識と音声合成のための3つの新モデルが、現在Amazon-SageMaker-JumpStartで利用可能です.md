# 音声認識と音声合成のための3つの新モデルが、現在Amazon SageMaker JumpStartで利用可能です

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月14日 18:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/speech-models-on-sagemaker-jumpstart/](https://aws.amazon.com/about-aws/whats-new/2026/05/speech-models-on-sagemaker-jumpstart/)

## 概要

本日、AWSは Qwen3-TTS-12Hz-1.7B-CustomVoice、Qwen3-TTS-12Hz-1.7B-Base、Qwen3-ASR-1.7BをAmazon SageMaker JumpStartで提供開始したと発表しました。これにより、AWSのお客様が利用できるファウンデーションモデルのポートフォリオが拡大します。Qwenのこの3モデルは、10言語以上にわたる高度な音声合成と音声認識機能を提供し、顧客がAWSインフラストラクチャ上で知能的な音声搭載アプリケーションを構築できるようにします。

これらのモデルは、企業向けのさまざまな音声・音響の課題に対応します。
