# Amazon Verified Permissions は現在、ポリシーストアの別名と名前付きポリシー、およびポリシーテンプレートをサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月06日 18:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-verified-permissions-policy-store/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-verified-permissions-policy-store/)

## 概要

<p>本日、AWSは Amazon Verified Permissions におけるポリシー ストア別名と名前付きポリシーおよびポリシーテンプレートのサポートを発表し、マルチテナント展開と日常のポリシー管理を簡素化します。Amazon Verified Permissions は Cedar ポリシーを用いてアプリケーション全体の権限を管理・適用するための、細粒度の認可サービスです。これらの新機能により、テナント識別子とポリシー ストア ID を関連付けるための別個のマッピング テーブルを維持する必要がなくなります。
