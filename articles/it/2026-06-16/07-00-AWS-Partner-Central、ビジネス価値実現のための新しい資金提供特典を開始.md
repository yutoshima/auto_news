# AWS Partner Central、ビジネス価値実現のための新しい資金提供特典を開始

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月16日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-partner-business-value-realization/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-partner-business-value-realization/)

## 概要

<p>AWS Partner Central は現在、Business Value Realization（BVR）モーションをサポートしています。これは、戦略的な AWS サービスを導入した後に顧客の採用とビジネス成果を推進するパートナーのための新しい体験と資金動機です。BVR は、定義された段階にわたって AWS サービスの採用の旅を構築し、実証された価値の実現に連動する資金を提供することで、パートナーが顧客のビジネス成果を推進するのを支援します。<br /> <br /> パートナーは現在、セルフサービス登録フローを通じて BVR に登録できます。
