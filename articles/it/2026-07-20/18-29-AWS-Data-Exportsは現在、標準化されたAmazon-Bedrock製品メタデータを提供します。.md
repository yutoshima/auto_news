# AWS Data Exportsは現在、標準化されたAmazon Bedrock製品メタデータを提供します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月20日 18:29

## 概要

<p>本日、AWSは、Bedrock のコストを理解するための一貫した構造化属性を提供する、AWS Data Exports（コストと使用量レポート）用の標準化された製品メタデータを発表します。AWS Data Exports では、AWS のコストと使用量データをカスタマイズしてエクスポートし、それを Amazon S3 に配信して Amazon Athena でクエリしたり、データウェアハウスに読み込んだりできます。これらの属性を用いることで、カスタムロジックを構築せずに Bedrock の支出を attribution（配分）できます。
