# AWS Elastic Disaster RecoveryがIPv6をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月13日 15:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-elastic-disaster-recovery-ipv6/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-elastic-disaster-recovery-ipv6/)

## 概要

AWS Elastic Disaster Recovery（AWS DRS）は、データ複製とコントロールプレーン接続の両方においてIPv6をサポートしました。IPv6専用ネットワーク環境やデュアルスタック環境で運用している顧客は、IPv6を使用してAWS DRSを複製するよう設定できるようになり、災害復旧設定でIPv4アドレスを必要としなくなります。

AWS DRSは、ローカルおよびクラウドベースのアプリケーションを、手頃なストレージ、最小限のコンピュート、ポイントインタイムの復旧を用いて、ダウンタイムとデータ損失を最小限に抑えた高速で信頼性の高い復旧を実現します。
