## 🚀 クイックスタート

### 1. 初期セットアップ（5分）

```bash
# 1. リポジトリをクローン
git clone <your-repo-url>
cd auto_news

# 2. 依存パッケージをインストール
pip install -r requirements.txt

# 3. 環境変数を設定
cp .env.example .env
# .env ファイルを編集してAPIキーとWebhook URLを設定

# 4. 接続テスト
python main.py --mode test
```

### 2. ローカル実行（即座に）

```bash
# 全ニュース配信（★3以上、24時間分）
python main.py --mode all

# 重要度を変更（★4以上のみ）
python main.py --mode all --importance 4

# 新型車専用モード
python main.py --mode new-cars
```

### 3. GitHub Actions設定（10分）

```bash
# 1. GitHubリポジトリ → Settings → Secrets → Actions
#    以下のSecretsを追加:
#    - POE_API_KEY
#    - IT_WEBHOOK_URL
#    - CAR_WEBHOOK_URL
#    - NEW_CAR_WEBHOOK_URL

# 2. Settings → Actions → General → Workflow permissions
#    "Read and write permissions" を選択

# 3. Actions タブから手動実行テスト
```
