# Amazon OpenSearch Ingestionは、OpenTelemetryデータの統合取り込みエンドポイントをサポートするようになりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月04日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-opensearch-ingestion-unified-ingestion-endpoint-opentelemetry](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-opensearch-ingestion-unified-ingestion-endpoint-opentelemetry)

## 概要

<p>Amazon OpenSearch Ingestionは現在、ログ、メトリクス、トレースの3つのOpenTelemetry観測信号を単一のパイプラインで受け付ける統一的なエンドポイントをサポートしています。以前は、3つのOpenTelemetryデータタイプをすべて取り込みたい顧客は、それぞれの信号タイプごとに3つの別々のパイプラインを作成・管理する必要がありました。今回のリリースにより、単一のパイプラインが任意の組み合わせのOpenTelemetry信号を受信できるようになり、パイプラインの構成を簡素化し、運用コストを削減します。</p>
