# Amazon SageMaker Feature Storeは現在、バッチ機能の書き込みとレコードのリスト作成をサポートしています。

## 概要

<p>Amazon SageMaker Feature Store は、AI モデルの学習とデプロイのための特徴量の計算・保存・取得を容易に行える、完全に管理された機能です。SageMaker Feature Store は現在、高スループットの特徴量取り込み、レコードの検索、およびオフラインストアのカタログ化を含む新機能をサポートしています。データサイエンティストは、BatchWriteRecord を使用して、単一のリクエストで複数の特徴量グループにまたがる複数のレコードを記述できるようになり、各レコードを個別に識別せずに特徴量グループに格納されたレコードをリスト表示できます。
