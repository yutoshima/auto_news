# AWS Marketplaceは現在、 Agreements API を使ったプログラム的な調達をサポートしています。

- **カテゴリ**: it
- **公開日時**: 2026年05月06日 18:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-marketplace-agreements-api/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-marketplace-agreements-api/)

## 概要

本日、AWS Marketplaceが「アグリーメントAPI」を発表します。これにより、プログラムでAWS Marketplaceの製品を調達し、アグリーメントを管理することが可能になります。今回のリリースで、見積もりの作成、オファーの承認、課金と権利の追跡、購入注文の更新、アグリーメントの管理を、既存のツールとワークフローの中で行うことができます。

Discovery APIと組み合わせることで、アグリーメントAPIは製品の発見から購入までのエンドツーエンドの調達プロセスを提供します。これらのAPIをあなたの環境に統合することができます。
