# Amazon EC2 C8gdおよびM8gdインスタンスは、追加のAWSリージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月11日 18:04
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-c8gd-m8gd-instances-available/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-c8gd-m8gd-instances-available/)

## 概要

Amazon Elastic Compute Cloud（Amazon EC2）の C8gd および M8gd インスタンスは、最大 11.4 TB のローカル NVMe ベース SSD ブロックレベルストレージを搭載し、追加のリージョンで利用可能になりました。C8gd インスタンスは現在、南米（サンパウロ）で利用可能です。M8gd インスタンスは現在、欧州（アイルランド）で利用可能です。これらのインスタンスは AWS Graviton4 プロセッサを搭載しており、Graviton3 ベースのインスタンスと比較して最大で 30% のパフォーマンス向上を実現します。また、I/O 集約型のデータベースワークロードでは最大で 40% 高い性能を発揮します。
