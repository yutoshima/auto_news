## 概要

本日、AWSは次世代の AWS Resilience Hub の一般提供を発表します。Resilience Hub は、AWS コンソール内の中心的な拠点として、プラットフォームエンジニアリングおよびサイト信頼性エンジニアリング（SRE）チームが AWS 上で実行される重要なワークロードの回復力を評価・強化するのを支援します。今回の新機能更新は、既存の回復力目標の達成体験を拡張し、新しいアプリケーションモデル、依存関係の発見、生成系AIを活用した障害モード分析、モジュール化された回復力などを導入します。
