# AWS CloudFormationとCDKのエクスプレスモードは、インフラストラクチャのデプロイを最大4倍速くします。

## 概要

<p><a href="https://docs.aws.amazon.com/cloudformation/">AWS CloudFormation</a> と <a href="https://docs.aws.amazon.com/cdk/v2/guide/home.html">CDK</a> express mode は、インフラを構築する開発者とAIエージェント向けのデプロイ時間を内部ベンチマークに基づき最大4倍短縮します。Express mode は、CloudFormation がリソース構成の適用を確認した時点でスタック操作を完了させ、トラフィックの準備完了、リージョン伝播、リソースの安定化といった長時間の検証を待つ必要をなくします。
