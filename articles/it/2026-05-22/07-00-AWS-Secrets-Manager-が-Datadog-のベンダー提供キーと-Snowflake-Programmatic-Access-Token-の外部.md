# AWS Secrets Manager が Datadog のベンダー提供キーと Snowflake Programmatic Access Token の外部秘密を管理する機能を追加しました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月22日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/secrets-manager-managed-external-secrets-datadog-snowflake/](https://aws.amazon.com/about-aws/whats-new/2026/05/secrets-manager-managed-external-secrets-datadog-snowflake/)

## 概要

AWS Secrets Managerは、管理対象外部シークレット機能を拡張し、DatadogキーとSnowflakeのプログラム的アクセス トークン（PAT）を追加しました。管理対象外部シークレットにより、サードパーティの資格情報をAWS Secrets Managerから自動的にローテーションできるようになり、対応するサードパーティサービスとの一流の統合を提供します。

このリリースにより、Datadogの資格情報のローテーション対象は、APIキー、アプリケーションキー、管理者クレデンシャルペアの3タイプに対応します。
