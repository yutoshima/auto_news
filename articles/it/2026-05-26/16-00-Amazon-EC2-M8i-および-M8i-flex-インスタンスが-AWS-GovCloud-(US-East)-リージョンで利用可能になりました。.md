# Amazon EC2 M8i および M8i-flex インスタンスが AWS GovCloud (US-East) リージョンで利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月26日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-m8i-m8i-flex-govcloud-east/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-m8i-m8i-flex-govcloud-east/)

## 概要

本日より、Amazon EC2 M8iおよびM8i-flexインスタンスがAWS GovCloud (US-East)リージョンで利用可能になりました。これらのインスタンスは、AWSだけで提供されるカスタムIntel Xeon 6プロセッサによって動作し、クラウド上の同等のIntelプロセッサの中で最高の性能と最速のメモリ帯域を実現します。M8iおよびM8i-flexインスタンスは、前世代のIntelベースのインスタンスと比較して、価格対性能が最大15％向上し、メモリ帯域幅が2.5倍向上します。さらに最大20％の向上をもたらします。
