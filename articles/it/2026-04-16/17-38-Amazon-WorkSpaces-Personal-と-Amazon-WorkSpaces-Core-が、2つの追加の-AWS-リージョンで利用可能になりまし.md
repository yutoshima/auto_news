# Amazon WorkSpaces Personal と Amazon WorkSpaces Core が、2つの追加の AWS リージョンで利用可能になりました。

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月16日 17:38
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-workspaces-available-two-new-regions/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-workspaces-available-two-new-regions/)

## 概要

<p>Amazon WorkSpaces Personal と Amazon WorkSpaces Core が、US East (Ohio) および Asia Pacific (Malaysia) の AWS リージョンで利用可能になりました。これにより、ユーザーに近い場所で WorkSpaces を展開できるようになり、国内データ所在の要件を満たし、よりレスポンスの良い体験を提供できます。US East (Ohio) では、組織が災害復旧ソリューションを導入し、地域のデータ所在コンプライアンス要件を満たし、仮想デスクトップへの一貫した低遅延アクセスで地域の従業員をサポートすることも可能です。</p>
