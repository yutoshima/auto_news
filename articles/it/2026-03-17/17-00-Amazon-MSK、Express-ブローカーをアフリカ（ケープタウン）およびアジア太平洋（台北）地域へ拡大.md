# Amazon MSK、Express ブローカーをアフリカ（ケープタウン）およびアジア太平洋（台北）地域へ拡大

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月17日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-msk-express-2-new-aws-regions/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-msk-express-2-new-aws-regions/)

## 概要

現在、アフリカ（ケープタウン）およびアジア太平洋（台北）地域で、Express ブローカーを使用した provisioned Amazon Managed Streaming for Apache Kafka (Amazon MSK) クラスターを作成できます。

Express ブローカーは、Amazon MSK Provisioned の新しいブローカタイプで、1ブローカーあたりのスループットを最大3倍、スケーリングを最大20倍速く、標準の Apache Kafka ブローカーと比較して回復時間を90%短縮することを目指して設計されています。Express ブローカーはあらかじめ設定されています。
