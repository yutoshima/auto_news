# アマゾン EC2 ハイメモリ U7in-24TB インスタンスが AWS アジア太平洋（ソウル）リージョンで利用可能になりました

## メタデータ

- **カテゴリ**: it
- **公開日時**: 2026年06月24日 01:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-u7in-24tb-aws-seoul/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-u7in-24tb-aws-seoul/)

## 概要

<p>Amazon EC2 High Memory U7in-24TB インスタンス（u7in-24tb.224xlarge）が、AWS Asia Pacific（ソウル）リージョンで利用可能になりました。U7iインスタンスはAWS第7世代の一部で、カスタム第4世代 Intel Xeon Scalable プロセッサー（Sapphire Rapids）を搭載しています。U7in-24TBインスタンスは24 TiBのDDR5メモリを提供し、急成長するデータ環境でトランザクション処理スループットのスケールを可能にします。U7iインスタンスは、従来のU-1インスタンスと比較して最大45%のコストパフォーマンス改善を実現します。</p>
