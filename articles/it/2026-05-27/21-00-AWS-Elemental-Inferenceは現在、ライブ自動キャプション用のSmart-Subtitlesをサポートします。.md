# AWS Elemental Inferenceは現在、ライブ自動キャプション用のSmart Subtitlesをサポートします。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月27日 21:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/elemental-inference-subtitles](https://aws.amazon.com/about-aws/whats-new/2026/05/elemental-inference-subtitles)

## 概要

AWS Elemental Inferenceは現在、スマート字幕をサポートしています。これはリアルタイムの字幕を自動生成するAI搭載機能で、ライブ動画ストリーム向けです。スマート字幕は高度な音声認識を用いて話された音声を文字起こし、TTML（Timed Text Markup Language）形式の字幕を低遅延で提供します。これにより、放送局や配信者は手動の字幕作成ワークフローや第三者サービスを使わずに、視聴者へアクセシブルなコンテンツを提供できます。

スマート字幕を使えば、ライブ字幕を追加できます。
