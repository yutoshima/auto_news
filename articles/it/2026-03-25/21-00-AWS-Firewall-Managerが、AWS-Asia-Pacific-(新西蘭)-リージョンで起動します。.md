# AWS Firewall Managerが、AWS Asia Pacific (新西蘭) リージョンで起動します。

- **カテゴリ**: it

## 概要

AWS Firewall Managerが、AWS Asia Pacific (New Zealand) Regionで利用可能になったことを発表します。AWS Firewall Managerは、クラウドセキュリティ管理者やサイト信頼性エンジニアが、ルールの手動設定・管理の運用オーバーヘッドを削減しつつ、アプリケーションを保護するのに役立ちます。  
AWS Firewall Managerと連携することで、顧客は防御の深さを重視したポリシーを提供し、アプリケーションとワークロードをホスティングする顧客のためのAWSのセキュリティサービス全体に対応するポリシーを実装できます。
