# AWSパートナーセントラルは、基盤的技術審査を数分で検証します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月16日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-partner-central-foundational-technical-review/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-partner-central-foundational-technical-review/)

## 概要

<p>AWS Partner Centralは現在、Foundational Technical Review (FTR) を数分で完了させるために、SOC 2 Type II監査報告書またはAWS Well-Architected Framework Reviews (WAFR) 報告書を受け付けています。このAI搭載の検証プロセスにより、APN要件に対するソリューションの検証結果について AWS パートナーへ即時フィードバックが提供されます。パートナーは、ソリューションの検証を加速し適格ソフトウェアを解放するために、数分以内に承認または実行可能なフィードバックを受け取ります。
