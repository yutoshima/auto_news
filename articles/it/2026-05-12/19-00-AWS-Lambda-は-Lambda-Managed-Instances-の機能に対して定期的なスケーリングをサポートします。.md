# AWS Lambda は Lambda Managed Instances の機能に対して定期的なスケーリングをサポートします。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月12日 19:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-lambda-managed-instances/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-lambda-managed-instances/)

## 概要

AWS Lambda は Lambda Managed Instances 上で実行される関数のスケジュールスケーリングを、Amazon EventBridge Scheduler を使ってサポートするようになりました。この機能により、期待されるトラフィックに備えて事前に関数の容量制限を調整する、1 回限りまたは繰り返しのスケジュールを定義でき、ピーク時のパフォーマンス目標を達成し、アイドル時のコストを抑えることができます。  

Lambda Managed Instances は、組み込みのルーティングと負荷分散を備えたマネージドな Amazon EC2 インスタンス上で Lambda 関数を実行できるようにします。
