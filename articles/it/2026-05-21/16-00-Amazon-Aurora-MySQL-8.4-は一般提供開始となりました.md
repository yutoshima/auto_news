# Amazon Aurora MySQL 8.4 は一般提供開始となりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月21日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-aurora-mysql/8-4/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-aurora-mysql/8-4/)

## 概要

Amazon Aurora MySQL-互換版は現在、コミュニティ版の Long Term Support（LTS）メジャーバージョンである MySQL 8.4 をサポートしています。Aurora MySQL 8.4 は、コミュニティ MySQL 8.4.7 との互換性を前提にリリースされ、互換性のあるバージョン番号を揃えることで、Aurora で実行するバージョン番号が、互換性のあるコミュニティ MySQL のバージョンと一致するようになります。Aurora は基盤となるパッチの適用を代替で管理するため、日常の運用を簡素化します。Aurora MySQL は現在、コミュニティ MySQL のメジャーバージョンを、コミュニティ版の 12 ヶ月以内に対象とします。
