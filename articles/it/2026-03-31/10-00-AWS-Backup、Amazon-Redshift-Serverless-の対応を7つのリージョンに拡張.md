# AWS Backup、Amazon Redshift Serverless の対応を7つのリージョンに拡張

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月31日 10:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-backup-redshift-serverless-seven-regions/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-backup-redshift-serverless-seven-regions/)

## 概要

AWS Backup の Amazon Redshift Serverless 対応が、以下の7つの追加 AWS リージョンで利用可能になりました：アジア太平洋（大阪、ハイデラバード、台北、クアラルンプール、オークランド）、ヨーロッパ（ミラノ）、アフリカ（ケープタウン）。

この拡張により、これらの新しくサポートされたリージョンの Amazon Redshift Serverless データウェアハウスに、ポリシーに基づくデータ保護とリカバリを提供します。

Redshift Serverless リソースを AWS Backup で保護を開始するには、それらを既存のバックアップ計画に追加するか、新しい計画を作成してください。
