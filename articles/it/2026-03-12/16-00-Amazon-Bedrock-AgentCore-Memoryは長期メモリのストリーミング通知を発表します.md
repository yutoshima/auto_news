# Amazon Bedrock AgentCore Memoryは長期メモリのストリーミング通知を発表します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月12日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/agentcore-memory-streaming-ltm/](https://aws.amazon.com/about-aws/whats-new/2026/03/agentcore-memory-streaming-ltm/)

## 概要

Amazon Bedrock AgentCore Memory は長期記憶のストリーミング通知をサポートするようになり、変更をポーリングする必要がなくなりました。長期記憶はエージェントのやり取りから洞察を抽出して、将来のやり取りでパーソナライズされた体験を提供します。開発者は、記憶レコードが作成または変更されるたびにプッシュ通知を受け取ります。更新情報は、記憶レコードが作成または変更されるたびに直接 Amazon Kinesis にストリームされます。これにより、開発者は下流のワークフローをトリガーできます。
