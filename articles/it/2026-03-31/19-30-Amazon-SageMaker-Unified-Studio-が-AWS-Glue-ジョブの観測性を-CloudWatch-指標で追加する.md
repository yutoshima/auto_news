# Amazon SageMaker Unified Studio が AWS Glue ジョブの観測性を CloudWatch 指標で追加する

## 概要

Amazon SageMaker Unified Studioはジョブの可観測性を追加し、AWS GlueジョブのCloudWatchメトリクスをジョブログとともに、1つの統合インターフェースで直接表示します。この機能により、SageMaker Unified Studioに可観測性が追加され、データエンジニアとETL開発者がトラブルシューティングの手順を効率化できます。

この機能により、リソース利用パターン（DPU利用、メモリ消費量など）を相関させることで、パフォーマンスの問題をより迅速に診断できます。
