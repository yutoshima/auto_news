# AWS HealthImagingは研究レベルの細粒度アクセス制御を発表します。

## 概要

AWS HealthImagingは現在、細かいアクセス制御をサポートしており、組織が医療画像データへのアクセスをDICOMのスタディ（研究）およびシリーズ単位で安全に管理できるようになりました。医療画像ワークフローは通常、DICOMスタディを中心に構成され、AWS HealthImagingには1つ以上のイメージセットリソースとして保存されます。これにより、顧客はDICOMスタディまたはシリーズの一連のイメージセットすべてへのアクセス権を、維持管理が容易なIAMポリシーを用いて簡単に付与できるようになりました。

顧客は現在、権限を付与できます。
