# Amazon Redshift は、複数の AWS リージョンで IAM Identity Center を用いたフェデレーテッド権限をサポートします。

## 概要

<p>複数の AWS リージョンで、Amazon Redshift のフェデレーテッド権限が AWS IAM Identity Center（IdC）に対応しました。IdCをメインの AWS リージョンから追加のリージョンへ拡張することで、ユーザーへの近接性と信頼性の向上を通じてパフォーマンスを改善できます。追加のリージョンでは、IdC を用いた既存の従業員識別情報を使って、テーブルおよび列レベルの Redshift の細かいアクセス権を、より簡便に管理できるようになります。<br /> <br /> IdC に新しいリージョンを追加すると、作成できます。
