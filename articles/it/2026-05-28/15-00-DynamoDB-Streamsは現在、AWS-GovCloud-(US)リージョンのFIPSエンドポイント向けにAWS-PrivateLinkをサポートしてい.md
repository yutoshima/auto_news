# DynamoDB Streamsは現在、AWS GovCloud (US)リージョンのFIPSエンドポイント向けにAWS PrivateLinkをサポートしています。

## 概要

<p>Amazon DynamoDB Streams は現在、AWS GovCloud (US) リージョンにおける FIPS（連邦情報処理標準）エンドポイント向けの AWS PrivateLink をサポートしています。DynamoDB Streams は DynamoDB テーブルのアイテムレベルの変更を時系列でキャプチャし、リアルタイムなデータ処理とイベントドリブンなアーキテクチャを可能にします。この機能強化により、政府機関や連邦規制要件を満たす組織は、VPC と DynamoDB Streams の FIPS エンドポイント間でプライベート接続を確立することができます。
