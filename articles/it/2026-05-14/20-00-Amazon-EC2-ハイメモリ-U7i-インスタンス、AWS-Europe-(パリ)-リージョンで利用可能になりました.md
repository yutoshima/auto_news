# Amazon EC2 ハイメモリ U7i インスタンス、AWS Europe (パリ) リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月14日 20:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-u7i-aws-europe-paris/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-u7i-aws-europe-paris/)

## 概要

<p>Amazon EC2 High Memory U7i-12TB インスタンス（u7i-12tb.224xlarge）および U7in-16TB インスタンス（u7in-16tb.224xlarge）は、AWS Europe（パリ）リージョンで利用可能になりました。U7i インスタンスは AWS の第7世代に属し、カスタムの第4世代 Intel Xeon Scalable プロセッサー（Sapphire Rapids）を搭載しています。U7i インスタンスは従来の U-1 インスタンスと比べて最大で約45％の価格性能を提供します。</p> 
<p>U7i-12TB インスタンスは 12 TiB の DDR5 メモリを、U7in-16TB インスタンスは 16 TiB の DDR5 メモリを搭載します。</p>
