# AWS Organizations は現在、API レスポンスに組織パスを提供します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月31日 18:59
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-organizations-paths-api-responses/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-organizations-paths-api-responses/)

## 概要

AWS Organizationsは現在、アカウントと組織単位（OU）の完全な組織パスをAPIレスポンスで直接返すようになり、組織階層をたどるための複数回のAPI呼び出しが不要になりました。以前は、アカウントとOUが組織構造内でどこに配置されているかを把握するには、複数のAPI呼び出しが必要でした。この改善は、複雑な大規模組織を管理するエンタープライズ顧客にとって特に有用です。
