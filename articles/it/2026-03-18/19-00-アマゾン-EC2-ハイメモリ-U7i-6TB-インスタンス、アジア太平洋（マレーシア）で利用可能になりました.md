# アマゾン EC2 ハイメモリ U7i-6TB インスタンス、アジア太平洋（マレーシア）で利用可能になりました

## メタデータ

- **カテゴリ**: it
- **公開日時**: 2026年03月18日 19:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/ec2-high-memory-u7i-6tb-asia-pacific/](https://aws.amazon.com/about-aws/whats-new/2026/03/ec2-high-memory-u7i-6tb-asia-pacific/)

## 概要

<p>Amazon EC2 High Memory U7i インスタンスは、メモリ6 TB（u7i-6tb.112xlarge）として、AWS アジア太平洋（マレーシア）で利用可能になりました。U7i インスタンスは AWS の第7世代に属し、カスタム第四世代 Intel Xeon Scalable Processors（Sapphire Rapids）を搭載しています。U7i-6tb インスタンスは 6TiB の DDR5 メモリを提供し、急成長するデータ環境で取引処理スループットを迅速に拡張することを可能にします。</p> 
<p>U7i-6tb インスタンスは 448 vCPU を提供し、最大 100 Gbps の Amazon EBS 帯域幅をサポートします。</p>
