# Amazon EventBridge Scheduler は、Lambda Managed Instances を含む619の新しい SDK API アクションを追加します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月12日 18:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-eventbridge-sdk-integrations/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-eventbridge-sdk-integrations/)

## 概要

<p>Amazon EventBridge Scheduler は、AWS SDK の連携を拡張し、新規および既存の AWS サービス全体で 13 の追加サービスと 619 の新規 API アクションを提供します。これには AWS Lambda Managed Instances も含まれます。カスタム統合コードを作成することなく、より広い範囲の AWS サービスの直接呼び出しをスケジュールできるようになりました。<br /><br /> EventBridge Scheduler はサーバーレスのスケジューラで、270 を超える AWS サービスにまたがる数十億のスケジュールイベントとタスクを作成・実行・管理することを可能にします。
