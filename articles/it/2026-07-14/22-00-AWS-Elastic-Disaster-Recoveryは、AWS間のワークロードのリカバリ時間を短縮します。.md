# AWS Elastic Disaster Recoveryは、AWS間のワークロードのリカバリ時間を短縮します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月14日 22:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/aws-drs-fast-recovery/](https://aws.amazon.com/about-aws/whats-new/2026/07/aws-drs-fast-recovery/)

## 概要

AWS Elastic Disaster Recovery（AWS DRS）は、AWSベースのワークロードの復旧をより高速にします。Amazon EC2で実行されているソースサーバーについて、DRSは不要となった準備手順をスキップできるようになり、Windowsで最大65%、Linuxで最大40%の復旧時間短縮を実現します。

災害発生時や演習時には、1分1秒が重要です。AWS上ですでに稼働しているワークロードにはAWS互換のドライバーと設定が用意されているため、DRSは手順を少なくして起動でき、アプリケーションを迅速に復旧できます。
