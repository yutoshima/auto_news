# AWS Site-to-Site VPNが既存のVPN接続のトンネル帯域幅の変更をサポートしました

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月06日 21:09
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-site-to-site-vpn-modify-bandwidth/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-site-to-site-vpn-modify-bandwidth/)

## 概要

<AWS Site-to-Site VPNは、既存の接続でトンネル帯域幅を標準（最大1.25 Gbps）と大容量（最大5 Gbps）の間で変更できるようになり、組織のニーズに合わせてVPN接続の帯域幅を更新しやすくなりました。<br /><br /> 以前はトンネル帯域幅を変更するには接続を削除して再作成する必要があり、これにより新しいトンネルIPアドレスが生成され、オンプレミスのVPN機器の設定やファイアウォールルールを更新する必要がありました。今回の開始（リリース）により、トンネルはアップグレードされます。>
