# Amazon Redshift RG インスタンスが AWS GovCloud (US) リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月07日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-redshift-rg-instances-aws-govcloud](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-redshift-rg-instances-aws-govcloud)

## 概要

Amazon Redshift RGインスタンスは、AWS Gravitonプロセッサを搭載しており、AWS GovCloud (US-West) および AWS GovCloud (US-East) リージョンで利用可能になりました。RGインスタンスは、データウェアハウスおよびデータレイクのワークロードを、従来世代のRA3インスタンスより最大2.4倍の高速で、vCPU当たりの価格を約30%低く提供します。RGインスタンスには、Redshiftが独自に構築したベクトル化データレイククエリエンジンが含まれており、クラスタノード上でApache IcebergとParquetデータを処理することで、SQLを実行可能にします。
