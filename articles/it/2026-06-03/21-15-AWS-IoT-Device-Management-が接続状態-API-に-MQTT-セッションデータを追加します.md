# AWS IoT Device Management が接続状態 API に MQTT セッションデータを追加します

## 概要

<AWS IoT Device Management> はMQTTセッションデータを接続状態APIに追加し、IoTデバイス群の接続性の問題をトラブルシューティングし、接続パターンを監査できるようにします。  
このリリースでは、AWS IoT Device Management の既存の <a href="https://docs.aws.amazon.com/iot/latest/developerguide/device-connectivity-status.html">接続性ステータス</a> APIが、AWS IoT Core の <span>接続性ステータス</span> APIと完全に同等になるよう parity を実現します。
