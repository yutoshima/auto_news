# Amazon CloudFront Functions は現在、CloudFront アクセスログへのログ記録をサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月14日 08:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/cloudfront-functions-access-logs/](https://aws.amazon.com/about-aws/whats-new/2026/07/cloudfront-functions-access-logs/)

## 概要

<p>CloudFront Functions 内から利用できる新しいヘルパーメソッドを使用して、CloudFront アクセスログにカスタムデータを直接書き込むことができます。CloudFront Functions は、URL のリライト、ヘッダー操作、リクエストルーティングなどのタスクのためにエッジ上で軽量な JavaScript を実行します。従来は、CloudFront アクセスログとは別のログファイルとして Amazon CloudWatch Logs にログデータを出力するしかありませんでした。このリリースにより、CloudFront アクセスログデータと関数の決定を関連付ける必要がなくなりました。
