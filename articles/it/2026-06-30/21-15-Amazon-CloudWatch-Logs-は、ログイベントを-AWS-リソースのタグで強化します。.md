# Amazon CloudWatch Logs は、ログイベントを AWS リソースのタグで強化します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月30日 21:15
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-cloudwatch-logs-resource-tags/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-cloudwatch-logs-resource-tags/)

## 概要

<p>Amazon CloudWatch Logsは現在、リソースタグでログイベントを強化します。これにより、組織にとって最も重要なメタデータ（チーム所有、環境、コストセンター、アプリケーション名など）でログをフィルター、検索、分析することが容易になり、ロギングの計Instrumentationの変更をする必要がなくなります。</p>
<p>タグ強化を使うと、CloudWatch Logsは取り込み時にリソースタグを直接ログイベントに追加します。タグをすぐにログクエリで使用して、分析の範囲を絞ることができます。</p>
