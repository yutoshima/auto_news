# AWS Compute Optimizer が、EBS ボリュームおよび ECS サービスのリサイズ推奨に対し、32日間の遡及期間をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月03日 18:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-compute-optimizer-ebs-ecs-32-day-lookback/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-compute-optimizer-ebs-ecs-32-day-lookback/)

## 概要

AWS Compute Optimizerは、デフォルトの14日から32日まで、Amazon EBSボリュームとAmazon ECSリサイズ推奨の見直し期間を拡張できるようになりました。追加費用は不要です。より長い見直し期間により、月次の利用パターン（例: 月末処理）を考慮してリサイズ推奨を生成できるため、ワークロードの最適化判断が向上し、コストとパフォーマンスの両方の成果が改善されます。
