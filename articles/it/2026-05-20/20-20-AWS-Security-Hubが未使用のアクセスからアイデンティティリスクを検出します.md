# AWS Security Hubが未使用のアクセスからアイデンティティリスクを検出します

## 概要

<p>今日、AWS Security Hubは、中心的なセキュリティチームが脅威、露出、セキュリティの現状をすでに管理している統一された同じコンソールへ、アイデンティティリスクを統合します。Security Hubは現在、AWS組織全体の未使用のIAM権限、ロール、資格情報を検出し、セキュリティチームがアイデンティティリスクを大規模に特定・低減するのを支援します。これまで数百のアカウントにわたるアイデンティティリスクの管理には、複数のツールを切り替える必要があり、未使用の権限とそれを結びつける統一ビューはありませんでした。
