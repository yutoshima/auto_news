# Amazon EC2 High Memory U7i インスタンスが追加のリージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月11日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-high-memory-u7i-instances/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-high-memory-u7i-instances/)

## 概要

<p>Amazon EC2のHigh Memory U7iインスタンスは、メモリ8TBの「u7i-8tb.112xlarge」がAWSアジア太平洋（ハイデラバード）で利用可能になり、メモリ12TBの「u7i-12tb.224xlarge」がAWSヨーロッパ（スペイン）で利用可能になりました。U7iインスタンスはAWSの7世代目に属し、カスタム第四世代Intel Xeon Scalable Processor（Sapphire Rapids）を搭載しています。U7i-8tbインスタンスは8TiBのDDR5メモリを提供し、U7i-12tbインスタンスは12TiBのDDR5メモリを提供します。これにより、顧客は取引規模を拡大できます。</p>
