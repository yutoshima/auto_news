# Amazon Aurora DSQLが5つの追加のAWSリージョンで利用可能になりました

## 概要

<p>Amazon Aurora DSQL のシングルリージョンクラスターが、アジア太平洋（香港）、アジア太平洋（ムンバイ）、アジア太平洋（シンガポール）、ヨーロッパ（ストックホルム）、南米（サンパウロ）で利用可能になりました。Aurora DSQL は、ほぼ無制限のスケーラビリティと最高の可用性、インフラ管理ゼロを実現する、最も高速なサーバーレス分散 SQL データベースです。これにより、アプリケーションのスケーリングとレジリエンスを手間なく実現し、常時利用可能なアプリケーションの構築を可能にします。
