# Amazon CloudFrontは現在、VPC IPAM統合を通じたIPv6のBYOIPをサポートしています。

- **カテゴリ**: it
- **公開日時**: 2026年03月31日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/cloudfront-byoip-ipv6-vpc-ipam/](https://aws.amazon.com/about-aws/whats-new/2026/03/cloudfront-byoip-ipv6-vpc-ipam/)

## 概要

<p>Amazon CloudFrontは現在、VPC IPアドレスマネージャー（IPAM）を介してAnycast Static IP用のBYOIP（独自のIPアドレスの持ち込み）をサポートしています。この機能により、ネットワーク管理者は自身の公開IPv4/IPv6アドレスプールをCloudFrontディストリビューションで使用でき、AWSのグローバルインフラストラクチャ全体におけるIPアドレス管理を簡素化します。</p> 
<p><br /> CloudFrontは通常、トラフィックを提供するために回転IPアドレスを使用します。CloudFront Anycast Static IPは、顧客が専用のIPアドレスリストを提供できるようにします。</p>
