# Amazon GuardDuty が機密ファイルの変更を検知する脅威検出機能を追加

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月01日 16:58
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-guardduty-sfm/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-guardduty-sfm/)

## 概要

<p>Amazon GuardDuty Runtime Monitoring には、Amazon EC2 インスタンス上で機微なファイルが変更された場合や、Amazon EKS または Amazon ECS で実行されているコンテナワークロードを監視してセキュリティチームに警告する、3 つの新しい脅威検知が追加されました。これらの検出は、構成ファイル、認証設定、システムログなどの重要なシステムファイルを監視することにより、侵害後の攻撃者の活動を特定するのに役立ちます。この機能は、セキュリティチーム、DevSecOps の専門家、およびクラウドセキュリティのアーキテクトを対象に設計されています。
