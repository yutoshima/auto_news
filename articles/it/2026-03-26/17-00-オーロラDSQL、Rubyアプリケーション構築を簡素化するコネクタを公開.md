# オーロラDSQL、Rubyアプリケーション構築を簡素化するコネクタを公開

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月26日 17:00

## 概要

本日、Aurora DSQL に対応した Ruby 用 DSQL コネクター（pg gem）をリリースしたことを発表します。Ruby コネクターは認証を簡素化し、従来のユーザー作成パスワードに伴うセキュリティリスクを排除します。接続ごとにトークンを自動生成し、有効なトークンが常に使用されるようにすることで、pg gem の既存機能との完全な互換性を維持します。  
コネクターは IAM トークンの生成と SSL を扱います。
