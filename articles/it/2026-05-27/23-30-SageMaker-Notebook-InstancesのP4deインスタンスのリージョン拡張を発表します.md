# SageMaker Notebook InstancesのP4deインスタンスのリージョン拡張を発表します

## 概要

<p>Amazon EC2 P4de インスタンスを Asia Pacific（東京）で SageMaker ノートブックインスタンス上で一般提供開始できることをお知らせします。</p>
<p><a href="https://aws.amazon.com/ec2/instance-types/p4/">Amazon EC2 P4de インスタンス</a> は、8 基の NVIDIA A100 GPU（高性能 HB M2e メモリ 80GB）を搭載し、現行の P4d インスタンスの GPU より 2 倍の性能を発揮します。新しい P4de インスタンスは合計 640GB の GPU メモリを提供し、ML のトレーニング性能を最大で約 60% 改善し、20% の低遅延を実現します。
