# Amazon EC2 P6-B200インスタンスが、アジア太平洋（ムンバイ）リージョンで利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月16日 03:08
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-p6-b200-mumbai/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-p6-b200-mumbai/)

## 概要

本日より、NVIDIA Blackwell GPU によって加速される Amazon Elastic Compute Cloud（Amazon EC2） P6-B200 インスタンスが、Asia Pacific (Mumbai) リージョンで利用可能になります。これらのインスタンスは、AI のトレーニングおよび推論において P5en インスタンスと比べて最大で2倍の性能を提供します。  
P6-B200 インスタンスは、8個の Blackwell GPU、1440 GB のハイパフォーマンスGPUメモリを搭載し、P5en（第5世代 Intel Xeon プロセッサ Emerald Rapids）と比較してGPUメモリ帯域幅を60%向上させ、最大で3.2テラビットの帯域幅を実現します。
