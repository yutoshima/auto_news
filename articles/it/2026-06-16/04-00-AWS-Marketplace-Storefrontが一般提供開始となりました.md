# AWS Marketplace Storefrontが一般提供開始となりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月16日 04:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-marketplace-storefront/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-marketplace-storefront/)

## 概要

AWS Marketplace Storefrontが一般提供を開始しました。これにより、AWSパートナーは自社のソリューションやサービスのブランド化されたカタログを、ウェブサイトやアプリケーション上で数時間で作成・展開できるようになります。チャネルパートナーと独立系ソフトウェアベンダーは、クラウドマーケットプレイス事業の管理をより簡単に行え、顧客がAWS Marketplaceから自社のソリューションを発見・購入しやすくなります。  
AWS Marketplace Storefrontを使用すると、パートナーは完全にブランド化されたストアを設定できます。
