# Amazon Route 53 Global Resolver が、任意のアスコアDNS解決のためのAWSリージョンの追加と削除を可能にしました

- **カテゴリ**: it

## 概要

<p>Amazon Route 53 Global Resolverは現在、任意のアスカ DNS解決のためにAWSリージョンを追加・削除できるようになり、DNSクエリの解決場所に柔軟性を持たせます。これにより、組織の成長に合わせてGlobal Resolverのカバレッジを容易に拡張したり、コンプライアンス要件に対応するためにリージョン別のデプロイを調整したりできます。<br /> <br /> Global Resolverは、パブリックインターネットドメインおよびプライベートRoute 53ホストゾーンの任意の場所から、任意のアスカDNS解決、DNSクエリのフィルタリングを提供します。
