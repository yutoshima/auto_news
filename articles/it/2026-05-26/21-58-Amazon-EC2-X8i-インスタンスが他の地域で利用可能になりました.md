# Amazon EC2 X8i インスタンスが他の地域で利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月26日 21:58
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-ec2-x8i-instances-SIN-SYD-PDT-region/](https://aws.amazon.com/about-aws/whats-new/2026/02/amazon-ec2-x8i-instances-SIN-SYD-PDT-region/)

## 概要

本日より Amazon Elastic Compute Cloud（Amazon EC2） X8i インスタンスが、 Asia Pacific (Singapore)、 Asia Pacific (Sydney)、および AWS GovCloud (US-West) リージョンで利用可能になります。これらのインスタンスは、AWS 専用のカスタム Intel Xeon 6 プロセッサで動作します。X8i インスタンスは SAP 認証を取得しており、クラウド上の同等の Intel プロセッサの中で最高の性能と最速のメモリ帯域を提供します。パフォーマンスは最大で 43% 高く、メモリ容量は 1.5 倍（最大 6TB）となります。
