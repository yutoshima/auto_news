# Apache Spark のトラブルシューティングとアップグレードのエージェントが、Kiro が提供する機能として利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月03日 21:49
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-emr-spark-troubleshooting-upgrade-kiro-power/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-emr-spark-troubleshooting-upgrade-kiro-power/)

## 概要

<p>Apache Spark のトラブルシューティングエージェントと Amazon EMR のアップグレードエージェントが、Kiro のパワーとして利用可能になりました。これにより、AI 支援の Spark 操作へワンクリックでアクセスできます。これらのパワーを使えば、データエンジニアはトラブルシューティング時間を数時間から数分へ短縮し、Spark バージョンのアップグレードを数か月から数週へ圧縮できます。<br /> <br /> Spark ジョブが失敗した場合、トラブルシューティングパワーは、EC2 上の EMR と EMR Serverless のログ、指標、および設定を横断して分析し、根本原因を特定します。
