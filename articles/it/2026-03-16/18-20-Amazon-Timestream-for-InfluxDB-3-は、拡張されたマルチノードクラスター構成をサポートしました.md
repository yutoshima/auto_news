# Amazon Timestream for InfluxDB 3 は、拡張されたマルチノードクラスター構成をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月16日 18:20
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-timestream-for-influxdb-3-multi-node-cluster/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-timestream-for-influxdb-3-multi-node-cluster/)

## 概要

<p>InfluxDB 3 Enterprise Edition向けのAmazon Timestreamは、要求の高い本番ワークロードに対する高い読み取りスループットと高可用性を実現するため、InfluxDB 3 Enterprise Editionの拡張されたマルチノードクラスタ構成をサポートします。最大15ノードのクラスタを構築でき、データ取り込みとクエリ用に1～4のライター/リーダーノード、クエリ性能を拡張するための専用リーダーのみノードを0～13個、さらに専用ノードを追加する構成です。</p>
