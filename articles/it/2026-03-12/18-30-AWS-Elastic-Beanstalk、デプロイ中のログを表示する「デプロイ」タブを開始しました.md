# AWS Elastic Beanstalk、デプロイ中のログを表示する「デプロイ」タブを開始しました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月12日 18:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/elastic-beanstalk-deployments-tab/](https://aws.amazon.com/about-aws/whats-new/2026/03/elastic-beanstalk-deployments-tab/)

## 概要

AWS Elastic Beanstalkは現在、環境ダッシュボードにデプロイmentsタブを提供しており、顧客はデプロイの履歴とリアルタイムのデプロイ進行状況を、ステップバイステップのデプロイログと共に一元的に確認できます。従来はデプロイが完了するまでログを取得できず、複数の情報源のイベントを照合して何が起きたかを把握する必要がありました。この新機能により、顧客はデプロイの状況、イベント、詳細なログを1つのインターフェースで直接確認できます。
