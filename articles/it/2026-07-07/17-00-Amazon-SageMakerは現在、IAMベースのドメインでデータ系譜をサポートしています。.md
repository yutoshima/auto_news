# Amazon SageMakerは現在、IAMベースのドメインでデータ系譜をサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月07日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-sagemaker-data-lineage-iam-domain](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-sagemaker-data-lineage-iam-domain)

## 概要

<p>Amazon SageMaker Unified Studio は、IAM ベースのドメインで OpenLineage に準拠したデータ系譜をサポートし、Amazon EMR、AWS Glue、SageMaker Visual ETL、ノートブックで実行される Apache Spark ジョブからの系譜イベントをキャプチャします。この機能はすでに IAM Identity Center ベースのドメインでも利用可能です。対話型の系譜グラフは、データが出所から消費へと移動する様子を集約して可視化し、グラフの深さやイベントのタイムスタンプモードを設定して、列レベルの系譜を詳細に表示できます。
