# AWS Glue  Interactive Sessions は、対話的ワークロードに対して Spark Connect をサポートするようになりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月17日 19:15
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-glue-interactive-sessions-spark-connect-smus-notebooks](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-glue-interactive-sessions-spark-connect-smus-notebooks)

## 概要

AWS Glue Interactive Sessions は現在 Apache Spark Connect をサポートしており、これを使用すると、Amazon SageMaker Unified Studio のマネージドノートブックや Jupyter、Visual Studio Code などの好みのノートブック環境や IDE で、AWS Glue のサーバーレスなインフラストラクチャ上でクラスターを管理することなく、好きな環境から Apache Spark アプリケーションを開発・実行できます。  
Spark Connect を使えば、薄いクライアントを介して AWS Glue Interactive Sessions に Spark ジョブを送信します。
