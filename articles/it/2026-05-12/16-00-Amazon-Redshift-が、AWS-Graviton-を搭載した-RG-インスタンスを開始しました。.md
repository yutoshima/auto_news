# Amazon Redshift が、AWS Graviton を搭載した RG インスタンスを開始しました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月12日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-redshift-rg-instances-powered-by-graviton](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-redshift-rg-instances-powered-by-graviton)

## 概要

Amazon Redshiftが、AWS Gravitonプロセッサを搭載した新世代のプロビジョニング済みクラスタノードである_RGインスタンス_の一般提供を開始したことを発表します。従来世代のRA3インスタンスと比較して、データウェアハウスおよびデータレイクのワークロードを最大2.4倍の高速で実行し、vCPUあたりの価格を30%低く抑えます。RGインスタンスには、Apache IcebergとParquetのデータを処理する、Redshiftが独自に構築したベクトル化データレイククエリエンジンが含まれています。
