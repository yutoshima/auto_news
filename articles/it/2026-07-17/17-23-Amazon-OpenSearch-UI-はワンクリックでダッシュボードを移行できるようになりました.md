# Amazon OpenSearch UI はワンクリックでダッシュボードを移行できるようになりました

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月17日 17:23
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-opensearch-ui-one-click-dashboard-migration](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-opensearch-ui-one-click-dashboard-migration)

## 概要

<p>Amazon OpenSearch Service は、従来の OpenSearch Dashboards から OpenSearch UI へのワンクリック移行を、OpenSearch ドメインおよびサーバーレスコレクションの両方に対応しました。OpenSearch UI は、検索と複数のデータソースにまたがる一元的な可観測性を提供する、新しいゼロダウンタイムのサーバーレスインターフェースです。このリリースにより、従来の OpenSearch Dashboards で作成した複数のテナントや数千の保存済みオブジェクトが OpenSearch UI アプリケーションで再利用可能となり、運用の複雑さを軽減します。
