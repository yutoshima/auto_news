# AWS Marketplaceがカタログデータへのプログラム的アクセスを可能にするDiscovery APIを発表しました

## 概要

<p>本日、AWS MarketplaceはDiscovery APIを発表しました。これにより、SaaS、AIエージェントとツール、AMI、コンテナ、機械学習モデルを含むAWS Marketplaceカタログ全体の製品情報と価格情報へプログラムでアクセスできます。<br /> <br /> Discovery APIを使えば、購入者はカタログデータを内部ポータルに組み込み、現在の価格と提供条件で調達ツールを強化し、ベンダー評価のワークフローを合理化できます。販売者とチャネルパートナーは、製品リストを表示し、公開することができます。
