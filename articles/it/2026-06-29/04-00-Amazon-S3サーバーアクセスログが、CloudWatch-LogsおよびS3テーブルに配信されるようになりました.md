# Amazon S3サーバーアクセスログが、CloudWatch LogsおよびS3テーブルに配信されるようになりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月29日 04:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-s3-cloudwatch-logs-tables/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-s3-cloudwatch-logs-tables/)

## 概要

<p>Amazon S3は現在、サーバーアクセスログをAmazon CloudWatch Logsへ配信する機能をサポートしており、即時のクエリ、アラーム、アカウント間およびリージョン間の集計、そしてアクセスログデータのAWS Key Management Service（KMS）暗号化を提供します。さらに、追加のストレージコストなしで、Apache Iceberg形式のAmazon S3 Tablesへログをミラーリングすることもできます。これらの新しい配信パスは、S3の一般用途バケットへの無料配信という既存のオプションを補完し、ログの取り扱い方法における柔軟性を高めます。
