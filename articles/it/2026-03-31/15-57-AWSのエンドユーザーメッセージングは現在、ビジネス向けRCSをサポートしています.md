# AWSのエンドユーザーメッセージングは現在、ビジネス向けRCSをサポートしています

## 概要

<p>本日、AWS End User Messagingはビジネス向けリッチコミュニケーションサービス（RCS）をサポートすることを発表します。従来のSMSは汎用の電話番号から送信されるため、顧客が認識しにくく、企業が信頼を築くのが難しくなります。RCSは、顧客がすでに使用している同じメッセージングアプリを通じてメッセージを配信し、名前とロゴを表示する検証済みのビジネスIDを提供することでこれを解決します。AWS End User Messagingは、RCSをアプリケーションへ組み込むための、スケーラブルで費用対効果の高い方法を提供します。
