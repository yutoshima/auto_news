# Amazon EC2専用ホストがAMD SEV-SNPに対応しました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月02日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/ec2-amd-sev-snp-dedicated-hosts](https://aws.amazon.com/about-aws/whats-new/2026/07/ec2-amd-sev-snp-dedicated-hosts)

## 概要

Amazon EC2は、専有ホスト上でAMD Secure Encrypted Virtualization-Secure Nested Paging（SEV-SNP）をサポートすることを発表します。これにより、機密コンピューティング作業負荷を、他の顧客と共有されず、完全に専有された物理サーバー上で実行できます。

顧客は、SEV-SNPが有効な専有ホストを割り当て、それ上でSEV-SNPインスタンスを起動できます。これにより、機密コンピューティング作業負荷に対して専有ホストの利点（インスタンス配置の制御など）を享受でき、ホストの資源を完全に専有できます。
