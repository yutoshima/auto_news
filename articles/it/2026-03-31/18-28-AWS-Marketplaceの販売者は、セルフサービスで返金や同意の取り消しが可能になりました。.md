# AWS Marketplaceの販売者は、セルフサービスで返金や同意の取り消しが可能になりました。

- **カテゴリ**: it
- **公開日時**: 2026年03月31日 18:28
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-marketplace-seller-self-service-refund-cancel/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-marketplace-seller-self-service-refund-cancel/)

## 概要

AWS Marketplaceは現在、出品者向けの返金および同意取消のセルフサービスプロセスを合理化し、これらのリクエストの処理に要する時間と労力を削減します。この新機能によりサポートチケットの提出が不要になり、出品者と購入者の双方が各リクエストの最新状況を完全に把握できるようになります。購入者はAWS Marketplaceのコンソールから取消リクエストを直接確認・承認できるようになり、請求明細に返金が反映されることで照合が容易になります。
