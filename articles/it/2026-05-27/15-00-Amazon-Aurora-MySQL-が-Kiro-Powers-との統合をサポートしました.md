# Amazon Aurora MySQL が Kiro Powers との統合をサポートしました

## 概要

<p>本日、AWSは Amazon Aurora MySQL-互換エディションが <a href="https://kiro.dev/powers/">Kiro Powers</a> との統合をサポートすることを発表しました。これにより、AIエージェントの支援を活用して、Aurora MySQLをバックエンドに持つアプリケーションをより迅速に構築できるようになります。Kiro Powers は、Kiro のパートナーによって検証され、特化型のソフトウェア開発とデプロイを加速させるための、整理された事前パッケージ化済みの Model Context Protocol (MCP) サーバー、ステアリングファイル、フックのリポジトリです。この統合は direc をバンドルします。
