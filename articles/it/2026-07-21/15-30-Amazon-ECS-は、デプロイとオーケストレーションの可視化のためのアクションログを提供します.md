# Amazon ECS は、デプロイとオーケストレーションの可視化のためのアクションログを提供します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月21日 15:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ecs-action-logs/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ecs-action-logs/)

## 概要

今日は、<a href="https://aws.amazon.com/ecs/">Amazon Elastic Container Service</a>（Amazon ECS）が新しい可観測性機能「Action Logs」を発表します。これは、サービスのデプロイ時や ECS Managed Daemon の更新時に、顧客の代わりに Amazon ECS が実行するアクションの詳細でタイムスタンプ付きの記録を提供します。これまで見えなかったサービス側のオペレーションを可視化することで、Action Logs は AWS Supportに連絡したり、手動で相関付けを行うことなく、直接ワークロードを監視・トラブルシュートするのに役立ちます。
