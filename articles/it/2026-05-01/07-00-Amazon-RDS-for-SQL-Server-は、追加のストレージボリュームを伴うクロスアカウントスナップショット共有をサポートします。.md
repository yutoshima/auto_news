# Amazon RDS for SQL Server は、追加のストレージボリュームを伴うクロスアカウントスナップショット共有をサポートします。

## 概要

<a href="https://aws.amazon.com/rds/sqlserver/">Amazon Relational Database Service (Amazon RDS) for SQL Server</a> は、追加ストレージボリュームを備えたデータベースインスタンスのためのクロスアカウントスナップショット共有をサポートしました。追加ストレージボリュームにより、プライマリのストレージボリュームに加えて、最大3つのストレージボリューム（各最大64 TiB）を追加することで、データベースのストレージ容量を最大256 TiBにスケールできます。この新機能により、顧客はデータベーススナップショットを作成・共有・コピーできるようになります。 AWSアカウント間での共有、コピーが可能です。
