# AWS Entity Resolutionは現在、高度なリアルタイムマッチングをサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月22日 19:00

## 概要

<AWS Entity Resolutionは現在、進化したマッチングワークフローを用いたリアルタイムマッチングをサポートしており、GenerateMatchId APIを通じて複雑なルールセットを使用してミリ秒単位でレコードをマッチさせることが可能です。以前は、リアルタイムマッチングは単純なルールベースのワークフローに限定されており、ExactやExactManyToManyなどの演算子をAND/OR論理と組み合わせた高度なルールセットは、分刻みから時間単位のバッチ処理でのみ利用できました。これは、再マッチングを必要とする顧客にとって重大なギャップを生んでいました。
