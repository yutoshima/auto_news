# Amazon Braket が Rigetti の 108-量子ビット Cepheus QPU をサポート追加

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月07日 09:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-braket-rigetti-cepheus/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-braket-rigetti-cepheus/)

## 概要

< p > Amazon Braket、AWS の量子計算サービスは現在、Rigetti の Cepheus-1-108Q デバイスへのアクセスを提供しており、Amazon Braket で利用できる最初の 100+ 量子ビット超伝導量子処理ユニット (QPU) です。 Cepheus-1-108Q は Rigetti のモジュラ・マルチチップ構造を採用しており、 tunable couplers およびチップレット間のモジュール間カプラを備えた、12 個の 9-量子ビット・チップレットを 3x4 の配列で構成しています。 </p> 
<p> Cepheus-1-108Q は CZ（制御位相）ゲートを導入し、以前の Rigetti において使用されていた iSWAP ゲートを置き換えています。 </p>
