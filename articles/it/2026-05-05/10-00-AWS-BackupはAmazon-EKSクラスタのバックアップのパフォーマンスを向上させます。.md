# AWS BackupはAmazon EKSクラスタのバックアップのパフォーマンスを向上させます。

- **カテゴリ**: it
- **公開日時**: 2026年05月05日 10:00

## 概要

<p>Amazon EKS向けのAWS Backupは、クラスタ状態のバックアップを従来より最大10倍の速度で完了します。このパフォーマンス向上により、名前空間やKubernetesリソースが多数あるAmazon EKSクラスタのバックアップを大幅に迅速化し、最大規模のクラスタではバックアップウィンドウを日単位から数時間へと短縮します。AWS Backupは、ポリシーに基づく完全管理型でコスト効率の高いソリューションであり、計算リソースを跨ぐ他のAWSサービスとともに、Amazon EKSのデータ保護を集中管理・自動化します。
