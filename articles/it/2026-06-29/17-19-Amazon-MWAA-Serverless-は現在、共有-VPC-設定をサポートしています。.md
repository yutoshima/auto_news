# Amazon MWAA Serverless は現在、共有 VPC 設定をサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月29日 17:19
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-mwaa-serverless-vpc/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-mwaa-serverless-vpc/)

## 概要

<p>Amazon Managed Workflows for Apache Airflow (Amazon MWAA) Serverless は、共有VPCサブネットをサポートするようになりました。これまで、AWS RAM（AWS Resource Access Manager）を介してサブネットを共有している顧客は、MWAA Serverless のワークフローを作成する際に検証エラーを受けていました。この更新により、MWAA Serverless は共有VPC構成におけるサブネットの所有権を正しく検証し、MWAA Provisioned 環境と一貫性を保ちます。<br /> AWS RAM を使用してアカウント間でVPCサブネットを共有することは、マルチアカウント環境で一般的なパターンです。
