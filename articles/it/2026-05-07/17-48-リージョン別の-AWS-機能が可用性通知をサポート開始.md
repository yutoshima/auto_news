# リージョン別の AWS 機能が可用性通知をサポート開始

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月07日 17:48
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-regional-planning-tool-notification](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-regional-planning-tool-notification)

## 概要

本日、AWSは、AWS Builder Center内の新しいサブスクリプション型システム「AWS Capabilities by Region」の提供通知を開始します。このシステムは、対象リージョンでAWSサービスおよび/または機能が利用可能になったときに、自動的にビルダーへ通知します。利用可能通知により、1,500以上のサービスと機能の利用可能性を37のAWSリージョン全体で追跡することが容易になり、インフラストラクチャの構築を加速します。
