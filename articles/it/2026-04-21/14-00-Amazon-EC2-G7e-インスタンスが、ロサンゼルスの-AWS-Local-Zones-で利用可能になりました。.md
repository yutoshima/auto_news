# Amazon EC2 G7e インスタンスが、ロサンゼルスの AWS Local Zones で利用可能になりました。

- **カテゴリ**: it

## 概要

本日、AWSはロサンゼルス（カリフォルニア州）にある AWS Local Zones において、一般利用可能となった <a href="https://aws.amazon.com/ec2/instance-types/g7e/" rel="noopener noreferrer" target="_blank">Amazon Elastic Compute Cloud (Amazon EC2) G7e インスタンス</a> を発表します。G7e インスタンスは NVIDIA RTX PRO 6000 Blackwell Server Edition GPU と第5世代 Intel Xeon Scalable（Emerald Rapids）プロセッサを搭載し、ロサンゼルスのエンドユーザーに高性能な GPU コンピューティングをより近づけます。
