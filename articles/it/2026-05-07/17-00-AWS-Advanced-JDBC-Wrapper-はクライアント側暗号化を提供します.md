# AWS Advanced JDBC Wrapper はクライアント側暗号化を提供します

## 概要

<p>AWS Advanced JDBC Wrapper は、KMS Encryption プラグインを通じて列レベルのクライアント側暗号化を提供します。ラッパーはフェイルオーバー処理、AWS 認証の統合、そして Amazon Aurora および Amazon RDS のオープンソースデータベース向けの高度な監視機能といった先進的な機能を提供します。これにより、Java アプリケーションのコードを変更することなく、データベースへ到達する前に機微なデータを暗号化することが可能になります。<br /> <br /> データの静止時暗号化と転送中の TLS は基盤となるセキュリティ要件です。
