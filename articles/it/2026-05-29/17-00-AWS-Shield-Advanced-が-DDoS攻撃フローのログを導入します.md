# AWS Shield Advanced が DDoS攻撃フローのログを導入します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月29日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-shield-ddos/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-shield-ddos/)

## 概要

AWS Shield Advanced は、DDoS（分散型サービス拒否）攻撃のフローログを提供します。これにより、DDoS攻撃中に Shield Advanced が保護するリソースへ向かうトラフィックをパケットレベルで可視化できます。ログデータは、フォレンジック分析とコンプライアンス目的のために Amazon S3、Amazon CloudWatch Logs、または Amazon Data Firehose に公開されます。

このDDoS攻撃フローログは、ソースおよびデスティネーションのIPアドレス、ポート、プロトコル、パケット数とバイト数など、重要なパケットレベルの詳細をキャプチャします。
