# AWS Cost Optimization HubがCSVダウンロードに対応しました

## 概要

<p>AWS Cost Optimization Hub は現在、コンソールで直接 CSV をダウンロードできるようになり、コスト最適化の推奨事項をワンクリックでローカル端末へエクスポートできます。この機能は、コンソールからのワンクリックエクスポートオプションを提供するだけでなく、自動エクスポートを Amazon S3 へ行う既存の Data Export 機能を補完します。</p> 
<p>CSV ダウンロードを使用すると、現在のコンソールのフィルター、並べ替えの設定、およびグループ化設定を使用した推奨事項を即座にエクスポートできます。
