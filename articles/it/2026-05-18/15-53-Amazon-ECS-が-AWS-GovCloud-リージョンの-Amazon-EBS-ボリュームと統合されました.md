# Amazon ECS が AWS GovCloud リージョンの Amazon EBS ボリュームと統合されました

## 概要

<p><a href="https://aws.amazon.com/ecs/">Amazon Elastic Container Service</a>（Amazon ECS）は、AWS GovCloud RegionsのAmazon Elastic Block Store（Amazon EBS）ボリュームと統合されました。この機能により、ETLジョブ、メディアのトランスコーディング、機械学習の推論ワークロードなど、ストレージやデータ集約型アプリケーションをECSを使ってより容易にデプロイできます。<br /><br /> Amazon ECSタスクでEBSボリュームを使用するには、タスク定義内でEBSボリュームをマウントしたいパスを設定するだけです。
