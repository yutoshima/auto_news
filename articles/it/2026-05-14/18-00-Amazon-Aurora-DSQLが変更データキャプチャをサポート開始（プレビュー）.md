# Amazon Aurora DSQLが変更データキャプチャをサポート開始（プレビュー）

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月14日 18:00

## 概要

<p>Amazon Aurora DSQL は CDC（変更データキャプチャ）のプレビュー機能を導入し、リアルタイムのデータベース変更を直接 Amazon Kinesis Data Streams へストリーミングすることを可能にします。これにより、カスタムのストリーミングパイプラインを構築・維持する必要がなくなり、イベント駆動型アプリケーションの構築、リアルタイム分析パイプラインの実現、システム間でのデータ同期が容易になります。<br /> <br /> Aurora DSQL は INSERT、UPDATE、DELETE 操作の結果を自動的にキャプチャします。
