# AWS Security Hub CSPMが、31の自動化コントロールを備えたAIセキュリティベストプラクティス標準を開始します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月30日 16:17
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-security-hub-cspm-ai-security/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-security-hub-cspm-ai-security/)

## 概要

本日、AWS Security Hub CSPMはAIセキュリティベストプラクティス標準を発表します。これは、展開したAIリソースがセキュリティベストプラクティスと一致していない場合を検出する31の自動化されたセキュリティコントロールのセットです。AWSのセキュリティ専門家によって開発されたこの標準は、Amazon Bedrock、Amazon Bedrock AgentCore、および Amazon SageMaker のワークロードを推奨されるセキュリティ構成と継続的に評価するのに役立ちます。手動評価やカスタムルールの作成を必要とせずに進められます。
