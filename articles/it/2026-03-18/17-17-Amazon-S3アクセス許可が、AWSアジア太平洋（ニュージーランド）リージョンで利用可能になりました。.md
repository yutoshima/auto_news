# Amazon S3アクセス許可が、AWSアジア太平洋（ニュージーランド）リージョンで利用可能になりました。

## 概要

<p>現在、AWS アジア太平洋（ニュージーランド）リージョンで Amazon S3 アクセス許可を作成できます。</p>
<p>Amazon S3 アクセス許可は、Microsoft Entra ID などのディレクトリのアイデンティティや AWS Identity and Access Management（IAM）プリンシパルを、S3 のデータセットに対応付けます。これにより、企業のアイデンティティに基づいてエンドユーザーに自動的に S3 アクセスを付与することで、データ権限の管理をスケールさせることができます。</p>
<p>訪問先: <a href="https://docs.aws.amazon.com/AmazonS3/latest/userguide/access-grants-limitations.html">https://docs.aws.amazon.com/AmazonS3/latest/userguide/access-grants-limitations.html</a></p>
