# AWS DevOps Agent が一般提供を開始しました

## 概要

現在一般公開されている AWS DevOps Agent は、常時利用可能な運用の仲間として、インシデントを解決・予防し、アプリケーションの信頼性とパフォーマンスを最適化し、AWS、マルチクラウド、オンプレ環境でオンデマンドの SRE タスクを処理します。プレビュー版のリリースを経て、DevOps Agent は新たなユースケースの追加、より広範な統合、強化された知能、エンタープライズ向け機能を提供します。これには、Azure やオンプレ環境のアプリケーションを調査できる機能も含まれます。
