# AWS BackupがAmazon FSxのサポートを5つの追加AWSリージョンに拡張し、クロスリージョンおよびクロスアカウントのコピーを14のAWSリージョンへ拡大します。

## 概要

<パラグラフ> AWS Backup は、Amazon FSx for Windows File Server、Amazon FSx for OpenZFS、Amazon FSx for Lustre のサポートを2つの地域拡張で拡大します。 </パラグラフ>
<パラグラフ> まず、AWS Backup はこれらの FSx ファイルシステムのバックアップと復元を、追加の5つの AWS リージョンでサポートするようになりました：アジア太平洋（マレーシア、台北、タイ）、カナダ西部（カルガリー）、メキシコ（中部）。これらのリージョンで、FSx のバックアップポリシーを一元管理し、バックアップスケジュールを自動化し、バックアップのアクティビティを AWS Backup で監視できるようになりました。 </パラグラフ>
