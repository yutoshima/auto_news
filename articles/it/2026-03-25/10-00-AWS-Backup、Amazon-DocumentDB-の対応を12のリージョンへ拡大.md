# AWS Backup、Amazon DocumentDB の対応を12のリージョンへ拡大

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月25日 10:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-backup-amazon-documentdb-regions/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-backup-amazon-documentdb-regions/)

## 概要

AWS Backupは現在、Amazon DocumentDB を12の追加AWSリージョンでサポートします：アジア太平洋（マレーシア、タイ、オーサカ、香港、ジャカルタ、メルボルン）、ヨーロッパ（ストックホルム、スペイン、チューリッヒ）、アフリカ（ケープタウン）、イスラエル（テルアビブ）、メキシコ（セントラル）。

この拡張により、これらの新しくサポートされたリージョンで、ポリシーベースのデータ保護と復旧をAmazon DocumentDBクラスタに提供します。

DocumentDBクラスタをAWS Backupで保護を開始するには、既存のクラスタにDocumentDBクラスタを追加します。
