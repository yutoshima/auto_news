## 概要

本日から、AWS Security Agent（現在は AWS Continuum の一部）は、以下の3つの追加 AWS リージョンで利用可能です：アジア太平洋（ムンバイ）、アジア太平洋（シンガポール）、南米（サンパウロ）。これらのリージョンの顧客は、開発ライフサイクル全体を通じてアプリケーションをプロアクティブに保護するための Security Agent のコア機能にアクセスできるようになります。  
  
この拡張により、顧客は STRIDE ベースの脅威モデリング（プレビュー）を利用できるようになります。
