# Amazon SageMaker Data Agent が現在、日本とオーストラリア向けの地理特化型推論をサポートしています

- **カテゴリ**: it

## 概要

<p>Amazon SageMaker Data Agent は、Amazon Bedrock を通じて日本およびオーストラリア向けのクロスリージョン推論プロファイルをサポートします。今回の更新により、アジア太平洋（東京）およびアジア太平洋（シドニー）地域の Data Agent からの推論リクエストは、それぞれの地理的領域内で処理され、日本およびオーストラリアの顧客向けのデータ所在要件をサポートします。</p>
<p>Data Agent は、データ探索、Python および SQL コード生成、トラブルシューティングのための AI 搭載の対話型エクスペリエンスを提供します。</p>
