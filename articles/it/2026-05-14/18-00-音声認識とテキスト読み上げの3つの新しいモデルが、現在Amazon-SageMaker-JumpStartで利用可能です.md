# 音声認識とテキスト読み上げの3つの新しいモデルが、現在Amazon SageMaker JumpStartで利用可能です

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月14日 18:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/speech-models-on-sagemaker-jumpstart/](https://aws.amazon.com/about-aws/whats-new/2026/05/speech-models-on-sagemaker-jumpstart/)

## 概要

今日は、AWSが Qwen3-TTS-12Hz-1.7B-CustomVoice、Qwen3-TTS-12Hz-1.7B-Base、Qwen3-ASR-1.7B を Amazon SageMaker JumpStart で利用可能にしたと発表しました。これにより、AWSのお客様向けのファウンデーションモデルのポートフォリオが拡大されます。Qwen のこの3モデルは、10言語以上にわたる高度な音声合成と認識機能を提供し、顧客が AWS インフラストラクチャ上でインテリジェントな音声対応アプリケーションを構築できるようにします。

これらのモデルは、企業向けのさまざまな音声・オーディオの課題に対応します。
