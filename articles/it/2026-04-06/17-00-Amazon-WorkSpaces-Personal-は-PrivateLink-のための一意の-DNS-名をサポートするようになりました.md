# Amazon WorkSpaces Personal は PrivateLink のための一意の DNS 名をサポートするようになりました

## 概要

<p>Amazon WorkSpaces Personal は現在、各 AWS PrivateLink Virtual Private Cloud (VPC) エンドポイントに対して固有で公開解決可能なドメイン名系 (DNS) 名を提供しており、企業のお客様が DNS 解決の競合を回避しつつ複数の AWS VPC およびアカウントにまたがって WorkSpaces を展開できるようにします。各インターフェイス VPC エンドポイントには、従来のエンドポイント全体で共有されていた汎用 DNS 名に加えて、グローバルに一意の AWS 管理 DNS 名が割り当てられます。<br /> <br /> この強化により、顧客は次のことが可能になります。
