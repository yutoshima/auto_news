# AWS Shield AdvancedがDDoS攻撃フローのログを導入します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月29日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-shield-ddos/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-shield-ddos/)

## 概要

AWS Shield Advancedは、分散型サービス拒否攻撃（DDoS）攻撃フローログを発表します。これにより、DDoS攻撃中にShield Advancedで保護されたリソースに向かうトラフィックをパケットレベルで可視化できます。ログデータは、法医学分析とコンプライアンス目的のため、Amazon S3、Amazon CloudWatch Logs、または Amazon Data Firehose に公開されます。

DDoS攻撃フローログは、ソースおよびデスティネーションのIPアドレス、ポート、プロトコル、パケット数とバイト数など、重大なパケットレベルの詳細をキャプチャします。
