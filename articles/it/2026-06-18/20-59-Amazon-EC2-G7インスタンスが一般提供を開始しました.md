# Amazon EC2 G7インスタンスが一般提供を開始しました

## メタデータ

- **カテゴリ**: it
- **公開日時**: 2026年06月18日 20:59
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-g7-generally-available](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-g7-generally-available)

## 概要

本日、AWSは NVIDIA RTX PRO 4500 Blackwell Server Edition GPU によって加速される Amazon Elastic Compute Cloud（Amazon EC2） G7 インスタンスの一般提供を発表します。G7 インスタンスは、G6 と比較してAI 推論性能を最大4.6倍、グラフィックス性能を最大2.1倍向上させます。

G7 インスタンスは、言語翻訳、動画・画像分析、音声認識、レコメンダーシステムなどのAI推論ワークロードにご利用いただけます。さらに、G7 インスタンスはグラフィックス作業も加速します。
