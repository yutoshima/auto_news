# Amazon EC2 P6-B300 インスタンスが米国東部（ノースバージニア）リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月06日 19:24
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-p6-b300-us-east](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-ec2-p6-b300-us-east)

## 概要

本日より、Amazon Elastic Cloud Compute（Amazon EC2）P6-B300インスタンスがUS East（N. Virginia）リージョンで利用可能となります。P6-B300インスタンスは、2.1 TBの高帯域幅GPUメモリ、6.4 TbpsのEFAネットワーキング、300 Gbpsの専用ENAスループット、4 TBのシステムメモリを備えた8基のNVIDIA Blackwell Ultra GPUを提供します。P6-B300インスタンスは、P6-B200インスタンスと比較してネットワーク帯域幅を2倍、GPUメモリサイズを1.5倍、GPU TFLOPS（FP4時、スパース性なし）を1.5倍向上させており、優れた性能を発揮します。
