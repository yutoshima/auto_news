# AWS、請求グループのアカウント向けに Cost Explorer の履歴データ保持を開始

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月15日 16:07
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-cost-explorer/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-cost-explorer/)

## 概要

本日、AWSは請求グループに所属するアカウントのCost Explorer歴史データ保持を発表します。  
顧客はAWS Billing ConductorとBilling Transferを使用してアカウントを請求グループに紐づけることができ、支払アカウントまたはBill-Transferアカウントが提供する仮想レートで価格設定された請求データを閲覧できるようになります。これまでは、請求グループの設定により、請求グループにマッピングされたアカウントの歴史的な請求データ（AWSの課金レートでの価格設定）へのアクセスが制限されていました。
