# AWS Artifactが、コンプライアンスに関する問い合わせのためのAssurance Assistantを搭載しました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月01日 20:01
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/aws-artifact-assurance-assistant/](https://aws.amazon.com/about-aws/whats-new/2026/07/aws-artifact-assurance-assistant/)

## 概要

<p>AWS Artifact は現在、Assurance Assistant を含みます。これは AI 搭載の機能で、AWS のサービスに関するセキュリティとコンプライアンスの質問に対して、出典付きの回答を生成します。AWS Artifact は、AWS が顧客にコンプライアンスレポート、認証、契約を提供するサービスです。Assurance Assistant は、サードパーティのリスク管理担当者、コンプライアンス担当者、セキュリティエンジニア、監査人が、ベンダー評価とデューデリジェンス質問票（DDQ）の完成を、出典付きの情報を提供することで加速させるのに役立ちます。
