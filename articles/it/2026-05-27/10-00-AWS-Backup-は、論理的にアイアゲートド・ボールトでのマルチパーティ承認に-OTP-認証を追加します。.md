# AWS Backup は、論理的にアイアゲートド・ボールトでのマルチパーティ承認に OTP 認証を追加します。

## メタデータ

- **カテゴリ**: it
- **公開日時**: 2026年05月27日 10:00

## 概要

AWS Backup は、論理的にアイアップグレッドされたボールトに対する複数承認アクションの承認者が投票する場合に、ワンタイムパスワード（OTP）認証を要求します。

承認者が複数承認リクエストに投票する際には、AWS IAM Identity Center に登録されたメールアドレス宛に送信される六桁のコードを入力する必要があります。これにより、認証済みの承認者のみが保護されたボールト操作を承認できるようになり、承認チームに対するセキュリティの追加レイヤーが提供されます。OTP認証は自動的に適用されます。
