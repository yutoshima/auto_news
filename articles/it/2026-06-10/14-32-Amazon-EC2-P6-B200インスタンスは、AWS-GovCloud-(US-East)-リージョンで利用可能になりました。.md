# Amazon EC2 P6-B200インスタンスは、AWS GovCloud (US-East) リージョンで利用可能になりました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月10日 14:32
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-p6-b200-aws-govcloud/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-p6-b200-aws-govcloud/)

## 概要

本日より、NVIDIA Blackwell GPU（P6-B200）によって高速化された Amazon Elastic Compute Cloud（Amazon EC2）P6-B200 インスタンスが AWS GovCloud (US-East) リージョンで利用可能になります。これらのインスタンスは、AI のトレーニングと推論において P5en インスタンスと比較して最大2倍の性能を提供します。P6-B200 インスタンスは、8基の Blackwell GPU、1440 GB の高帯域幅 GPU メモリ、P5en、第5世代 Intel Xeon プロセッサ（Emerald Rapids）と比較して GPU メモリ帯域幅を60%向上させ、最大 3.2 テラビット/秒を実現します。
