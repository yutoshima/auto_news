# AWS Backupは復元テストのサポートを6つの追加AWSリージョンに拡張します

## 概要

AWS Backup のリストアテストが、アジア太平洋（台北）、アジア太平洋（マレーシア）、アジア太平洋（ニュージーランド）、アジア太平洋（タイ）、メキシコ（中部）、カナダ西部（カルガリー）の6つの追加リージョンで利用可能になりました。

リストアテストは、ストレージ、コンピュート、データベースサービスにわたるサポート対象リソースのリストアテストを自動化し、定期的に実行するのに役立ちます。リストアテスト計画を作成して、回復点を自動的に選択し、スケジュールに従ってリストアを実行し、リストアの測定を行うことができます。
