# AWS Batchは現在、顧客注文のインスタンス割り当て戦略をサポートしています。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月22日 16:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/batch-ordered-allocation-strategies/](https://aws.amazon.com/about-aws/whats-new/2026/06/batch-ordered-allocation-strategies/)

## 概要

<p>AWS Batch は現在、Best Fit Progressive Ordered（BFPO）と Spot Capacity Optimized Prioritized（SCOP）割り当て戦略を提供しており、計算環境におけるインスタンスタイプの優先度をより細かく制御できます。BFPO および SCOP を使用すると、ワークロードの特性に基づいてインスタンスタイプの並び順を手動で定義できます。<br /><br /> AWS Batch でこれらの機能を利用するには、オンデマンド計算環境の割り当て戦略として BEST_FIT_PROGRESSIVE_ORDERED を指定します。</p>
