# Amazon OpenSearch Service は現在、JWT 認証のための JWKS URL 設定をサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月29日 02:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-opensearch-jwks-url-jwt/](https://aws.amazon.com/about-aws/whats-new/2026/04/amazon-opensearch-jwks-url-jwt/)

## 概要

<p>Amazon OpenSearch Serviceは現在、JWT認証のためのJSON Web Key Set（JWKS）URL設定をサポートしています。JWKS URLをJWT認証設定の一部として構成することで、OpenSearchドメインがIDプロバイダーのJWKSエンドポイントから公開鍵を自動的に取得・検証できるようになります。</p>
<p>従来、JWT認証では静的な公開鍵を手動で構成・更新する必要がありました。JWKS URLのサポートにより、ドメインは最新の公開鍵を自動的に取得します。</p>
