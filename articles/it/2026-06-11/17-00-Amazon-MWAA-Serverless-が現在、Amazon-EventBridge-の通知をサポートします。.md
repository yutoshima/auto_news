# Amazon MWAA Serverless が現在、Amazon EventBridge の通知をサポートします。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月11日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-mwaa-serverless-eventbridge/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-mwaa-serverless-eventbridge/)

## 概要

<p>Amazon Managed Workflows for Apache Airflow（MWAA）Serverlessは現在、ワークフローおよびタスクの状態変更イベントを <a href="https://aws.amazon.com/eventbridge/">Amazon EventBridge</a> に対応させ、データエンジニアリングやプラットフォームチームが Apache Airflow のワークフローに対するイベント駆動型の自動化を構築できるようになりました。</p> 
<p>以前は、ワークフローの実行状況を監視するにはカスタムポーリングロジックや手動での観察が必要でした。今回のリリースにより、MWAA Serverless はワークフローが状態を遷移する際にイベントを発行できるようになります。</p>
