# Amazon Connectのアウトバウンドキャンペーンが複数連絡先の時差検出機能を追加

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月07日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-connect-campaign-multitimezone](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-connect-campaign-multitimezone)

## 概要

<p>Amazon Connect Outbound Campaigns は、顧客のタイムゾーンを、主要な連絡先フィールドだけでなく、顧客プロファイル上のすべての電話番号と住所を使用して検出します。以前はタイムゾーン検出は主電話番号のみを使用していたため、複数のタイムゾーンにまたがる顧客を見逃す可能性がありました。</p>
<p>プロフィールの連絡先情報が複数のタイムゾーンにまたがる場合、システムは、検出されたすべてのタイムゾーンのうち、設定したウィンドウ内に該当する時刻のみで配信します。</p>
