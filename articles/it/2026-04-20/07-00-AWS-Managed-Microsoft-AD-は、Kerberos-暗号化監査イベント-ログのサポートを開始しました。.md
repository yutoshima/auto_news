# AWS Managed Microsoft AD は、Kerberos 暗号化監査イベント ログのサポートを開始しました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月20日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/aws-managed-microsoft-ad-kerberose-encrption-logs/](https://aws.amazon.com/about-aws/whats-new/2026/04/aws-managed-microsoft-ad-kerberose-encrption-logs/)

## 概要

本日より、AWS Managed Microsoft AD は Kerberos 暗号化監査イベント ログ（イベント ID 201～209）の Amazon CloudWatch Logs への転送をサポートします。これらのログは、アプリケーションやサービスが使用している暗号化方式の可視化を提供し、RC4 暗号化と AES 暗号化のどちらを使用しているリソースかを特定するのに役立ちます。この可視化により、クライアントを AES 暗号化へアップグレードする（セキュリティ向上のため推奨）か、RC4 のサポートを維持するかを判断できます。
