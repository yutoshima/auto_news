# AWS Direct Connect が BGP モニタリングのための CloudWatch 指標を追加します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月30日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-direct-connect-cloudwatch-bgp-monitoring/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-direct-connect-cloudwatch-bgp-monitoring/)

## 概要

AWS Direct Connect は現在、仮想インターフェイス（VIF）向けの新しい3つの Amazon CloudWatch 指標を公開しました。これにより、Border Gateway Protocol（BGP）セッションの健全性とルート数を可視化できるようになります。ハイブリッドクラウド接続を管理するネットワークエンジニアや運用チームは、CloudWatch を通じてネイティブに BGP セッションを監視できるため、カスタムソリューションの構築や API のポーリングが不要になります。

これらの指標は、従来はカスタム Lambda 函数やオンプレミスのネットワーク監視で対処していた重要な監視ギャップを解消します。
