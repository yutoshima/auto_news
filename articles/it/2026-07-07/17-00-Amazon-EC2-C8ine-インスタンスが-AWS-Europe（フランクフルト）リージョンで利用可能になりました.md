# Amazon EC2 C8ine インスタンスが AWS Europe（フランクフルト）リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月07日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-c8ine-aws-frankfurt/](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-ec2-c8ine-aws-frankfurt/)

## 概要

本日より、Amazon Elastic Compute Cloud（Amazon EC2）C8ine インスタンスが AWS Europe (Frankfurt) リージョンで利用可能になります。C8ine インスタンスは、AWS 専用のカスタム第6世代 Intel Xeon Scalable プロセッサで動作します。これらのインスタンスには最新の第6世代 AWS Nitro カードが搭載されており、前世代の C6in インスタンスと比べて最大43%の性能向上を実現します。

C8ine インスタンスは、前世代と比較して vCPU あたりのパケット処理性能を最大2.5倍向上させます。
