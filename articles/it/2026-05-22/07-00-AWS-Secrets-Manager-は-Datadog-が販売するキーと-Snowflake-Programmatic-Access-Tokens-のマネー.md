# AWS Secrets Manager は Datadog が販売するキーと Snowflake Programmatic Access Tokens のマネージド外部シークレットのサポートを追加します

## 概要

<p>AWS Secrets Manager は、外部のマネージド秘密機能を拡張し、Datadog Keys と Snowflake Programmatic Access Tokens (PATs) を含むようになりました。マネージド外部秘密機能は、AWS Secrets Manager から直接サードパーティの資格情報を自動的にローテーションできるよう、サポート対象のサードパーティサービスとの一体化を提供します。</p>
<p>このローンチにより、Datadog の3種類の認証情報のローテーションを管理できます — API キー、アプリケーションキー、そして管理者認証情報ペア</p>
