# Amazon RDS for Oracle は、Oracle SE2 ライセンス込みの M8i および R8i インスタンスをサポートします。

## 概要

<Amazon RDS for Oracle>は現在、Oracle Database Standard Edition 2（SE2）とライセンス込み（LI）を組み合わせたM8iおよびR8iインスタンスを提供しています。M8iおよびR8iインスタンスは、AWS限定で提供されるカスタムIntel Xeon 6プロセッサを搭載し、クラウド上の同等のIntelプロセッサの中で最高の性能と最も高速なメモリ帯域を実現します。新しいインスタンスは、従来比でコスト性能を最大15％向上させ、メモリ帯域も2.5倍向上しています。
