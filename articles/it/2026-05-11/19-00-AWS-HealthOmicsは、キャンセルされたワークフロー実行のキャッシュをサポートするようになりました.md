# AWS HealthOmicsは、キャンセルされたワークフロー実行のキャッシュをサポートするようになりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月11日 19:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-healthomics-caching-cancelled-runs/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-healthomics-caching-cancelled-runs/)

## 概要

AWS HealthOmicsは、キャンセル済みの実行の完了済みタスク出力をキャッシュする機能を追加しました。これにより、出力を再利用し、すでに完了したタスクを再計算する必要を避けることができます。キャッシュが有効でランがキャンセルされた場合、HealthOmicsは自動的に完了済みタスクの出力を顧客のS3バケットに保存し、キャンセル地点から再度実行を開始できるようにします。 <a href="https://aws.amazon.com/healthomics/">AWS HealthOmics</a> はHIPAA適格なサービスで、医療・ライフサイエンス分野を支援します。
