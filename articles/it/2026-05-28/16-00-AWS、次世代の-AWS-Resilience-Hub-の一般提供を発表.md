# AWS、次世代の AWS Resilience Hub の一般提供を発表

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月28日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-announces-next-gen-aws-resilience-hub/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-announces-next-gen-aws-resilience-hub/)

## 概要

本日、AWSは AWS Resilience Hub の次世代製品を一般提供開始します。AWSコンソール内の中心的な場所で、プラットフォームエンジニアリングとサイト信頼性エンジニアリングのチームが、AWS 上で実行される重要 workload のレジリエンスを評価・強化するのを支援します。この新たなアップデートは、レジリエンス目標の達成に向けた既存の AWS Resilience Hub の機能を拡張し、以下を新たに導入します：アプリケーションモデル、依存関係の発見、生成AIを活用した障害モード分析、モジュラー型のレジリエンス。
