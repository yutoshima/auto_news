# Amazon EC2 C8gn インスタンスが追加のリージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月19日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-c8gn-instances-additional-regions/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-c8gn-instances-additional-regions/)

## 概要

本日より、最新世代の AWS Graviton4 プロセッサを搭載した Amazon Elastic Compute Cloud（Amazon EC2） C8gn インスタンスが、AWS リージョン Asia Pacific（ジャカルタ、ハイデラバード、東京）、South America（サンパウロ）、Europe（チューリッヒ）で利用可能になります。新しいインスタンスは、Graviton3 ベースの Amazon EC2 C7gn インスタンスと比較して、最大 30% の計算性能向上を提供します。Amazon EC2 C8gn インスタンスは最新の 6 世代目 AWS Nitro Cards を搭載しており、最大 600 Gbps のネットワーク帯域を提供します。
