# AWS Lambda マネージド・インスタンス、追加の AWS リージョンへ展開

## 概要

AWS Lambda Managed Instances（LMI）は、イスラエル（テルアビブ）、中東（バーレーン）、中東（UAE）、アジア太平洋（オークランド）を除くすべての商用 AWS リージョンで利用可能になりました。

LMI を使用すると、マネージド Amazon EC2 インスタンス上で Lambda 関数を実行でき、Lambda の操作的なシンプルさを維持しつつ、専用の計算構成や EC2 の価格利点にアクセスできます。LMI はインスタンスのライフサイクル、OS およびランタイムのパッチ適用、ルーティング、ロードバランシング、オートスケールを完全に管理します。
