# Security Hub Extended は、9つのカテゴリにまたがる21の厳選パートナーソリューションへ拡張されました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月20日 18:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-security-hub-extended/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-security-hub-extended/)

## 概要

AWS Security HubのExtendedプランには、9つのセキュリティカテゴリにまたがる21の厳選パートナーソリューションが追加され、SentinelOne（エンドポイント）、CyberArk（アイデンティティ）、Sublime（メール）、Varonis（データセキュリティ）、LayerX（ブラウザ）、Native Security（クラウド）、Zenity（AIセキュリティ）が含まれます。これらの追加により、企業のセキュリティ要件に最も適合するソリューションを柔軟に選択できるようになります。すべてのソリューションは従量課金制の価格設定を公表しており、単一のAWS請求、エンタープライズ割引プログラムの自動適用が提供されます。
