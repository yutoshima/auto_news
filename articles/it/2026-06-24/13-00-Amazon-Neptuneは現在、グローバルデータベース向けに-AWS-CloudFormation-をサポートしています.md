# Amazon Neptuneは現在、グローバルデータベース向けに AWS CloudFormation をサポートしています

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月24日 13:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-neptune-aws-cloudformation/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-neptune-aws-cloudformation/)

## 概要

Amazon Neptune は現在、Neptune のグローバルデータベースのプロビジョニングと管理に AWS CloudFormation を対応しています。新しい AWS::Neptune::GlobalCluster リソースタイプを使用すると、マルチリージョングラフデータベースのトポロジーをコードとして定義でき—展開を自動化し、設定をソース管理に保存し、CI/CD パイプラインと統合します。
