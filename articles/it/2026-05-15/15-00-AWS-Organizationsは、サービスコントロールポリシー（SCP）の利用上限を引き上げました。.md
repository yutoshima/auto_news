# AWS Organizationsは、サービスコントロールポリシー（SCP）の利用上限を引き上げました。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月15日 15:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-organizations-increased-scp-quotas/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-organizations-increased-scp-quotas/)

## 概要

AWS Organizationsは、サービスコントロールポリシー（SCP）のクォータを引き上げました。単一ノード（ルート、OU、またはアカウント）にアタッチできるSCPの最大数は5件から10件へ、SCPの最大サイズは5,120文字から10,240文字へそれぞれ増えました。

このより高いクォータにより、より細かい権限や条件を含むSCPを作成し、ノードごとにより多くのSCPをアタッチして、より包括的なセキュリティ管理を構築できます。
