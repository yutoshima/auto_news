# AWS Cost Explorer が Amazon Q によって強化された自然言語クエリ機能を開始します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月07日 18:30

## 概要

AWS Cost Explorer は、Amazon Q Developer の生成系AI機能を直接費用分析ワークフローに組み込みました。自然言語のクエリを使用して、AWS のコストと使用データについて Amazon Q に質問できます。質問への回答を得られるだけでなく、Cost Explorer の自動更新されるビジュアルも取得できるようになりました。これにより、より rápida な費用分析が可能となり、洞察までの時間を短縮し、費用の可視性をすべてのチームメンバーが利用できるようになります。
