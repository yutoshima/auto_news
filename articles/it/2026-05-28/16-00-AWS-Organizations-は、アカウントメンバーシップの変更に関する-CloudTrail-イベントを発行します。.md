# AWS Organizations は、アカウントメンバーシップの変更に関する CloudTrail イベントを発行します。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月28日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/aws-organizations-cloudtrail/](https://aws.amazon.com/about-aws/whats-new/2026/05/aws-organizations-cloudtrail/)

## 概要

AWS Organizationsは現在、アカウントが組織に参加・退団するたびに、管理アカウントへCloudTrailイベントを自動的に出力します。新しいイベントであるAccountJoinedOrganizationおよびAccountDepartedOrganizationは、組織のメンバーシップ変更に対するセキュリティチームやクラウド管理者の可視性を強化し、以前は見逃されがだった不正行為や潜在的なセキュリティインシデントを検出するのに役立ちます。
