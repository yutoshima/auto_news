# AWS HealthOmics が実行時に Nextflow のバージョン固定をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月01日 23:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-healthomics-nextflow-version-pinning-at-runtime/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-healthomics-nextflow-version-pinning-at-runtime/)

## 概要

AWS HealthOmicsは現在、StartRun APIを介して実行時にNextflowエンジンのバージョンを指定できるようになり、特定のNextflowバージョンにランを固定して移行を管理可能です。このローンチにより、利用可能なNextflowバージョン（22.04、23.10、24.10、25.10、26.04）を新しいengine-settingsパラメータから選択でき、実行時点での明示的な制御を提供します。<a href="https://aws.amazon.com/healthomics/">AWS HealthOmics</a>はHIPAA適格なサービスです。
