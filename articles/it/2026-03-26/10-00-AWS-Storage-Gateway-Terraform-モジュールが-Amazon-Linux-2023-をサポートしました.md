# AWS Storage Gateway Terraform モジュールが Amazon Linux 2023 をサポートしました

## 概要

<p>AWS Storage Gateway TerraformモジュールはAmazon Linux 2023ベースのデプロイを可能にし、Infrastructure as Code（IaC）プロビジョニングのセキュリティ、信頼性、運用のシンプルさを向上させます。更新されたモジュールは、Amazon EC2およびVMware環境の両方で、Amazon S3 File Gateway、Tape Gateway、Volume Gatewayを含むすべてのゲートウェイタイプをサポートします。</p> 
<p>新しいTerraformモジュールを使用して、EC2デプロイメント時にデフォルトでIMDSv2を強制するAL2023ベースのゲートウェイをデプロイできます</p>
