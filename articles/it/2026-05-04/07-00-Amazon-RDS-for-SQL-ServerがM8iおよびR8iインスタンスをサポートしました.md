# Amazon RDS for SQL ServerがM8iおよびR8iインスタンスをサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月04日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/rds-sqlserver-supports-m8i-r8i-instances/](https://aws.amazon.com/about-aws/whats-new/2026/05/rds-sqlserver-supports-m8i-r8i-instances/)

## 概要

<p><a href="https://aws.amazon.com/rds/sqlserver/">Amazon Relational Database Service (Amazon RDS) for SQL Server</a>は、M8iおよびR8iインスタンスをサポートしました。これらのインスタンスは、AWSのみで利用可能なカスタムIntel Xeon 6プロセッサーを搭載し、クラウド上の同等のIntelプロセッサと比較して最高のパフォーマンスと最速のメモリ帯域を実現します。M8iおよびR8iインスタンスは、同等の7世代Intelベースと比べて、価格対性能が最大で15％向上し、メモリ帯域幅が2.5倍になります。
