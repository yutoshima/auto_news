# Amazon FSxは現在、ファイルシステムのバックアップをAWSのオプトイン地域間でコピーする機能をサポートしています

## 概要

Amazon FSx は現在、ファイルシステムバックアップを「 opt-in Regions」（デフォルトで無効化されている AWS リージョン）間でコピーすることをサポートしています。対象は Amazon FSx for Windows File Server、Amazon FSx for Lustre、および Amazon FSx for OpenZFS です。この機能の追加により、事業継続性・災害復旧・コンプライアンス要件を満たすための手段が拡張され、顧客の利便性が向上します。
