# Amazon EC2 Capacity Blocks for ML が AWS GovCloud (US) リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月12日 14:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-capacity-blocks-ml-govcloud/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-capacity-blocks-ml-govcloud/)

## 概要

<p>Amazon EC2 Capacity Blocks for ML は、政府機関および規制対象産業の顧客が機械学習ワークロードのためにGPU容量を予約できるよう、AWS GovCloud (US-West) および AWS GovCloud (US-East) で利用可能になりました。</p>
<p>EC2 Capacity Blocks for ML を使用すると、定められた期間にあらかじめGPUインスタンスを予約でき、短時間の事前学習、ファインチューニング、迅速なプロトタイピング、推論需要の急増に対して加速计算への確実なアクセスを提供します。Capacity Blocks は低遅延を提供します。</p>
