# AWS Elemental MediaTailor が収益化機能を開始します

## 概要

AWS Elemental MediaTailor は現在、収益化機能をサポートしています。これは、広告パーソナライズ再生中の ADS（広告決定サーバー）リクエストの構築方法やセッションデータの管理を顧客がカスタマイズできる新機能です。収益化機能を使えば、顧客は外部 API を呼び出したり、再生セッションの定義済みポイントでインラインデータ変換を実行したりすることができます。これにより、プレーヤーとミドルウェアの間を構築・運用する必要がなくなります。
