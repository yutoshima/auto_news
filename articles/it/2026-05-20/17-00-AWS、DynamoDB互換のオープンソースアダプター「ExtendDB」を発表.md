# AWS、DynamoDB互換のオープンソースアダプター「ExtendDB」を発表

## 概要

<p>本日、Amazon Web Services（AWS）は、プラグイン可能なストレージバックエンドを備えた Amazon DynamoDB API を実装するオープンソースプロジェクト ExtendDB のバージョン 0.1を発表しました。Amazon DynamoDB は、任意の規模で単桁ミリ秒のパフォーマンスを提供するサーバーレスで完全管理された NoSQL データベースです。ExtendDB は、開発者のラップトップを含む DynamoDB のマネージドサービスが利用できない環境において、アプリケーション開発者、プラットフォームチーム、エンタープライズアーキテクトが DynamoDB のプログラミングモデルを活用できるようにします。
