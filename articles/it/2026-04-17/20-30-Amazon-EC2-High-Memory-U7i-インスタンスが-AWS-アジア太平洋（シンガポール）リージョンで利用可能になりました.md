# Amazon EC2 High Memory U7i インスタンスが AWS アジア太平洋（シンガポール）リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月17日 20:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/ec2-high-memory-u7i-asia-pacific/](https://aws.amazon.com/about-aws/whats-new/2026/04/ec2-high-memory-u7i-asia-pacific/)

## 概要

<p>Amazon EC2 High Memory U7i-8TB インスタンス（u7i-8tb.112xlarge）および U7i-12TB インスタンス（u7i-12tb.224xlarge）は、AWS Asia Pacific（シンガポール）リージョンで利用可能になりました。U7i インスタンスは AWS の第7世代に属し、カスタムの第4世代 Intel Xeon Scalable プロセッサー（Sapphire Rapids）によって動作します。U7i-8tb インスタンスは 8TiB の DDR5 メモリを提供し、U7i-12tb インスタンスは 12TiB の DDR5 メモリを提供します。これにより、成長著しいデータ環境におけるトランザクション処理のスループットを迅速にスケールさせることが可能です。</p>
