# Amazon Redshift が AWS Graviton 搭載の RG インスタンスを開始

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月12日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-redshift-rg-instances-powered-by-graviton](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-redshift-rg-instances-powered-by-graviton)

## 概要

Amazon Redshiftは、新世代のプロビジョニングクラスタノードである RG インスタンスの一般提供を発表します。これらは AWS Graviton プロセッサを搭載し、データウェアハウスおよびデータレイクのワークロードを前世代の RA3 インスタンスより最大2.4倍高速に実行し、vCPU単価を30%削減します。RG インスタンスには、Apache IcebergとParquetのデータを処理するRedshiftが自社開発したベクトル化データレイククエリエンジンが含まれています。
