# Amazon RDSとAuroraは、R8gおよびM8gデータベースインスタンスを追加のAWSリージョンでサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月15日 17:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/7/amazon-rds-aurora-r8g-m8g-regions/](https://aws.amazon.com/about-aws/whats-new/2026/7/amazon-rds-aurora-r8g-m8g-regions/)

## 概要

AWS Graviton4ベースのR8gデータベースインスタンスが、アジア太平洋（ハイデラバード、メルボルン、マレーシア）、ヨーロッパ（ロンドン、パリ、チューリッツァー）、AWS GovCloud（US-East）、南米（サンパウロ）、メキシコ（セントラル）地域のAmazon Aurora（MySQLおよびPostgreSQL互換性）およびAmazon RDS for PostgreSQL、MySQL、MariaDBで一般提供開始されました。さらに、M8gインスタンスがUS West（ノースカロリナ）、アジア太平洋（ムンバイ、シドニー、香港）でのAmazon RDS for PostgreSQL、MySQL、MariaDBでサポートされるようになりました。
