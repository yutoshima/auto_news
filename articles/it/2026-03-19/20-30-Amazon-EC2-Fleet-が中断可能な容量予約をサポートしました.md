# Amazon EC2 Fleet が中断可能な容量予約をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月19日 20:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-fleet-interruptible-capacity-reservations/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-ec2-fleet-interruptible-capacity-reservations/)

## 概要

<p>Amazon EC2 Fleet は、割り込み可能な Capacity Reservations をサポートしました。EC2 Fleet を使用すると、複数のインスタンスタイプとアベイラビリティゾーンにまたがってインスタンスを起動できます。本日から、Launch Template 全体に割り込み可能な Capacity Reservation ID を指定して、1 回の EC2 Fleet 呼び出しでインスタンスを用意できます。</p>
<p>オンデマンド Capacity Reservations が使用されていない場合、顧客は組織内でそれらを一時的に割り込み可能な予約として利用可能にすることができ、利用率を向上させます。</p>
