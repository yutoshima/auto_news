# Security Hub Extendedは、9つのカテゴリにわたり、21の厳選パートソルーションへ拡張されます。

## メタデータ

- **カテゴリ**: it
- **公開日時**: 2026年05月20日 18:00

## 概要

AWS Security Hub Extendedプランには、9つのセキュリティカテゴリにわたる21件の厳選パートナーソリューションが追加され、SentinelOne（エンドポイント）、CyberArk（アイデンティティ）、Sublime（メール）、Varonis（データセキュリティ）、LayerX（ブラウザ）、Native Security（クラウド）、Zenity（AIセキュリティ）が加わりました。これらの追加により、企業のセキュリティ要件に最も適したソリューションを柔軟に選択できるようになります。すべてのソリューションは課金制の公開価格、単一のAWS請求、Enterprise Discount Programの自動適用などを提供します。
