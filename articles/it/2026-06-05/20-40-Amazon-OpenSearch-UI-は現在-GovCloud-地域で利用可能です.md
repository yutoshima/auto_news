# Amazon OpenSearch UI は現在 GovCloud 地域で利用可能です

## 概要

<p>Amazon OpenSearch Serviceは、GovCloudリージョン（AWS GovCloud (US-East) および AWS GovCloud (US-West) を含む）に現代化された運用分析エクスペリエンスを拡張し、単一のエンドポイントから、管理ドメインとサーバーレスコレクションのデータ全体にわたる洞察を得られるようにしました。拡張には、チームのコラボレーションと生産性を向上させる Workspaces の追加が含まれ、専用スペースを作成できるようになります。Discover は統一されたログ探索エクスペリエンスを提供するよう再設計され、言語をサポートします。
