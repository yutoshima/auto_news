# AWS HealthImagingが欧州（ロンドン）で提供開始されました

## メタデータ

- **カテゴリ**: it
- **公開日時**: 2026年03月23日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-healthimaging-europe-london/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-healthimaging-europe-london/)

## 概要

AWS HealthImagingがAWS Europe（ロンドン）リージョンで利用可能になりました。AWS HealthImagingはHIPAA適格サービスで、医療提供者、ライフサイエンス組織、およびそのソフトウェアパートナーがペタバイト規模の医用画像を保存・分析・共有できるようにします。

AWS HealthImagingは、医用画像データの保存のための完全に管理されたインフラストラクチャを提供します。既存アプリケーションとの容易な統合のためのDICOMWeb APIと、クラウドファーストの実装のためのAWSネイティブAPIの両方を備えています。
