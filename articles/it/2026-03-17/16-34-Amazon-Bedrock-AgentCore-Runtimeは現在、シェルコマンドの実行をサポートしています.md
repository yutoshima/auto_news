# Amazon Bedrock AgentCore Runtimeは現在、シェルコマンドの実行をサポートしています

## 概要

<p>Amazon Bedrock AgentCore Runtimeは現在、Shellコマンドを実行する新しいAPI「InvokeAgentRuntimeCommand」をサポートしています。これにより、実行中のAgentCore Runtimeセッション内で直接コマンドを実行でき、開発者はコマンドを送信し、HTTP/2経由でリアルタイムに出力をストリーミングし、終了コードを受け取ることができます。コンテナ内で独自のコマンド実行ロジックを構築することなく実現します。</p> 
<p>AIエージェントは、テストの実行、依存関係のインストール、またはgitコマンドの実行など、決定論的な操作を伴うワークフローで動作することがよくあります。</p>
