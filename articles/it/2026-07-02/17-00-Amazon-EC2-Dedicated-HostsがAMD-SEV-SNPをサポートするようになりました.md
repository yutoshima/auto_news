# Amazon EC2 Dedicated HostsがAMD SEV-SNPをサポートするようになりました

## 概要

<p>Amazon EC2は、Dedicated Hosts上でAMD Secure Encrypted Virtualization-Secure Nested Paging（SEV-SNP）のサポートを発表します。これにより、機密計算ワークロードを、顧客が専用で使用する物理サーバー上で実行できるようになります。<br /><br /> 顧客は、SEV-SNPが有効なDedicated Hostを割り当て、それ上でSEV-SNPインスタンスを起動できます。これにより、機密計算ワークロードに対するDedicated Hostの利点（インスタンス配置の制御、ホストの保護など）を得ることができます。
