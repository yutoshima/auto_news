# Oracle Database on AWS は現在、12 の AWS リージョンで利用可能です。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年04月08日 07:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/oracle-database-aws-available-twelve-regions/](https://aws.amazon.com/about-aws/whats-new/2026/04/oracle-database-aws-available-twelve-regions/)

## 概要

Oracle Database@AWS が、EU-West-1（ダブリン）、EU-West-2（ロンドン）、AP-South-1（ムンバイ）、AP-South-2（ハイデラバード）、AP-Northeast-2（ソウル）の計5つの追加AWSリージョンで一般提供されるようになりました。Oracle Database@AWS は、OCI（Oracle Cloud Infrastructure）管理のOracle ExadataシステムをAWSデータセンター内で利用できるようにします。このリリースにより、欧州およびアジア太平洋地域で、地域内データ居住要件を満たす顧客が、オンプレミスのOracle ExadataおよびOracle Real Application Clustersを移行することが可能になります。
