# Amazon EC2 M6in および M6idn インスタンスが、ヨーロッパ（ロンドン）リージョンで利用可能になりました。

## 概要

本日より、<a href="https://aws.amazon.com/ec2/instance-types/m6i">Amazon Elastic Compute Cloud（Amazon EC2） M6in および M6idn インスタンス</a> が AWS ロンドンリージョンで利用可能です。第六世代のネットワーク最適化インスタンスで、3世代目 Intel Xeon Scalable プロセッサと AWS Nitro System に基づいて構築され、最大200Gbps のネットワーク帯域幅を提供し、同等の第5世代インスタンスと比べてネットワーク帯域幅を2倍にします。
