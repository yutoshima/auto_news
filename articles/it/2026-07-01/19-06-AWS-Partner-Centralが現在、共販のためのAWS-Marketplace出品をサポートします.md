# AWS Partner Centralが現在、共販のためのAWS Marketplace出品をサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月01日 19:06
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/aws-marketplace-co-selling-support/](https://aws.amazon.com/about-aws/whats-new/2026/07/aws-marketplace-co-selling-support/)

## 概要

本日、AWSはパートナーが自社の AWS Marketplace カタログから、1つ以上の AWS Marketplace ソリューションおよび製品リストを AWS Partner Central の共同販売機会に直接関連付けられるようになったことを発表します。従来、共同販売機会には、共同販売専用に作成されたソリューションを使用する必要があり、パートナーは AWS Marketplace カタログのソリューションと共同販売用のソリューションを別々に管理していました。パートナーは今後、既存の AWS Marketplace リスティングを共同販売機会と関連付けることができます。
