# AWS Elastic Beanstalk が現在、Network Load Balancer の TLS リスナーをサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月06日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/04/elastic-beanstalk-tls-support/](https://aws.amazon.com/about-aws/whats-new/2026/04/elastic-beanstalk-tls-support/)

## 概要

AWS Elastic Beanstalk は、ネットワークロードバランサーを構成に含む環境に対して、TLSリスナーのサポートを開始しました。TLSリスナーを SSL 証明書とセキュリティポリシーとともに設定することで、ロードバランサーが安全な接続を処理し、復号済みのトラフィックをインスタンスに転送します。Elastic Beanstalk のコンソールまたは CLI から TLS リスナーを設定できます。

以前は、Elastic Beanstalk は管理対象の設定として NLB 環境の TLS リスナーをサポートしていませんでした。
