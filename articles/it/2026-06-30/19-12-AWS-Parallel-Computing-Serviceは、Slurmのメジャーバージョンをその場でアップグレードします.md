# AWS Parallel Computing Serviceは、Slurmのメジャーバージョンをその場でアップグレードします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月30日 19:12
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-parallel-computing-service-upgrade/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-parallel-computing-service-upgrade/)

## 概要

AWS Parallel Computing Service（PCS）は、既存クラスターに対して管理されたインプレース Slurm バージョンアップグレードをサポートするようになりました。実行中のジョブを妨げることなく、クラスターをSlurmのメジャー版を最大3つ先へ移行できます。

アップグレードするには、AWS Management Console、AWS CLI、または UpdateCluster API を使用して、対象の Slurm バージョンを指定してクラスター構成を更新します。PCS は、コントローラ、会計データベース、REST API を含む、すべての管理済み Slurm コンポーネントのアップグレードを処理します。実行中のジョブは継続します。
