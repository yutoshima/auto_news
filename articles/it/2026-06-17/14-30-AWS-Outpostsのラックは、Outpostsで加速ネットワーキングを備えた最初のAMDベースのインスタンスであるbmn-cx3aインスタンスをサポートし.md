# AWS Outpostsのラックは、Outpostsで加速ネットワーキングを備えた最初のAMDベースのインスタンスであるbmn-cx3aインスタンスをサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月17日 14:30
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-outposts-amd-bmn-cx3a/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-outposts-amd-bmn-cx3a/)

## 概要

AWSは、第2世代のAWS Outpostsラックでbmn-cx3aインスタンスの提供を発表しました。bmn-cx3aインスタンスは、最大周波数4.1GHzの第5世代 AMD EPYCプロセッサとNVIDIA ConnectX-7（CX7）ネットワークインタフェースカードを搭載し、ほぼラインレート近い速度で動作する裸金属加速ネットワーク帯域幅を最大800Gbpsで提供します。  

bmn-cx3aインスタンスは、2つのサイズ（bmn-cx3a.metal-32xlとbmn-cx3a.metal-64xl）にまたがり、最大256コアと1.5TBのメモリを提供します。構成は2つのNVMe SSDで、容量は8TB/8TBとなります。
