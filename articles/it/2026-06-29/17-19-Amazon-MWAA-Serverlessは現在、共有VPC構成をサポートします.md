# Amazon MWAA Serverlessは現在、共有VPC構成をサポートします

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月29日 17:19
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-mwaa-serverless-vpc/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-mwaa-serverless-vpc/)

## 概要

<p>Amazon Managed Workflows for Apache Airflow（Amazon MWAA） Serverlessは、共有VPCサブネットをサポートしました。従来、AWS Resource Access Manager（AWS RAM）を介してサブネットを共有していた顧客は、MWAA Serverlessワークフローを作成する際に検証エラーを受けていました。今回の更新により、MWAA Serverlessは共有VPC構成でのサブネット所有権を正しく検証し、MWAA Provisioned環境と整合するようになりました。<br /> AWS RAMを用いてアカウント間でVPCサブネットを共有することは、複数アカウント環境で一般的なパターンです</p>
