# AWS Builder ID は現在、GitHubでのサインインとAmazonでのサインインをサポートします。

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月10日 18:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/aws-builder-id-sign-in-github-amazon/](https://aws.amazon.com/about-aws/whats-new/2026/03/aws-builder-id-sign-in-github-amazon/)

## 概要

AWS Builder ID は、AWS Builder Center、AWS Training and Certification、Kiro などの AWS アプリケーションへアクセスするためのあなたのプロフィールですが、GitHub と Amazon の2つの新しいソーシャルログインをサポートするようになりました。Google や Apple のソーシャルサインイン機能を拡張したこのサインインオプションの追加により、GitHub や Amazon のユーザーは、AWS のリソースにアクセスする際に個別の認証情報を管理することなく、よりスムーズに利用できるようになります。
