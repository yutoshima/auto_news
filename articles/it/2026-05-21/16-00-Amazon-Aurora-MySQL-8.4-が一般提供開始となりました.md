# Amazon Aurora MySQL 8.4 が一般提供開始となりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年05月21日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-aurora-mysql/8-4/](https://aws.amazon.com/about-aws/whats-new/2026/05/amazon-aurora-mysql/8-4/)

## 概要

Amazon Aurora MySQL-互換エディションは現在、コミュニティ版 MySQL の長期サポート（LTS）メジャーバージョンである MySQL 8.4 をサポートしています。Aurora MySQL 8.4 はコミュニティ MySQL 8.4.7 との互換性を前提に開始され、バージョン番号を整合させることで、Aurora で実行するバージョンが互換性を持つコミュニティ MySQL のバージョンと一致するようになりました。Aurora は基盤のパッチ適用も代行して管理するため、日常の運用が簡素化されます。Aurora MySQL は現在、コミュニティ MySQL のメジャーバージョンのうち、リリース後12か月以内のものをターゲットにしています。
