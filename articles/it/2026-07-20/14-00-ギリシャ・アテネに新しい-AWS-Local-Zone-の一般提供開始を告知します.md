# ギリシャ・アテネに新しい AWS Local Zone の一般提供開始を告知します

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月20日 14:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/aws-local-zone-athens-greece/](https://aws.amazon.com/about-aws/whats-new/2026/07/aws-local-zone-athens-greece/)

## 概要

本日、AWSはギリシャ・アテネに新しいローカルゾーンを正式提供開始したことを発表します。アテネのローカルゾーンは、EMEA地域で2番目のローカルゾーンで、Amazon Simple Storage Service（Amazon S3）および Amazon Elastic Block Store（Amazon EBS）ローカルスナップショットをサポートし、ギリシャ国内でデータを保存・処理できるようにして、地域データ居住要件の遵守を支援します。

アテネのローカルゾーンは、C7i、M7i、R7iインスタンスを搭載したAmazon Elastic Compute Cloud（Amazon EC2）および、Amazon S3をサポートします。
