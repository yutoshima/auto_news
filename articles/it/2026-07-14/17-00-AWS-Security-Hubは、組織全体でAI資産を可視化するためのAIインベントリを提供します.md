# AWS Security Hubは、組織全体でAI資産を可視化するためのAIインベントリを提供します

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年07月14日 17:00

## 概要

<p>本日、AWSは AWS Security Hub が AIインベントリを提供するようになったと発表しました。これにより、中央のセキュリティチームは組織全体のAI資産とそのセキュリティ状況を継続的に更新されるビューで把握できるようになります。組織が急速にAIエージェント・モデル・パイプラインを展開する中、セキュリティチームは組織全体に存在するAI資産を把握できていない場合があります。AI資産を活動中の脅威や設定ミスと結びつける中央集権的な可視性がなければ、組織は自分たちが「何が存在するのか」を知らないものを保護することはできません。</p>
