# AWS IAM Identity CenterがFedRAMP Class C認証を取得しました

## 概要

<p><a href="https://aws.amazon.com/iam/identity-center/" rel="noopener noreferrer" target="_blank">AWS IAM Identity Center</a> は、US East (Ohio)、US East (N. Virginia)、US West (N. California)、US West (Oregon) の各リージョンにおいてFedRAMP Class Cの対象範囲に追加されました。これにより、FedRAMP Class Cのコンプライアンスが求められるAWSアカウントおよびアプリケーションへのアクセスを、IAM Identity Centerを用いて従業員に提供できるようになります。</p> 
<p>Federal Risk and Authorization Management Program (FedRAMP) は、米国政府のリスクと認証の管理プログラムです。</p>
