# Amazon Route 53 のプロファイルが、リソースおよび VPC の関連付けに対する粒度の高い IAM 権限をサポートしました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年03月25日 18:34
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-route-53-profiles-granular-iam/](https://aws.amazon.com/about-aws/whats-new/2026/03/amazon-route-53-profiles-granular-iam/)

## 概要

<p>Amazon Route 53 Profilesは現在、細かなIAM（Identity and Access Management）権限をサポートしており、ユーザーが特定のリソースタイプとVPCアソシエーションを自分のProfiles内で管理できるかどうかを制御できます。このリリースにより、個々のリソースタイプ（プライベートホステッドゾーン、Resolverルール、DNS Firewallルールグループなど）に対して、ユーザーを特定の操作（関連付け、関連付け解除、更新）のみに制限するIAMポリシーを作成できます。また、リソースARNに基づく権限の定義も可能です。</p>
