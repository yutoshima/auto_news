# AWS Backup の Amazon EKS 向けサポートが、AWS ヨーロピアン・ソブリン・クラウド（ドイツ）リージョンで利用可能になりました。

## 概要

AWS Backup の Amazon Elastic Kubernetes Service（EKS）向けサポートが、AWS European Sovereign Cloud（ドイツ）リージョンで利用可能になりました。この拡張により、新たにサポートされたリージョンの EKS クラスターに対し、完全管理型でポリシーベースのデータ保護とリカバリを提供します。自動スケジューリング、保持管理、変更不可のボールト、リージョン間およびアカウント間のコピーなどが含まれます。

AWS Backup for Amazon EKS を使用して、EKS クラスター全体、特定のネームスペース、または個々のリソースを保護できます。
