# Amazon DocumentDB（MongoDB 互換性を備えた）は、AWS用エージェントツ toolkitのスキルとして利用可能になりました

- **カテゴリ**: it
- **公開日時**: 2026年07月13日 18:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-documentdb-agent-skill](https://aws.amazon.com/about-aws/whats-new/2026/07/amazon-documentdb-agent-skill)

## 概要

<p>Amazon DocumentDB（MongoDB互換版）は現在、<a href="https://aws.amazon.com/products/developer-tools/agent-toolkit-for-aws/">AWSエージェントツールキット</a>の専門データベーススキルとして提供されています。このスキルを使えば、AIコーディングエージェントがステップバイステップのベストプラクティスワークフローに従って、Amazon DocumentDBクラスターの設定・管理・移行・最適化・トラブルシューティングを実行でき、エラーを減らし、開発者がドキュメントの操作ガイダンスを手動で調べることなく、より迅速に作業を進められます。</p>
