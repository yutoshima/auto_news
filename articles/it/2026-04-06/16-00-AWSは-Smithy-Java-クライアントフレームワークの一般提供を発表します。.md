# AWSは Smithy-Java クライアントフレームワークの一般提供を発表します。

- **カテゴリ**: it
- **公開日時**: 2026年04月06日 16:00

## 概要

AWSは本日、Smithyモデルから型安全なクライアントやスタンドアロンのクラスを生成するオープンソースのJavaフレームワーク「Smithy-Java」の正式リリースを発表しました。Smithy-Javaは、エンタープライズ系Smithyユーザーから長らく要望のあった機能の1つ、つまり実運用レベルのJava SDK生成を実現します。このフレームワークを使うと、モデルと、現代的なJavaアプリケーションの開発者の認知負荷と保守負担を高める非同期パターンからクライアントを生成できます。
