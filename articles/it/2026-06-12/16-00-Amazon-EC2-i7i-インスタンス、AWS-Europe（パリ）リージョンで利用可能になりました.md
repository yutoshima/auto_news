# Amazon EC2 i7i インスタンス、AWS Europe（パリ）リージョンで利用可能になりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月12日 16:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-i7i-instances-europe-paris-region/](https://aws.amazon.com/about-aws/whats-new/2026/06/amazon-ec2-i7i-instances-europe-paris-region/)

## 概要

AWSは、AWS Europe (Paris) リージョンで高性能のストレージ最適化型 Amazon EC2 I7i インスタンスの利用可能性を発表します。第5世代 Intel Xeon プロセッサを搭載し、全コア・ターボ時の動作周波数は3.2GHz。これらの新しいインスタンスは、前世代の I4i インスタンスと比較して、計算性能を最大23%向上させ、価格性能も10%以上向上させます。第3世代 AWS Nitro NVMe SSDを搭載する I7i インスタンスは、最大45TBの NVMe ストレージを提供し、リアルタイムストレージ性能を最大50%向上させます。
