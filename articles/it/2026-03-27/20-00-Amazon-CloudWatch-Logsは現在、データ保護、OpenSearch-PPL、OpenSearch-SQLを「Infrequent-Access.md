# Amazon CloudWatch Logsは現在、データ保護、OpenSearch PPL、OpenSearch SQLを「Infrequent Access」取り込みクラスでサポートします。

- **カテゴリ**: it
- **公開日時**: 2026年03月27日 20:00

## 概要

<p>Amazon CloudWatch Logsは現在、Infrequent Access（Logs IA）取り込みクラス向けの拡張分析機能とデータ保護機能をサポートしています。データ保護、OpenSearchのPiped Processing Language（PPL）およびOpenSearch SQLのサポートを含みます。これらの強化により、顧客は柔軟な分析を実行し、機密データを保護しつつ、AWS上でネイティブにすべてのログをコスト効果的に統合することが容易になります。Logs IAは、頻度の低いトラブルシューティングと事実解析に最適です。
