# Amazon RDS for Oracleは現在、Oracle SE2ライセンス込みのM8iおよびR8iインスタンスをサポートしています。

- **カテゴリ**: it
- **公開日時**: 2026年05月13日 07:00

## 概要

Amazon RDS for Oracleは現在、Oracle Database Standard Edition 2（SE2）とライセンス込み（LI）を搭載したM8iおよびR8iインスタンスを提供しています。M8iおよびR8iインスタンスは、AWS限定のカスタムIntel Xeon 6プロセッサで動作し、同等のクラウド上のIntelプロセッサの中で最高の性能と最速のメモリ帯域を実現します。新しいインスタンスは、従来と比較して価格性能が最大15%向上し、メモリ帯域幅は2.5倍になります。
