# AWSマネジメントコンソールのプライベートアクセスがインターネット接続なしでも機能するようになりました

## メタデータ

- **情報源**: AWS What's New
- **カテゴリ**: it
- **公開日時**: 2026年06月15日 23:00
- **URL**: [https://aws.amazon.com/about-aws/whats-new/2026/06/aws-management-console-private/](https://aws.amazon.com/about-aws/whats-new/2026/06/aws-management-console-private/)

## 概要

AWS Management Console Private Accessは現在、インターネット接続なしのVPCからAWSコンソールへアクセスできるようになり、企業が空閉環境で厳格なネットワークセキュリティ管理を維持しつつ、コンソールを通じてAWSインフラを管理できるようにします。以前は、AWS Management Console Private Accessにより、認可されたAWSアカウントと企業ネットワークへのコンソールアクセスを制限することはできても、依然としてインターネット接続が必要でした。これがローンチされました。
